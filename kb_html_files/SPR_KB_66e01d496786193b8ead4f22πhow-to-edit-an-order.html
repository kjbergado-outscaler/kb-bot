Edit an order


General Information


We are only able to edit customer; invoices or delivery details - up to order Booked status.


  • Customer name

  • Email

  • Contact number

  • Delivery address

  • Invoice address


Products, prices, and payment methods are not editable. For any change related to it: the order should be canceled, and a new order should be placed.

Edit Delivery Information in or before status Order Processing - Step by Step


If the Order status is moved to Processing, modification is not possible within our system, so ensure you follow up by step:

1. If there's no delivery label (tracking label/number), it's never possible to request the carrier to change the address.  If that's the case, kindly ask the customer to wait for the tracking number to arrive (status in OMS = “In transit”).  

2. Once / If there’s a tracking label/number available: 

1. Leave a Comment in the order to document the change.

Comment Guideline in Fluent Commerce:

Currently, Fluent Commerce has a 200 characters limitation for comments. In this way, comments should follow the pre-defined template. Please find below the meaning of each abbreviation:

AGT: ___________________ (Agent's name, as shown on Freshdesk)
GDPR: Y/N ("Y" if the customer was authenticated, or didn't require authentication / "N" if the customer was not authenticated)
TN: ____________________ (Ticket Number)
ENQ: __________________ (Enquiry)
ACT: __________________ (Actions taken)

If the agent requires more than 1 entry to leave complete notes, these should be divided into PT1 (part 1), PT2, etc.



2. Contact the Carrier and inform the changes directly to them.

  • While replying to the customer, let them know that:

    • You are going to follow up on the request but that they’re also able to contact the carrier quicker by themselves by using the options / carrier portal available in the email or HERE ).

    • They can also find the carrier contact information on the Order Status Page.  

    • Due to the logistics of the order process, the order might change status once a request to change the address has been made. The address change is never a guarantee.

Some scenarios:

  1. In case the carrier contacts us reporting an incorrect address: Update the carrier with the correct info.

  2. In case the order is delivered to the incorrect customer: Categorize the ticket as a Logistic Claim

  3. In case the order is rejected (in the wrong address), create a supplement with the correct address.

  4. In case none of the above applies escalate the ticket to a supervisor.
