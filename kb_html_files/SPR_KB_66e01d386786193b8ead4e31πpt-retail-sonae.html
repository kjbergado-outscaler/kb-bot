Orders

Sonae Web orders enter through Sonae Portal and are created manually by the Backoffice, without any CS interfervention.

​

For the Sonae Stores orders & Centralizations, these may arrive through the CS and must be escalated to Backoffice for order creation.

​

  • Stores -> Delivered directly in the customer's house. An estimated delivery date can be provided.

  • ARP -> Delivered in Sonae. Warehouse. An estimated delivery date can not be provided because the delivery is scheduled. Please always escalate to Emma before providing an ETA for this type of orders.

​

Create Orders

Open the Retail Backoffice Portal and choose Order Creation -> Manual Order Creation -> Sonae

​

Please give the following information to the Backoffice team:

  • Account ID

  • External Number

  • Client’s Name

  • Client's E-mail

  • Delivery Address

  • SKUs

  • PO File

After creating the request, paste the Jira number as an internal comment in the Freshdesk Ticket.

Once the Jira is completed, confirm that the order is correct and inform the retailer that the order was placed. If any of the products appear as Out Of Stock, please inform that these SKUs may arrive later due to lack of stock.
