What is Agent Skills configuration?

When a customer has a problem, they want to talk to someone who knows how to fix it. The Skills configuration is what makes sure that the customer's problem goes to the right agent. For this to happen, the Agent needs to have the right Skills set-up, and those skills are: market, language (if applicable), channel, line of business, BPO code.

IMPORTANT: The Skills mentioned above are already configured when the user is onboarded at Emma and their account is created, based on the information provided by BPOs to the Emma IT team.

How to check the Skill configuration per agent?

1) On the Supervisor Console, you need to open the "Team Monitoring" tab, and there you can check each agent. Next to the agent name, you have to click on the small black arrow that indicates "Open details" and then click on Properties" in the right side menu.

2) After you open the Agent Properties, you have to check the following 4 fields:

  • "Skill IDs": indicates the Line of Business that the agents supports (i.e.: Support X Online Reputation X Offline, etc)

  • "User Channel Handling": indicates which channel the agent supports (i.e.: Email X Voice X Chat, etc)

  • "Market": indicates the market/country the agent supports, and it can be more than one if applicable.

  • "Voice direction": If the agent supports Voice channel, this field it's irrelevant. However, for agents that support other channel and still have to make calls to customer, this field need to indicate "Outbound".

Note: Don't worry if you see different information in other fields like "Channel" and "Line of business". The only fields that will determine the case assignment are the ones indicated on the list above.

What is the expected Skill configuration per Line of Business?

​

D2C Customer Support:

  • “Skills Ids” required: Support and Sleep Experts

  • “User Channel Handling” options: Email and/or Voice and/or Chat

  • Market options: according to the market(s) supported


 D2C Online Reputation & HLE:

  • “Skills Ids” required: Online Reputation and High Level Escalations

  • “User Channel Handling” options: Social and Email

  • Market options: according to the market(s) supported

  • "Voice Direction" field: mandatory to indicate "Outbound".

B2B (Wholesale) 

  • “Skills Ids” required: Offline and/or Marketplaces

  • “User Channel Handling” options: Email and/or Voice

  • Market options: according to the market(s) supported

 

O&O (Store)

  • “Skills Ids” required: Store

  • “User Channel Handling” options: Email and/or Voice (only for ES and PT that Voice is applicable)

  • Market options: according to the market(s) supported

  • "Voice Direction" field: mandatory to indicate "Outbound".


Backoffice - N/A - don't use

Disclaimer: In case of doubts regarding the correct channels to use per line of business, please reach out to your BPO account manager.

When to make changes?

Changes can be made for situations such as:

- Agent moved to another team. Example: CS agent moved to Online Reputation team; or to Offline team.

- Agent is temporarily giving support to another market or channel. Example: Voice only agent for FR is now supporting FR Email queue.

​

Once you make any change to the Agent Skills configuration, it means the agent will now have the criteria of another Queue and only receive the cases related to that Queue.

How to make changes on Skills configuration:

You will only need to make those changes if one agent will give support to a new/different working queue. The first step indicates how to change the Line of Business; and the second step shows how to modify the channel and/or market.

-> Changing Line of Business (Skills IDs) for Agents:

​

Note: After 4th March, the only Skills visible to be changed on that option are: Emma Skills and BPOs. This is the expected behavior. If you need to change other criteria on the agent configuration, check the next part of this article.

Please follow the instructions below:

  1. At the Supervisor Console, open the "Team Monitoring" option present on the left hand side menu;

  2. Select the necessary users;

  3. Click on the "Edit/Add Skill";

  4. On the new screen, select the skill category you want to change/edit and add it to the agent. On the option Proficiency, DO NOT change, keep always on 100.

    ​

IMPORTANT Note: It is MANDATORY TO NOT change anything under the part named "Capacity"! If the Capacity field is empty for any agent, please raise a ticket on the IT Portal asking for the capacity to be configured.

-> Changing Channels/Markets for Agents:

Then for this, you need to change the Channel and the Market too. For that, check the following instructions:

  1. At the Supervisor Console, open the "Team Monitoring" option present on the left hand side menu;

  2. Select the necessary users;

  3. Click on the "Macro" option and select "Set Agent Fields" macro;

  4. Select the relevant Channel, Market and Language (if applicable) to put an agent to diffeent user groups and ultimately putting them to correct work queue.

  5. The "Voice Direction" field: If the agent supports Voice channel, this field it's irrelevant. However, for agents that support other channel and still have to make calls to customer, this field need to indicate "Outbound".
