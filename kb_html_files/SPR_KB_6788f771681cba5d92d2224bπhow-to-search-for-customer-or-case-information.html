Search Bar

On the Agent Care Console, you have the "magnifying glass" button that by clicking it opens a pop-up window where searches can be done.

On that part, you need always and only use the following options for search: "Case", "Message" or "Profile". The other options are not relevant to Emma's search.

Step 1: Select the option that suits you best.

Step 2: Type the content.

Using Search Bar options

As mentioned, you will only use either "Case", "Message" or "Profile" for the searches. Now let's see which information you should input in each case to make your search more accurate.

Case:

You need to have the case number in order to find it on the system. At the example below, in red you can see the "case" option select and the case number that was typed, and on yellow the case that was found.

Message:

If you need to search for a more detailed level of information, like specific key words that were used in the case. This is more usefull for Non-Voice cases (live-chats, emails, social media).

If the search is done with key-words it is mandatory to use quotation marks ( " " ) at the begining and at the end, in order to make the search accurate for that expression, sentence.

IMPORTANT:

Searching by Order Number only works if the customer mentioned the order number in their contact with us (via email or live chat). The search by Order number is not integrated with FluentCommerce or Magento. Or even postal code, address, or phone number. However, the search like that will only work if tha information was mentioned in written in the communication form the customer to us.

Profile:

If you have the name of the customer (order placeholder) you ned to select the option "Profile" and type the name. Then the system will show you all the different channels on which the customer have reached out to us.

You can also use the customer e-mail address to search for their profile. Then, from the Profile, you can open all the cases and conversation history from that customer.
