Packed mattresses and accessories in the UK are returned via DPD, Evri or DHL. To create a return with for an accessory in Magento, please do the following.

For a general refresher on how to create a return on the order management system, check: How to: Product return (Magento)

1. ​​​How to create manual returns for drop off label and packed products with Magento in the UK

When do we use DPD, and when DHL? For Magento orders of accessories, by default, they are returned via DPD. Only if there is an issue when creating the return via DPD, we ask you to select 'DHL label Accessories' instead. In the latter case, the customer will automatically receive a drop off label from DHL.

  1. Open Magento

  2. Click the 'Return prodcuts' button on the top right of the order page

  3. Fill all the required fields

    1. Return products: product will be skipped if checkbox not set 

    2. Quantity: quantity of product for return

    3. Quality: Available options: packed

    4. Refund to the customer: Available options: full refund, no

       Depending on selected value, an email template will be choosen

    5. Pickup/Return-service: Available options Based on Manual return assignment rules

      For UK orders, this should be selected to 'DPD UK Self Return'.

    6. Carrier Info:

      1. Track number Track

      2. Link

    7. Warehouse Address: Warehouse address for returned product

    8. Reason: Available options:

      1. The item is received too late

      2. Product is too firm

      3. Product is too soft

      4. Cannot adjust the height for myself

      5. Product is too warm, cause sweating

      6. The cover does not have a nice touch

      7. Ordered wrong product size

      8. Other reasons (the comment is compulsory)

      9. Ordered twice

      10. Product Claim

      11. Logistic Claim RTS / Cancellation Reason of return

    9. Comment: Comment to return

    10. Pickup Date Pickup Date if available

    11. Credit Memo Comment(B2B only!)

    12. Credit External #(B2B only!)

    13. Product back at warehouse: If set yes, return will be moved to complete status

2. How status updates are handled

Returns on Magento will have to be tracked on the carrier's portal (see info below) and manually updated on the order management system.

Note: For Magento returns, the carrier’s tracking number must be added to the ticket whenever possible.

When using DPD's services, it's essential to know how to manage your delivery preferences effectively. This guide will walk you through the steps to change your delivery to collection, access more detailed options, and contact DPD for further assistance.

Step-by-Step Guide

Change the pick up address or date

  • Navigate to the top right corner of the DPD interface.

  • Select the option to change your delivery to collection.

  1. Initiate the Search

    • Click on the "Search" button. This will help you locate your specific delivery or collection details.

  2. Access More Information

    • Once you have performed the search, click on "More Information."

    • This action will open a new page where you will find various options.

Additional Options Available

On the new page, you will have the ability to:

  • Change the collection address.

  • Alter the delivery dates.
