Order Management

Validating Orders

  • Go to Orders > Orders to Accept.

  • Review order details and click Accept.

  • If an order cannot be fulfilled (e.g., out of stock), Reject the order and add a reason.

​

​

In-Progress Orders

  • View order statuses under Orders > In Progress.

  • Ensure shipments and status updates are properly reflected from Fluent Commerce.

​

​

​

Note: Use the “ID commande” number to find the order in Fluent Commerce.

​

Returns

Managing Returns

  • Go to Orders > Returns.

  • You'll see:

    • Return Requests: New returns to review.

    • Ongoing Returns: Returns in progress.

  • Process the return first in Fluent Commerce.

  • Then update status in Mirakl if needed (e.g., confirm reception).

​

​

Messages

  • Go to Messages > Inbox.

  • All customer messages (e.g., claims, delivery queries) are located here and should be handled via Mirakl.

  • Make sure messages are responded to daily.

​

​

Incidents / Claims

  • Go to Incidents or “Claims” depending on marketplace terminology.

  • Review the incident message, identify the issue (e.g., missing item, non-delivery).

  • Cross-check order details in Fluent Commerce.

  • Respond with a proposed resolution or escalate if needed.

Cancellations

  • If a customer requests a cancellation:

    1. Attempt to cancel the order in Fluent Commerce.

    2. If successful, accept the cancellation request in Mirakl (portal).

    3. If Fluent cannot cancel (e.g., already shipped), reject the cancellation in Mirakl.

Late Deliveries

Mirakl may falsely mark on-time orders as late due to missing tracking.

  • Go to Orders > Late Deliveries (or check the delay report).

  • Verify in Fluent Commerce if the order is actually late.

    • Sometimes tracking info fails to sync from Fluent to Mirakl.

    • If so, manually add tracking info in Mirakl.

  • Add tracking number, carrier, and date.

Reports & Exports

  • You can export order, return, and incident lists using the export button in each view.

  • Use this for audits or escalations.
