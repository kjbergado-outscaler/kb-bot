This SOP provides a detailed walkthrough of managing orders, tracking, cancellations, and returns within the John Lewis Portal. It outlines each step with clear instructions and indications of where screenshots should be added for better understanding.

​

Accessing the Portal

Logging In

  1. Open the John Lewis Portal.

  2. Enter your login credentials (username and password).

  3. Click Login.

​

​

​

If you encounter login issues, report to your supervisor.

​

Navigating the Portal

Overview of the Dashboard

Upon login, you will see:

  • New Orders: Pending orders.

  • Processing: Orders being handled.

  • Dispatched: Orders sent to the customer.

  • Back Orders: Orders delayed due to stock issues.

  • Returns: Return requests.

  • Cancelled: Cancellation requests.

​

​

Managing Orders

Finding an Order

  1. Go to Orders → Find Orders.

  2. Enter the External Order Reference Number (first six digits of the distribution number).

  3. Click Search.

  4. Select the order from the search results.

​

​

Checking Order Details

  1. Click on the order reference to open order details.

  2. Review key information:

    • Distribution Number

    • Status (New, Processing, Dispatched, Back Order)

    • Item details (product name, quantity, price)

​

​

  • You can find the Fluent Commerce order number under “Activity”.

​

​

Updating Order Status

Updating Supplier Delivery Date

  1. Navigate to the order details.

  2. Click Status Updates.

  3. Verify replenishment information using the Stock Report.

  4. If needed, adjust the supplier delivery date:

    • Example: If replenishment is scheduled for Feb 3rd-7th, add a buffer and enter Feb 14th.

  5. Click Save Changes at the bottom of the page.

​

​

Back Orders

Updating the order status to Back Order ensures that customers are informed about stock shortages and expected fulfillment timelines. These updates help manage expectations and reduce customer escalations.

  1. Open the order.

  2. Select Actions → Back Order.

  3. Choose Reason → Out of Stock.

  4. Enter the estimated delivery date based on stock availability.

  5. Ensure the delivery method remains Direct to Customer.

  6. Click Submit.

  • Criteria for setting the date:

    • Check the replenishment date from the Stock Report.

    • Make sure to add a 7-day buffer from the expected replenishment date.

​

​

​

Cancellations

Pre-Delivery Cancellations

  1. Open the order.

  2. Select Actions → Cancel Order.

  3. Choose a Reason (e.g., Customer Delay, Out of Stock).

  4. Click Save Changes.

​

Post-Delivery Cancellations

  • Not applicable in the portal; must be handled via Sprinklr and escalated if necessary.

Handling Returns

Creating a Return

It’s not necessary to enter the Returns information in the JL Portal.

​

Sprinklr Communication

Handling Customer Inquiries

  1. Verify the customer's Order Number, Name, and Delivery Address.

  2. Respond based on order status:

    • If Processing, provide estimated delivery time.

    • If Delayed, update using stock reports.

  3. Tag the case correctly in Sprinklr.

Common Issues & Troubleshooting

Amending Address

  • Usually happens due to an incorrect address format. You would know that the address needs to be amended if the fulfillment order status is CARRIER_ALLOCATION_PENDING.

  • Shimul reaches out to us if there is an address error in the Order.

  • Update the address format in Fluent Commerce and ensure proper allocation.

    • Open the order in Fluent Commerce;

    • Click in the More Actions button (three-dots option) → Change Address.

​

​

​

​

The change in delivery address is only available until the BOOKED WAITING_FOR_STOCK status. 

Order Errors in Sprinklr

Sometimes orders appear as "New Order" in the John Lewis portal, but when checked in Fluent Commerce, they are already being processed. The resolution for this issue is to manually update the tracking and update the Retailer through Sprinklr accordingly, ensuring accurate information is provided.
