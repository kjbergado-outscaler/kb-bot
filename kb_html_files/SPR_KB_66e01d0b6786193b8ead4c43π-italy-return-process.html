Returns Process for Customer Service Agents

Before starting always make sure you have all the information needed for the return:

A. Reason to return (if customer doesn't mention it, please ask it);

B. Product status (packed or unpacked): we can deduce it from what customer wrote (" the mattress is too soft" - which implies it was tested). When this details is not deductible, we need to ask it.

C. Items to be returned: if the information is not provided and the order has multiple products, always double check with the Customer.

Step 2: Returning a bed

We have different processes based on the Quality of the bed.

​

1. Packed

Whenever a bed is still fully packed in the original boxes, we will follow the same process as above (without the need of the plastic bag).

Note: we will need to add to the voice "collo" has many boxes as customer is returning.

2. Unpacked: the bed has been assambled.

A. Order from Fluent

1. Create the return on Fluent

2. Select Invio as courrier

3. Reply to customer by useing canned Response: IT - Return Pick up Unpacked Bed.

4. Leave the return as created and not organised.

​

The returns with Invio are manually checked: creating a return will trigger a request in to Invio system. The courrier will contact the customer to schedule a return together.

​

B. Order from Magento

1. Create the return in Magento;

2. Select Loghistes as courrier;

3. Fill this file: Beds return Magento.xlsx

4. Reply to customer by useing canned Response: IT - Return Pick up Unpacked Bed.

The customer needs to disassemble the bed. Packing is not mandatory, however, if they still have the box it would be nice to put it in there. If not, the courier will pack it themselves.

Return and refund status

What should we do if a Customer contact us about their return status and refund?

Check the tracking on Poste Italiane page: https://www.poste.it/cerca/index.html#/risultati-spedizioni

Was the return picked up?

​

- Yes: if the status is Presa in carico, In transito, in consegna or consegnata please mark the return as complete on Fluent and inform customer.

Canned response: IT - Stato Reso - Consegnato / IT - Stato Reso - In transito presa in carico

​No - if the tracking is not available - ask customer for the CP (codice di prenotazione provided to them when they booked the return) and then contact Bracchi (courier.bassano@bracchi.it) via Sprinkrl following the Logistic Claim process.​


 

Tracking stopped midway:

 

if the return was picked up but the tracking info stopped midway for more than 5 business days,

 

mark the return as complete on fluent and inform customer.

 

Open a Logistic Claim for Bracchi following the proper KB Article.
