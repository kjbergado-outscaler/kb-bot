Cette fiche à pour but :


- D´expliquer comment saisir les informations pour créer un ordre de retour sur Relais Colis

- De savoir ou trouver les informations pour l´ordre de reprise.

- D´expliquer les différents problèmes pouvant se produire lors de la création d´une reprise.

- Process Update à la fin de l'article


Dans un premier temps il va falloir se connecter sur l´interface Relais Colis

http://service.relaiscolis.com/Enseignes/Accueil/Index


Une fois sur relais colis : Aller sur "Ordre" puis "Créer une Expédition"



Voici la page que vous devrez ensuite avoir



Toujours sélectionner "Reprise pour retour enseigne"



La référence de reprise se crée automatiquement en cliquant sur "Générer"

Puis dans nombre de paquet indiquer le nombre d´articles à reprendre.


Après avoir indiqué le nombre de paquet cette fenêtre va s´ouvrir. Pour trouver les informations de poids. vous devrez vous rendre sur le site Emma puis sur la fiche de l´article.



Une fois sur la page de l´article descendez jusqu´a atteindre cette interface.


Cliquez ensuite sur "Dimensions et poids du matelas Emma.

Vous pourrez ensuite voir le poids de chaque article



Saissisez les dimensions et le poids de chaques articles puis cliquer sur le bouton valider.


Dans la partie ci-dessous. Insérer le numéro de commande dans l´onglet "N de Commande"


Dans type d´article toujours choisir Literie


Dans Prestations, toujours choisir Livraison à 2


Compléter obligatoirement les cellules en Jaune.

Si la ville n´est pas proposée dans les choix vérifier sur google le code postal de la ville en question et son nom.

Il arrive que certain client continuent de mettre le nom d´une ville qui n´existe plus, dans ce cas la corriger pour le client et mettre la ville ui n´existe plus en "Complément".




Attention Dans commentaire libre toujours indiquer "Prévoir une housse"


En cas de reprise de deux matelas indiquer prévoir deux housses etc car il peut arriver que pour la reprise de plusieurs produit le livreur ne vienne qu´avec une housse.




Remplir les 3 onglets téléphone, même si le client en a seulement un.



Dans "Libellé de l´article" saisir le n° de la commande

et dans "Codif Stock" insérer le SKU à reprendre.


Le "Code fournisseur" n´est pas à remplir.



Une fois les informations saisies, cliquer sur "valider"


Après avoir validé la reprise aller sur Magento ,faire une demande de retour et modifier le statut en "Sent to Relais Colis" et cliquer sur "Save"



Process Update 1:

  • Retours Commandes de matelas uniquement : Utiliser le Code 00-EMMA MATELAS dans la case Site de retour sur Relais colis
  • Retours Commandes de matelas et autres produits : Utiliser le Code 01-EMMA MATELAS dans la case Site de retour sur Relais colis

Pour les commandes multi produits, bien remplir dans le cadre LIVRAISON le nombre de produits avec toutes les quantités correspondantes.


Process Update 2:


-Pour le matelas Original et le matelas hybride : il faut faire le retour seulement sur Magento et mettre le statut en "Sent to Relais Colis" ==> Pas besoin de faire le retour sur RC


-Les surmatelas ne sont pas considérés comme accessoires, il faut bien faire le retour manuellement sur Relais Colis
