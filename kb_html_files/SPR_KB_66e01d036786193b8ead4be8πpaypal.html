Order status

Pending: Payment has not been received.

Payment Review: Signifyd has not confirmed payment

Payment Received: Payment is there, order is being processed Processing Payment is there, Delivery note is out

Complete: Payment is there, product is out, Invoice is out Refunded Whole order is returned to us, order is refunded

Cancelled: Order has been cancelled (before delivery note)

Cancel requested: Complete or partial cancellation is requested but is still pending (Order Management needs to accept it)

Order Status - Pending

Possible Issues:

  1. Contact information issue with PayPal express orders

  1. Signify Issues: Sometimes API is not working properly so we cannot verify if the order is legitimate or not. It happened just one times in the past

Resolution

  1. In situations where customers insist their monies have been taken, Request for a payment confirmation in order to investigate with Refund team, together with the confirmation note received from customer.
  2. Customers need to cross check to be sure if payment was indeed received as sometimes, we have not taken any monies. Advise customer to crosscheck with service provider or bank
