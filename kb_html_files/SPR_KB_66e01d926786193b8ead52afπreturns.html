Returns should only be processed within the Trial period of the customer's country or if related to a warranty claim.

Check the product status.

We should determine under which category does the return fall:


Packed Product : These are the ones that are in the box and/or still in vacuum package. Packed products are returned separately because they can be returned to the warehouse to be used for another order.


Unpacked Product : These are the ones that are out of the vacuum packaging. Unpacked products are a separate return category as they cannot be returned to the warehouse and need to be picked up by a ONG or disposed.



Packed product.


If the product is still packed, we should request for it to be picked up and returned to the warehouse.

Once we have confirmed that the product has been picked up, we can process a refund (if applicable). If the return is due to a product claim, we can create a supplemental order for a replacement.


Keep in mind that if the customer is outside of the Trial period, we can only offer a replacement.

Unpacked product.


If the mattress is unpacked, the return will be done via pick-up by ONG. (ONG_LATAM.xlsx)


Because ONG pickups can take some time, we should immediately send a letter of responsibility to the customer. (Letter of responsibility: MX, Chile and Colombia)


After the customer has signed the letter of responsibility and we have received it, we can process the refund or the replacement without waiting for the ONG pickup. To process a refund use the following:


RegionFile
LATAMAM Refunds.xlsx
US/CAUSCA_database.xlsx


While we are waiting for the ONG pickup, we should leave the case In Progress and follow up through the file to expedite the pickup. Make sure you update the customer on the status every 3 days or as agreed with the customer.


After the ONG pickup, once you have provided proof of the refund to the customer or the customer has received their replacement, you can consider the case as waiting for customer.
