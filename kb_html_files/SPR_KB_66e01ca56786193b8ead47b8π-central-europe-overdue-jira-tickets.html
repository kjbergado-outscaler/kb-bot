Introduction


If Jira tickets are overdue (ticket must start with ECE), you can follow the below procedure.


If a ticket is overdue you can add it to this sheet. This sheet will be checked by Emma every day and there is therefore no need to push these tickets via email.


When is a ticket overdue?

It depends on the escalation queue, when a ticket is overdue:


Queue
Ticket is overdue after
Backoffice
1 working day
OM&OPS: OPS escalation3 working days
OM&OPS: Last mile3 working days
Refund issue5 working days


Escalation queues


As seen in the sheet you need to add the tickets pr. escalation queue.

The below overview you can find in the Jira portal for Central Europe by clicking here.


You need to use the queue that the ticket is currently in.


Where to add the overdue ticket

The overdue tickets should be added to this sheet in the correct column for the queue the ticket is in.



You need to fill out all the fields for the queue.
