Instructions

Order Creation

  • Orders came via Freshdesk, and should come via Sprinklr (should the client send us any new orders).

  • Locate the order in the assigned inbox.

  • Confirm Purchase Order (PO) number.

  • Ensure delivery is scheduled for Collingwood's warehouse.

Update the Order Creation tracker with the following details:

  • PO Number

  • Store Code

  • BU Key

  • Delivery Address

    • Orders should always be directed to Collingwood’s warehouse.

  • Order details

  • Customer’s Name

  • Email

  • Telephone

Returns and Refunds

Customer Inquiry Handling

  • If a customer contacts us to request a Return, redirect them to Collingwood.

Processing Returns

  • Wait for Collingwood to confirm a return request.

  • Once confirmation is received, process return completion in Fluent Commerce.

Issuing Refunds

  • Refunds are processed once the collection is completed.

  • Credit memos are generated within 2 days after return completion.

Note: Customers should not be given order details beyond what’s available in Magento.
