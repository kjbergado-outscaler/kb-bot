Understanding Return reasons

There may be different scenarios in which customers may request to return their product. First, you should always understand if the return is:

  • Trial period: It means the customer would like to return for certain reasons under the risk-free trial period, the exclusive benefit offered to our Emma customers that can vary between product and country (100-nights, 50 nights, etc).

Caution: Jysk and Lidl don't have trial nights. Please only accept Claim Related Returns.

  • Claim related: (logistic or product claim): It means the return is related to a logistic claim (delay, out of stock, damage) or a product claim (defective product). If the return is related to a claim, please ensure you handle the request as per related claim steps, not as a Return (Trial period) request.

Pick up declaration

When a collection is requested by a charity, we must ask the retailer to send the collection declaration once it is done. The institution can send the declaration, but if they do not, we ask the retailer.

 

How long to wait for the declaration?

If the retailer does not send the collection declaration within 2 days, we can send a new email asking if the collection has already been done. If they do not respond, we can call the store. We can also contact the charity (by email or phone) to confirm if it has been done.

In the case of replacements, this confirmation is enough, but for returns, we need the proof. The declaration is mandatory for issuing the credit note, but we must also ensure that the item was collected even when there is no credit note, so we can ask the institution or carrier.

If the institution informs us that the collection can only be scheduled within 2-3 weeks due to the customer, we can wait up to two weeks. If it takes longer or no time frame is indicated, we inform the retailer that the matter will be closed until they contact us again with the declaration.

When the collection is done by the carrier, we should not wait for it to return to the warehouse to proceed with the credit note issuance process, only until it is collected from the customer’s house. From the moment the customer returns the item and has proof, it is no longer their responsibility to wait for the goods to arrive at the warehouse, but the carrier’s responsibility.

Cabeceiras’ Pickups

Cabeceiras is making its own pickups of opened mattresses. There is no problem; we can proceed with the credit note or send a new mattress depending on the situation if they inform us that the collection has already been done.

Comfort Layer

To avoid a return, the retailer can offer the final customer a comfort layer, as long as the order is within the 100-night period. This is not a new order; it is a supplement request for a Comfort Layer. We do not sell Comfort Layers; it is a non-commercialized product that customers can request only within the 100-night trial period if they complain about the mattress firmness.

The color of each component can help us identify them: if it is blue, it is Firm, and if it is white, it is Soft.

Return Packs

Regarding returns, if we verify that it is a Pack (e.g., Mattress + protective cover), it is not possible to return just one product; all items included in this Pack must be returned.
