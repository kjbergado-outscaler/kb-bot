Introduction

This article if for the use of Supervisors and Team Leads

If you are a Supervisor or a Sleep Experts that need to request access to Talon.One, please check the article Overview & How to Access.



General rules for code creation in Talon.One

Please read those rules carefully:


  • Always create 1 code per customer and inform them that the code can only be used 1 time (single use).

  • Make sure to inform the customers that the code has a validity of 10 days. After this, it will expire and cannot be used;

  • If you cannot find the Sleep Experts campaign, NEVER create a campaign and or a code in another random campaign.

  • Please delete codes every end of the month: We have a limit of existing vouchers on TallonOne. Please delete batches of vouchers (that are expired) at the end of every month to keep the number of vouchers under the limit.


How to create discount codes


Step 1: Log in at the Talon.One platform


Please, log into the Talon.One platform with your new Emma credentials. The initial page will show a dashboard; on the left, your country and the total number of campaigns activated:






If you work only for one Country, it will appear by default only one option (like the image above). If you have requested access to more than one market, in the top left corner will appear all the countries you work for, and please make sure to select the one Country in which the coupon needs to be created.



Step 2: Select the campaign and create the codes

Select the Sleep Experts campaign you want to create the coupon in. The other un-highlighted vouchers come and go as per market demand, so you might see other random vouchers on the list, but remember that you will only use the one for your market and named as "Sleep Expert."

This campaign should always be there. However, if you notice that the campaign for sleep experts is missing, please immediately communicate it to your supervisor and they will reach the Emma team responsible for it. And NEVER create a new campaign or a code in another random campaign.





After you click on the campaign, as per the example above, the "DE Sleep Expert 5%", you go to the menu on the left and click on Coupons. And then, click on Create Coupons in the top right corner:







Step 3: Create coupons with the right definitions

After opening the above page, you need to select the option Random Code and fill in the options as explained in the next step:



Please follow the instructions below to fill in the fields to complete the correct creation of the coupons:


- Number of Coupons: Insert how many coupons you want to create. Remember, each customer receives one coupon. but you can already create more than one coupon.

- Per-customer redemption limit: Always insert "1" in this field. It indicates how many times the customer can use the coupon.

- Box of "Allow unlimited redemptions": NEVER select the box that says We DONÂ´T allow coupons to be used more than once.

- Start date: DON'T insert a start date since we have customers in different time zones. Just leave it empty as it will make the coupon valid right away.
- Expiration date: Select the date that represents 10 days from the code creation moment. Each code needs to have a 10 days validity.

- Advanced box: You don't have to select anything there. Don't use it.

- Finally, click on Create on the bottom right page.







After clicking on Create, the next page will show the Batch ID, which can be used to track all the coupons created by one user.

The Batch ID helps you to find all codes you created, check the redemption limit, and also when each one of them will expire.


Additionally, via the Batch ID, it is how you can delete all of them at once. Remember: at the end of every month, please delete batches of vouchers THAT ARE NOT EXPIRED YET.
