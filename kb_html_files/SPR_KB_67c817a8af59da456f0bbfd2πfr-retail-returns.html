There are three main reasons why a return may be necessary:

​

Late Cancellation

This happens when a customer wishes to cancel the order, but it is already too far along in the process and can no longer be cancelled. In this case, the customer has two options:

  • “Refusal at the door” or "Refusal to collect the parcel at the relay point": this refusal will result in the product being returned to our warehouse by the carrier.

  • "Trial Period return": if the customer was unable to refuse delivery, proceed with the return as a trial period return.

Return during Trial Period

This means that the customer wishes to return the product for certain reasons during the risk-free trial period, which may vary according to product and country (100 nights, 50 nights, etc.). Please only accept returns related to product complaints.

For mattress returns, we must create the return in the Order Management System (Magento or Fluent Commerce) and organize the return on the carrier's platform.​

Warning Group: https://euromatic.warning.fr/colisweb

​

Important: Jysk and Lidl do not offer trial nights.

Returns related to Logistic or Product Claims

This means that the return is linked to a logistical complaint (delay, stock shortage, damage) or a product complaint (defective product).

​

Note: Returns are managed by Emma's sales staff. They must then be created and tracked in Fluent Commerce or Magento, and updated manually.

​

Internal Organization

  • The carrier's tracking number must be added to the ticket as soon as possible.

  • All return requests must be made by the store via a returns form sent to the following e-mail address: distributeurs@emma-matelas.fr

  • Update B2B Returns Tracking File. This file was created for tracking store returns and refunds.

How to create a Return in Fluent Commerce

Search for the Order in FluentCommerce

  • Locate the external order number in Fluent Commerce.

    • Confirm that the external order number matches the one you copied.

Check Comments Section

  • Review the "Comments" section for any context regarding previous requests or issues related to the order.

Creating a return with a refund automatically creates a credit memo.

Conditions for Accepting a Return with Refund

  1. Product Claim:
    A return with refund can be processed if the customer requests a refund instead of a new product.

  2. Incorrect Delivery:
    If there has been a wrong delivery, a return with refund may be issued upon the customer’s request.

  3. Undelivered Mattress (as per carrier tracking):
    If the mattress is marked as delivered by carrier but the customer has not received it, a return and refund can be processed upon customer request.

  4. Failed Delivery (Returned to Sender):
    If the mattress was unable to be delivered and has been returned to us, a return and refund may be processed if the customer requests it.

Check Delivery Status

  • Confirm the delivery status by checking the carrier portal.

Create Return Process in Fluent Commerce

  • Click the three dots in the top-right corner → Select "Create Return with Refund".

​

Fill Out the Return Form

  • In the return form, you may fill out blank fields with a dash (“-”). Since it’s a retailer order, the form will auto-fill those fields.

    • The fields Company Name, Street, City, Postcode, and Email are automatically filled and you can leave them as they are.

  • Quantity of Items: Select the quantity of items to be returned by clicking the “+” icon.

​

​

  • Item Condition: Choose the appropriate condition:

    • Packed - Never used

    • Packed - Used/Re-packed

    • Unpacked

​

​

  • Return Reason: Select “Other Reason”.

  • Comments: Add the Retailer's name in the "Comments" section.

  • Click Submit on the top right corner.

Complete Return + Refund

  • Open the order in FluentCommerce;

  • Go to the “Related” tab;

  • Click on Return Order Ref;

  • Click on the Mark Return Organised to proceed with the refund in the top right corner → Confirm.

​

​

​

  • Tick the Mark Complete box → Yes;

  • Add a Comment on the order page to document the return and refund.

  • Close your Sprinklr case.
