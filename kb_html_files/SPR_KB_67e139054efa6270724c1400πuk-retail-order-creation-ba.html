Mirakl

This SOP provides step-by-step guidance on how to effectively use the Mirakl portal for managing orders, processing returns, handling incident reports, and responding to customer messages. This platform is currently in a testing phase.

​

Logging into Mirakl

Log into Mirakl periodically to monitor new orders, incidents, and messages.

Accessing the Portal

  • Open the Mirakl login page in your browser.

  • Enter your credentials (username and password).

  • Click "Login."

  • Verify if you have the correct access level.

    • If you do not have the necessary permissions, contact your Supervisor.

​

​

Dashboard

Dashboard Overview

  • The dashboard provides a summary of:

    • Orders and their statuses

    • Returns requests

    • Incident reports

    • Customer messages

    • Overall activity

​

​

Order Management

Viewing Orders

  • Navigate to the "Orders" section.

  • Use the search bar to find an order by:

    • External Order Number

    • Customer Name

    • Status

  • Select an order by clicking on the Order ID.

​

​

Order Statuses

  • Awaiting Shipment: Order has been placed and needs processing.

  • Shipped: Order has been dispatched.

  • Delivered: Order has been successfully received by the customer.

  • Cancelled: Order has been voided.

Processing an Order

  • Orders are now integrated directly from Mirakl to Fluent Commerce.

  • Verify the order details.

  • Ensure stock availability.

  • Proceed with fulfillment in Fluent Commerce.

  • Update the order status in Mirakl if necessary.

​

​

Cancelling an Order

  • Verify that the order is still in "Booked" status in Fluent Commerce.

  • Cancel the order in Fluent Commerce.

  • Update Mirakl to reflect the cancellation.

Refer to this article for detailed Cancellation process instructions.

Incident Reports & Messages

What is an Incident Report?

Incident reports are messages or complaints submitted by customers, requiring attention within 48 hours.

Responding to an Incident Report

  • Open the Customer Care → Incidents section.

  • Select the incident you wish to analyze and reply.

​

​

  • Go to the “Unread Messages” session.

​

​

  • Identify the request type (e.g., return request, cancellation request, duplicate order issue).

  • Check the order details in Fluent Commerce.

  • Select the recipient of the message and provide an appropriate response based on the issue.

​

​

  • Mark the incident as "Resolved" once completed.

Accessing Messages

  • Navigate to the "My Messages" section.

  • Filter messages based on priority and status.

  • Review customer inquiries.

Answering Messages

  • Check the order details in Fluent Commerce.

  • Select the recipient of the message and provide an appropriate response based on the issue.

​

​

  • Mark the incident as "Resolved" once completed.

Messages and Incident reports must be handled within 48 hours.

  • Mirakl sends notifications for new incident reports and customer messages.

  • Check Outlook regularly for Mirakl-related notifications.

​

Processing Returns

Handling Return Requests

  • Check the return request in Mirakl.

  • Manually organize the collection for the return.

  • Process the return in Fluent Commerce.

  • Once the collection is completed, update Mirakl so that the refund can be processed automatically.

Note: Refer to this article for detailed Returns process instructions.
