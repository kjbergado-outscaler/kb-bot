Order Cancellation Process​

​

Cancel in Fluent Commerce (FC)

  • Orders must be cancelled before they are dispatched to Fulfillment Center (FC).

  • Go to the pending order → Click "Cancel order item" → Input quantity and reason → Submit.

  • Add the cancellation information in this tracker (Cancel Sheet): MKP CS TO OM request for channel engine support.xlsx

If cancellation in FC is not possible, CS must fill out the Cancellation Request File and notify Shimul.

Cancellation Request File

  • Access the file here;

  • Use this file when cancellation cannot be completed via FC and notify Shimul;

  • He will then proceed with the cancellation.

  • Fields to fill:

    • Order number

    • External order number

    • SKU to cancel

    • Fulfillment reference

    • Return to sender status

    • Tracking number

    • Carrier code

    • Reason for cancellation

  • Color Codes:

    • White: Not processed

    • Orange: Ticket raised

    • Green: Completed

Note: If an order has multiple items that need to be cancelled, fill out one line by Fulfillment/SKU.

​

  • Notify the Customer:

    • Customers must be informed that refunds are processed by Groupon.

    • Send a confirmation message through Groupon’s customer support interface.

​

Cancel the Order in the Groupon Portal

  • Search for the order using the External Order Number.

  • Click in the engine to open the order details.

  • Click Action > Cancel.

  • Confirm Pending for Approval Status:

    • Groupon must approve the cancellation request.

    • If the order is already shipped, cancellation is not possible.

Customer Support & Issue Resolution

  1. Access the Groupon Customer Support Portal.

  2. Check for new customer inquiries.

  3. Identify the nature of the issue:

    • Product Issues: (e.g., defective, wrong item received, missing parts).

    • Delivery Issues: (e.g., delayed, lost shipment, incorrect tracking).

    • Return & Refund Requests.

  4. Standard Resolution Process:

    • If a product issue is reported, request photos and additional details.

    • If a refund is requested, confirm return status before approval.

    • If an order is missing tracking, retrieve details from Fluent Commerce.

  5. Update Customer on Resolution:

    • Respond via Groupon Support Portal.

    • If escalation is required, forward to Groupon Support Team.

Warranty

All Refurbished products have a 6 month warranty. New products have a 10 year warranty.
