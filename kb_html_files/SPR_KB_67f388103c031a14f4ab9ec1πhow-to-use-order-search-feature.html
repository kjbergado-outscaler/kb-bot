What is the Order Search Feature?

The Order Search feature enables you to quickly find and access cases associated with a specific order reference directly from the Sprinklr Care Console. This help you gain complete context about a customer's interactions and improve resolution times, regardless of the channel the customer used previously.

Sprinklr; Search; Order; Reference

Why Use Order Search

  • Quickly find all related customer cases.

  • Gain full context about previous customer interactions.

  • Improve efficiency and customer satisfaction.

How to Use the Order Search Feature (Step-by-Step

  1. Navigate to the Order Search Button
    In the Agent Care Console, locate and click the     Order Search button at the top right.

    ​

  2. Enter the Order Reference
    In the pop-up window, enter the customer's order reference number and click     Next.

    ​

  3. Review the Related Cases
    A list of cases related to the entered order reference will appear, including information like:

    - Case Number

    - Creation Date

    - Channel

    - Initial Channel

    - Market

    - Status

    - Category

    ​

    ​

  4. Select the Relevant Case
    Click the radio button next to the case you want to view in detail.

    ​

  5. Open the Selected Case
    After selecting the relevant case, click     Next to open it directly within the Sprinklr Care Console.

Note: You will see cases from all markets but can only open cases for markets assigned to your profile.
