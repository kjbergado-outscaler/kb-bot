This SOP outlines the processes for handling returns, cancellations, and other customer service tasks related to Home24. Since there is no predefined process, we apply similar steps used in B2B operations while adapting to Home24’s platform and workflow.

Login Information

Portal: Access here.

Use your personal credentials to access the platform.

Order Management

Viewing Orders

  • Navigate to the "Orders" section.

  • Use the search bar to find an order by:

    • External Order Number

    • Customer Name

    • Status

  • Select an order by clicking on the Order ID.

Order Statuses

  • Awaiting Shipment: Order has been placed and needs processing.

  • Shipped: Order has been dispatched.

  • Delivered: Order has been successfully received by the customer.

  • Cancelled: Order has been voided.

Note: The orders are integrated directly from the portal to Fluent Commerce.

Searching for the Order

  • Extract the external order number from the request.

  • Search for the order in Fluent Commerce.

Receiving and Identifying Messages

  • Requests appear in the "My Messages" section of the portal. These messages may come from either end customers or Home24 operators.

  • Select the message you want to answer.

  • This is how the message thread is shown:

Answering Messages

Roll the page down to answer the message.

  • When responding directly to the end customer, decide whether to CC the operator based on the nature of the inquiry:

  • If you respond directly to the operator, the end customer will not be cc’d in the message.

Important: If you open the message and close it, it will automatically move from Ungelesen (Unread) to “All”. Make sure to answer the message before closing it to avoid lost messages.

Processing Returns

  • If the item is within the 100-day guarantee, proceed with the return.

    • If the order is over the 100-day guarantee period and the customer reaches out about a claim, request pictures to analyze the situation.

  • If the customer is returning smaller products (e.g. pillows, toppers, blankets) or wants to return a mattress that is still in its original packaging and opts to deliver it to a parcel shop, generate the return label and send it to the customer.

  • For open mattresses, book pick-up with DHL.

Please refer to this article for detailed instructions on Return Label creation and to this article for DHL Pick-up.

  • After the customer returns the item:

    • Verify if GLS has accepted the package.

    • Book the return in Fluent.

    • Mark the return as complete.

    • Register the return in Home24’s portal.

Booking Return in Fluent Commerce

  • Click the three dots in the top-right corner → Select "Create Return with Refund".

Fill Out the Return Form

  • In the return form, you may fill out blank fields with a dash (“-”). Since it’s a retailer order, the form will auto-fill those fields.

    • The fields Company Name, Street, City, Postcode, and Email are automatically filled and you can leave them as they are.

Returned Item Details

  • Quantity of Items: Select the quantity of items to be returned by clicking the “+” icon.

  • Item Condition: Choose the appropriate condition:

    • Packed - Never used

    • Packed - Used/Re-packed

    • Unpacked

  • Return Reason: Select “Other Reason”.

  • Comments: Add “Home24” in the "Comments" section.

  • Click Submit on the top right corner.

Complete Return + Refund

  • Open the order in FluentCommerce;

  • Go to the “Related” tab;

  • Click on Return Order Ref;

  • Click on the Mark Return Organised to proceed with the refund in the top right corner → Confirm.

  • Tick the Mark Complete box → Yes;

Inform the Customer

  • Notify the customer that the return has been booked in the system.

    • Add the Operator in cc to let them know the Return is booked on our system.

  • Add a Comment on the order page to document the return.

Booking Returns in the Portal

  • Open the Order

  • Select Ruckerstattung (Refund);

  • Choose if you wish to proceed with a Full Refund (Volle Rückerstattung) or Partial Refund (Erstattung eines Teils der Bestellung)

  • Select the reason for return and quantity of products;

  • Make sure to keep the refund as All Amounts (Alle Beträge)

  • After you select the reason and quantity, the total amount will appear in the screen.

  • Click Refund Order (Bestellung rückerstatten).

Important: If the return is not processed correctly in Home24 due to any errors, request the refund manually via this link.

Processing Cancellations

Search for the Order in Fluent

Use the external order number to search for the customer’s order in Fluent.

Check the Order Details

  • Check if the external order number matches the one you copied.

    • Confirm that the customer is listed as Home24.

​Cancellation Options

  • You will be presented with two options:

    • Option 1: If you cannot proceed with the cancellation, you will see a message indicating that the items have already been sent to the fulfilment location.

    • Option 2: If cancellation is possible, you will be given the option to proceed with the cancellation.

​

Note: Usually we’re able to process the cancellation up until the Booked status. After that the option is most likely not available, but always make sure to check!

Start Order Cancellation

  • Click the three dots in the top-right corner of the order page.

    • Select Cancel Order Items.

If Cancellation is Possible

  • Cancel the order

  • Check the “Order Progress” tab, it should be marked as “Cancelled”.

  • Inform the customer: Notify the client that the cancellation has already been done in our system.

  • Add a Comment to the Order: Document the situation on the order page for visibility.

  • Cancel the order in the Platform.

Cancelling the Order in the Portal

  • Open the Order

  • Select Ruckerstattung (Refund);

  • You’ll be presented with the option to Cancel.

  • Select the items to be cancelled.

  • Click Refund Order (Bestellung rückerstatten).

Note: Visual examples pending.

If Cancellation is Not Possible

  • Inform the Customer: Notify the client that the cancellation cannot be completed as the order has already been shipped. Advise them to refuse the delivery so that the return process can be initiated.

  • Add a Comment to the Order: Document the situation on the order page for visibility.

    • To add a comment:

      • Go to the Comments section on the order page.

      • Click the three dots in the top-right corner, then select Add Comment.

      • Fill out the comment template with the relevant details and click Submit.

Note: If necessary, coordinate with the operator via the platform.
