Scope

This SOP outlines the Beter Bed Returns process. This task should be done every Monday, to ensure that all returns from the previous week are properly reviewed.

​

Access the Mailbox

  • Open Outlook and go to your Emma inbox.

    • In the top-right corner, click on your profile icon;

    • Click Weiteres Postfach Öffnen (Open Another Mailbox);

    • Enter the e-mail address returns@bettzeit-gruppe.de in the designated field;

    • Click Öffnen (Open) to access the mailbox.

Select the E-mail

  • Find and select the oldest email from the previous week.

    • Be sure to mark the email as Read.

  • Click on the attached email dropdown and select Vorschau (Preview).

​

Download the PDF

  • Download the attached PDF file.

This file is the Return Invoice.

​

Add Information into the Spreadsheet

  • Open this spreadsheet;

    • Copy and paste the following information into the spreadsheet:

      • Return Date

      • Order Number

      • City

      • SKU

      • Quantity

Note: Perform the above steps for each return order from the previous week.

​

Organize the e-mails

After adding the information to the spreadsheet, move the e-mail to the correspondent folder in the Mailbox.

​

​

Organize the PDF Files

  • After processing all returns, access this folder;

    • Move the downloaded PDF files into the corresponding monthly folder.

Congratulations, you’re all done!
