Introduction

When opening a Jira ticket for a Refund request at the AR D2C Portal, one key field that needs to be properly inserted is the refound amount, to ensure a quick and standard process we have created the following explanation for you.

Insert Refund amount:

  • Numerical Value only regardless of currency : Sample 220.23 / 1,600.21 / .10 for Centavos 

  • Use "," to indicate Hundreds, Thousands value  

  • Use "." to indicate Centavos  

  • No words, no percentage in Amount Fields
