Managing Orders

Navigate to the 'Orders' section:

  • Click on 'Orders' in the left sidebar of the eBay Seller Hub.

  • Select 'All Orders' to view the complete list of orders.

​

​

Search for Orders:

  • Use the eBay Order Number provided by the customer to locate the order.

  • Orders can also be found using the Order Number, Customer Name, or Postcode.

Handling Delayed Orders:

  • If an order is delayed, contact the customer to check if they still want to proceed or if they prefer to cancel the order.

  • Document the customer's decision and proceed accordingly.

Managing Messages

Accessing the Inbox:

  • Go to 'Messages' in the top navigation bar.

  • ONLY check messages under 'From Members'.

  • Do not access messages 'From eBay', as these are managed internally by eBay.

​

​

​

Responding to Customer Inquiries:

  • Keep interactions simple.

  • Request the eBay Order Number for verification.

  • Avoid sending external links or images unless strictly necessary and permitted by eBay’s policy.

Cancellations

​

Cancel in Fluent Commerce (FC):

  • Orders must be cancelled before they are dispatched to Fulfillment Center (FC).

  • Go to the pending order → Click "Cancel order item" → Input quantity and reason → Submit.

  • Add the cancellation information in this tracker (Cancel Sheet): MKP CS TO OM request for channel engine support.xlsx

If cancellation in FC is not possible, CS must fill out the Cancellation Request File and notify Shimul.

Cancellation Request File

  • Access the file here;

  • Use this file when cancellation cannot be completed via FC and notify Shimul;

  • He will then proceed with the cancellation.

  • Fields to fill:

    • Order number

    • External order number

    • SKU to cancel

    • Fulfillment reference

    • Return to sender status

    • Tracking number

    • Carrier code

    • Reason for cancellation

  • Color Codes:

    • White: Not processed

    • Orange: Ticket raised

    • Green: Completed

Note: If an order has multiple items that need to be cancelled, fill out one line by Fulfillment/SKU.

Returns

  1. Customer submits return request on eBay portal.

  2. Create Return on Fluent Commerce.

  3. Add the Returns information in this tracker (Return Sheet): MKP CS TO OM request for channel engine support.xlsx

  4. Log into eBay:

    • Dashboard → "Returns" or "Return Requests" under Tasks.

    • Click “See Details” under "Action".

CS provides return label

  • Click "Confirm you sent a label" and then click "Continue".
