4. Return (Trial Period request & Claim outcomes)


In our order management system, there are two circumstances where you may need to return an item.

1. It will be a return due to a claim - Logistic or product claim (mentioned above).

2. It will be a return request under the trial period - where you can review the details here


Either way, while triggering the return in the OMS you should always ensure you are providing the proper reason category related to it:

5. Unable to deliver: "Others" (Exceptional Return reason)


There are scenarios that we are unfortunately, not able to deliver the item to the customer and the item is returned to the warehouse (address was not found, customer rejected the order, etc).


Please be aware this situation is not a return, as the order was never received by the customer and we are not triggering a return to pick up an item. This is an unable to deliver scenario.



If you need to trigger a refund to the customer, exceptionally, in Magento, you will trigger a return by using Other reasons and adding the relative comment as per description bellow.



Return reason
Mandatory commentScenario
