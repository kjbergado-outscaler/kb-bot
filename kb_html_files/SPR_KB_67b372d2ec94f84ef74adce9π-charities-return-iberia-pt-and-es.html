Collections through charities are only available for unpacked products: mattresses or bundles without beds.

Step-by-Step process

  1. Create the Return in the OMS adding "CHARITY" on the Carrier field.

  2. Send an email to the charity requesting the return, and that the pick-up should be done within three (3) days.

  3. Once confirmation from charity is received, inform customer.

  4. If no reply from charity after 2 days, close the child ticket, fall back to using a normal carrier. See the instructions in this article.

  5. Check return status on planned pickup date / after 2 days since last charity confirmation.

  6. Charities should send the proof of pick-up: https://emmasleep.sprinklr.com/care/knowledge-base/article/66e01d0e6786193b8ead4c62/view?returnTo=%2Fcare%2Fknowledge-base%2Fcategories%2F66e01d0d6786193b8ead4c5c

  7. If charity does not pickup in due time:

    1. Inform the charity that we will close the donation with them.

    2. Create a return with a carrier using the instructions in this article.

How to ensure the Refund gets processed?

Keep in mind: that the process below is regarding Returns inside the Trial period!

When should we start the refund process?

Once the customer reaches out to you informing the product was picked up, we only require one piece of confirmation! The pickup confirmation document.

If the pickup was done by the charity: They will provide a signed document confirming the pickup. (Only way to confirm)

If the pickup was completed, and you have the pickup confirmation please proceed with the refund, by marking the Return as Complete in the OMS.
