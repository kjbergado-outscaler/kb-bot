Bracchi Portal

Return Label Creation

  • Navigate to "SDA - Create Labels".

  • Select New Shipment.

  • Enter the following information in the Sender Data field:

    • Company name

    • ZIP code

    • City (ensure the province matches the city, or the label will not be saved)

    • Contact phone number

    • Reference field: Enter the Fulfillment Number

    • Shipping reference: Enter the Fulfillment Number again

    • Number of packages

    • Weight (refer to the provided weight list)

  • In Recipient Data:

    • Click "Recipient Registry"

    • Click Select after the pop-up window appears. This will auto-fill Emma's information.

  • Click Confirm Shipping.

    • The system will generate a PDF file containing a barcode.

    • Send the label to the retailer.

​ Note: Is takes 10-20 seconds for the barcode to be generated.

Retailer Self-Pickup Booking

  • Retailers may book pickups themselves using this link, or the process can be done manually within the Bracchi Portal.

Booking Manual Pickup

  • Navigate to SDA - Request Pickup > New Pickup.

  • Enter retailer details:

    • Company name

    • Contact person

    • Address (make sure the province matches the city)

    • Contact phone number

  • Input availability details:

    • Morning opening and closing time

    • Afternoon opening and closing time

    • Preferred pickup date (allow 2-3 days buffer for reliability)

  • Enter package details:

    • Number of parcels

    • Weight, Height, width, and depth

    • If the pickup is bulky, specify it

  • Confirm and book the pickup.

    • The system will generate a PDF file containing a barcode.

    • Send the label to the retailer.

Note: After the pick-up is confirmed, you'll receive a reference number with this format: "CP10XXXXX". Make sure to take note and save this number for tracking purposes.

Tracking and Monitoring Pickups

All pickup bookings are manually tracked on this spreadsheet. After booking, update the tracking file with:

  • Order number

  • Pickup request date

  • Retailer name

  • Fulfillment number

  • SKU

  • Pickup status

Conditions for Accepting a Return with Refund

  1. Product Claim:
    A return with refund can be processed if the customer requests a refund instead of a new product.

  2. Incorrect Delivery:
    If there has been a wrong delivery, a return with refund may be issued upon the customer’s request.

  3. Undelivered Mattress (as per Bracchi tracking):
    If the mattress is marked as delivered by Bracchi but the customer has not received it, a return and refund can be processed upon customer request.

  4. Failed Delivery (Returned to Sender):
    If the mattress was unable to be delivered and has been returned to us, a return and refund may be processed if the customer requests it.

Fill Out the Return Form

  • In the return form, you may fill out blank fields with a dash (“-”). Since it’s a retailer order, the form will auto-fill those fields.

    • The fields Company Name, Street, City, Postcode, and Email are automatically filled and you can leave them as they are.

  • Quantity of Items: Select the quantity of items to be returned by clicking the “+” icon.

​

​

  • Item Condition: Choose the appropriate condition:

    • Packed - Never used

    • Packed - Used/Re-packed

    • Unpacked

​

  • Return Reason: Select “Other Reason”.

  • Comments: Add the Retailer's name in the "Comments" section.

  • Click Submit on the top right corner.

  • Go to the “Related” tab;

  • Click on Return Order Ref;

  • Click on the Mark Return Organised to proceed with the refund in the top right corner → Confirm.
