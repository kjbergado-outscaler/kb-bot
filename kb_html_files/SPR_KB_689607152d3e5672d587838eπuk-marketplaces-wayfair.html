Scope

This SOP outlines the step-by-step process for handling returns from Wayfair via Wayfair Partner Home.

Frequency

This task should be performed at least twice per week (e.g., on Monday and Friday)

Handling Returns

Currently, it is not possible to create returns for Wayfair in Fluent Commerce. Once the feature becomes available, this SOP will be updated accordingly.

Log in to the Wayfair Portal

Portal URL: Wayfair Parner Home​

  1. Open Wayfair Portal and select “Finance” on the left tab

    ​

  2. Under Finance select “Accounts Receivable” and then “Deductions”

    ​

  3. At the top, select 'Filter,' then in the filter options, go to 'Deduction Type,' choose 'Return,' and apply the filters

    ​

  4. After that, you’ll be able to see all the return requests we currently have open

    ​

  5. Then you need to click on each PO number to see the reason for each return

    ​​


    Based on the reason, you should either 'Confirm Deduction' or 'Dispute' it. See below the list of possible reasons and how to handle each one:

    Return Reason

    • Change Mind (Accept)

    • No longer needed (Accept)

    • Warranty (Accept)

    • Quality Concerns (Dispute)

      • Dispute Reason

        • Out of Policy (x)

      • Description (required): This should be treated as an incident, not a return.

    1. Every time you accept a return, please fill in the required information so that the OAM or OM can generate the return in Channel Engine. The fields marked in yellow are the ones that need to be added in the Excel file MKP CS TO OM request for channel engine support.xlsx (Tab: Return Wayfair)
