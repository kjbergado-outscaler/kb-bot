Overview

While FluentCommerce is our system used for most orders, Magento is still utilized for older cases, particularly warranty requests.

Home Page

This is the first page you’ll see when you login to Magento. From here, you can access the Sales tab and adjust the language settings (we recommend using English, as it ensures the best interface performance).

​

Accessing "Orders" Page

  • Click “Sales” in the main tab → Orders.

​

​

When accessing orders in Magento, the interface presents a list of orders, which can be searched and sorted based on multiple information, such as order number, external order number, customer details, and shipping information.

​

Searching for Orders

You can search for orders in Magento using multiple criteria:

  • Order Number: Emma’s order reference.

  • External Order Number: Partner’s order reference.

  • Email Address: Customer's email address.

  • Billing Name: Name associated with the billing details.

  • Shipping Name: Name associated to the shipping address. This can be helpful when the billing and shipping names differ.

The fields you'll use most frequently are the Order # and External # fields. After filling this information out, click Search.

​

​

Orders in Magento specifically for Germany begin with the number "10". E.g. “10345555”.

Order Page Overview

​

​

 

​

  • Order Information: Here you can see the order number, purchase date, status and related orders.

  • Billing Address: This is an additional reference number that may sometimes be important.

  • Shipping Address: This is de delivery address and customer’s information.

  • Account Information: On the right side of the order screen, you can see account details such as the Retailer’s name (e.g., Otto) and email address.

​

​

​

By clicking on the account name, you can access the account details and view related orders.

​

  • Comments: The comments section is used for tracking important the interactions. So, every time the retailer contacts us about any topics, we need to add a comment to the order. These comments can be highlighted and marked as important so all agents can see the note when they access the order.

  • Product Details: The SKU and product details will be displayed for each order, and you can also view the tracking number for shipped items.

​

​

If you click on the SKU, you’ll be able to see the product’s status in the “Log” page.

​

Adding Comments

To add an important note, write a comment (e.g., "Product replaced three times, please don’t exchange this mattress again") and mark it as "important." This will cause the comment to pop up at the top of the order for easy visibility.

​

Marking the Comment as “Important”

  • Click on the drop-down menu beside the comment.

  • Select Important.

​

​

This way, this comment will be easily visible by all agents working on the order.

​

Creating a Supplement

Access Supplement Orders

  • Click on Supplement to access the supplement order creation screen.

Add Items to the Supplement

  • Enter SKU: In the supplement order overview, locate the SKU field.

  • Price Adjustment: Enter the price of the product as 0. We never charge customers for replacement orders.

  • Update Items and Quantities: Click on the Update Items and Quantities button to apply changes.

​

Removing Items

  • If there are any unwanted items in the supplement, select the item and click Remove. Then click Update Items to remove it.

Adding New Products

  • Search for Products: Click on Add Products and search for the product you need by SKU or name. For example, to search for a specific cover, enter the SKU or product name.

  • Select Product: Choose the appropriate product and select the quantity needed.

  • Add to Order: Once selected, click Add Product to Order to include it in the supplement order.

  • Verify Prices: Make sure all products are listed with a price of 0.

​

​

​

​

Shipping Details

  • Confirm Shipping Address: Ensure the shipping address is correct. This information should never be changed unless there is a necessary adjustment (e.g., changing the recipient’s name/delivery address).

​

​

​

You should never change the billing address information.

​

  • Shipping Method: Select the Versandkostenfrel (Free Shipping) option at the top. This is the default and should always be applied.

​

​

External ID (Optional)

  • External ID: In most cases, leave the External ID unchanged. However, if required by a retailer, you may modify it.

​

​

Order Information

  • Order History: Enter a description or reference (e.g. "Cover Request"), to clarify the reason for the supplement order.

  • Once all fields are completed, click Submit to create the supplement order.

​

​

Review the Order

  • The supplement order should now be created and visible. The status will indicate whether the order has been processed and whether a barcode has been generated.
