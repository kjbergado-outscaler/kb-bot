Scope

This SOP was created to outline the procedure for handling self pick-up orders upon request from the store manager. This applies to all customer service (CS) agents involved in self pick-up orders requests.

Instructions

  • The store manager contacts the CS team to change a barcode as a self pick-up order.

  • CS checks if the order has been transmitted to the Warehouse.

If the Order Has NOT Been Transmitted to WH:

  • CS raises a ticket with the BO team.

  • In the ticket, CS requests the barcode to be marked as "Delivered in Store".

If the Order HAS Been Transmitted to WH:

  • CS creates a supplement order with a custom delivery date set to 1 month later to prevent automatic re-transmission.

  • The agent raises a ticket with the BO team, requesting the supplement order to be marked as "Delivered in Store".

  • CS contacts the carrier and requests the original order to be returned to the warehouse.

  • CS also contacts the customer and asks them to refuse delivery of that specific item.

Notes

  • Always document the communication and ticket reference number in Magento.

  • Confirm action taken with the store manager once resolved.
