Introduction


Make sure to start your reading about Data Breach by the article Data Breach: What it is & Best practices. In this current article, you will find information about how to act in cases of possible data breaches.


Cases examples of Data Breaches


  1. Misaddressed Email: An agent sends an order confirmation to the wrong customer, revealing order details and personal information.

  2. Incorrect Attachment: An agent attaches a file containing another customer’s order history to an email.

  3. Unauthorized Access by Former Employees: A former employee still has access to the email system and customer information, which they use maliciously.

  4. Data Entry Error: An agent mistakenly enters another customer’s information into a customer’s account, leading to a mix-up of orders and personal details.

How to handle potential data breaches


Check below the mandatory procedure to be followed in case you discover a potential data breach:


  • Please report it immediately, by sending an e-mail to: privacy@emma-sleep.com.


ATTENTION: Make sure to provide the Legal team with a clear and detailed description of the situation using the list of questions:

  • What happened?
  • When did it happen?
  • How did it occur?
  • Who was involved and affected?
  • What actions were taken?
  • When was the data breach detected and reported?
  • Other relevant tickets, documents, information.
