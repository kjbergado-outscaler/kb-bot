Instructions

Scope

This SOP outlines the process for managing order tracking, delivery status, cancellations, and obtaining proof of delivery (POD) using the UPS Carrier Portal.

Instructions

Accessing the UPS Portal

  1. Log in to the UPS Portal:

    • Portal URL: UPS Tracking

    • No Access needed for UPS.

  2. Navigate to the Tracking Section:

    • Enter the tracking number obtained from Magento.

​

​

Order Tracking and Status Check

Search for an Order:

  • Paste the tracking number into the Search field.

Review the Order Status:

  • Possible statuses:

    • In Transit: Package is moving to the destination.

    • Delivered: Package has reached the recipient.

    • Out for Delivery: Package is on the way for the final delivery.

​

​

View Delivery Details:

  • Click on View Details to access the delivery timeline and specific delivery attempts.

Managing Delivery Actions

  1. Request a Delivery Change:

    • If a delivery needs to be re-routed or held, select Change Delivery.

  2. Obtain Proof of Delivery (POD):

    • Click on Get Proof of Delivery, and if available, download the POD document.

Locating Consignment Numbers in Magento

Find the UPS Tracking Number:

  • Magento displays the UPS tracking number, which is used for searching orders in the UPS portal.

​

​

Search by Tracking Number:

  • Input the tracking number in the UPS search field to retrieve order details.

Common Issues & Solutions

Issue

Solution

Tracking not found

Double-check the tracking number in Magento.

Parcel status is 'Exception'

Review the exception details and contact UPS support if necessary.

Delivery Change Request Failed

Ensure the address is correct and the package is not yet out for delivery.

Proof of Delivery not available

Check if the parcel status is Delivered, then look again for POD.

Parcel Returns via UPS and B&A Quilting

EMMA Customer Service adds return details to the shared file. From this information, we create the requested return shipment via our UPS portal. This process generates an electronic return label, which is automatically transmitted to UPS, the UPS driver will then collect the parcel. Once the parcel has been collected, the product is returned to B&A.

​

Returns must be requested through UPS if the product is unopened.

​

Instructions

  1. The client initiates a return request.

  2. The B2B client shares the return information with Emma’s customer service agents.

  3. A customer service agent records the return request details in the shared file with B&A Quilting. The link to the file that needs to be filled: EMMA Returns Master File 2025.xlsx
    Note: Please ensure that no fields are deleted or changed and make sure all fields are checked again before saving the file.

  4. B&A Quilting compiles the data and creates a return booking on the UPS portal.

  5. UPS collects the returned products from the customer's address.

  6. UPS delivers the returned products to B&A Quilting.

  7. B&A Quilting refurbishes the received products.
