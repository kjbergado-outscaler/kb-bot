How to log in into Bol.com


To log in into Bol.com, you should go to Bol.com.



In the banner click on 'inloggen'




Then fill in the credentials of the account you wish to log in with.



Then on the left side, press 'verkopen'.



For security reasons you have to log in again, after that you will get into the portal.


Bol.com Dashboard Overview





The screenshot above is the most important thing you need to check. You see 3 main topics:


  1. Openstaande Klantvragen (open tickets)

  2. Openstaande Bestellingen (open orders)

  3. Openstaande retouren (open returns)


Open Tickets






When you open up the tickets that the customer has sent to us, you will see a list like the screenshot above.

Click on one of the tickets to open it and read the conversation.



A conversation should look like this. Where the black messages are from the customer, the blue messages are from a customer service agent of Bol.com.




On the bottom of a conversation you have the following options:


Klant e-mailen

If you want to reply to the customer, you can press the top option, which is to answer the customer.


Als opgelost sluiten

In some cases you can use this one, for example when a customer sends back a 'thank you' message. Only use this when there is no reason to reply anymore and when the problem or question is solved.


Notitie voor jezelf toevoegen

Use this if you need to add a note for yourself, but please keep in mind that you need to close the case within 8 business hours.


Hulp nodig bij klantvraag

If you need help from Bol.com, for example when a return was closed automatically and it needs to be reopened again.



Open orders



When you go to open orders you will find a list of orders and it looks like the one above. You don't have to do much in here, except for checking if there is a cancelation request.


Cancelation request


In the screenshot above you can find a cancelation request. You can find it under status and it will be written in orange letters.

If you are still able to cancel the order in Magento and you can confirm its been canceled, then press on the 'Annulering accepteren' button on the right side.



You will come in a confirmation screen. In there you can press the blue button with 'Ja, annuleer' to confirm the cancelation.

Note: As soon as you press the cancelation button, the order will be refunded by Bol.com.



Open returns


In the open returns overview you can see all the open returns. When a customer request a return he should do it trough Bol.com and not trough us. We simply can't create a return at Bol.com.

It is important that you check the new returns every day. So for example, if you do this daily, you only have to check the new returns that came in since the last time you checked. Also, you need to check ALL the returns in Magento 2 or 3 times a week.


First we will handle the new returns at Bol.com


New returns (daily)


So if you check the returns daily you only have to check the new returns since the last time. What you need to do is the following:


First you need to create the return in Magento. To do so, copy the order number from Bol.com and sears for it in Magento. You can fill in the order number at 'external' with 2 asterix's, like this:


If you found it, then open op the order.



First thing you need to do is to copy the email address from Retail Customer E-mail and paste it into the Email field, like above.

This is a encrypted e-mail address due to privacy reasons. So it looks weird and long, but you can use this to send an e-mail to the customer. In this case we use it to send the return label or to send the email address to VOS, so they can reach out to the customer.

After changing the email address, please create the return. Do not do this the other way around.

After the return is created you have to wait until the package came back to us.


Checking all the returns (2 or 3 times a week)


You should do this like 2 or 3 times a week, to save yourself some time.

What you need to do is to check every single order in the 'return' list of Bol.com to see if the package came back to us already.


What you need to do is to check if the return came back to us in Magento.


So first copy the order number of the return in Bol.com and go to Magento.


In Magento



Go to the middle of the page where you can find the returns at 'Product Returns'. Click on the blue number which is a link. This is a unique number for the return.



You will come in the following screen as in the screenshot above. You have this per product, but every product should work the same.


Status

Expected - The product is still on its way, and cant be refunded by Bol.com

Complete - The product is back at our warehouse and could be refunded, see below.


Kunden-Erstattung

Pending - Not refunded yet, its either waiting for the product to come back or the product just came back and its not processed yet.

Refunded - Product is refunded. In this case we will refund to Bol.com



Return complete in Magento


When the return is complete in Magento, you should complete the return in Bol.com as well so they can start the refund for the customer.


When a return is complete you have to press on the 'Behandelen' on the right side in the overview of the Bol.com returns.



You will come in the following screen. In most of the times you click on 'Retour goed ontvangen' and then on 'ja, behandel retour'.
Note: in some cases you have to fill in the amount of products that came back. This is only when there's a return for multiple products in the same order.
