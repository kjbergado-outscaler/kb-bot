Bed Spare Parts SOP (Standard Operating Procedure)


What is a "part", "spare part" or a "component"?

It is the detachable component that can be assembled and form a bed.


After receiving an Emma Bed, it can (unfortunately) happen that some customers receive it with missing or defective parts. If this happens, the customers might contact our Emma CS team to ask for help.


In order to provide a smooth and effective solution to the customer, we need to deliver the parts by creating a supplement order with the related SKU number, as per the SOP below:


Spare Part Handling Process


1. Identify the Part: Understand which part is needed by the customer and use the Searching Tool for Spare Parts to identify the SKU.


2. Complete Spare Part Requests: Take action considering the current scenario (Spare Part in or out of stock? Find the information on how to act for each scenario below.)


3. Record in the OMS (Magento or Fluent Commerce): Leave a comment/note.





Spare Part Process


There are 2 possible outcomes from a missing or damaged part: 1) send only the damaged part to the customer or 2) send the box that has the part. Preferably, the 1) should be chosen. However, in some cases, there aren't spare parts registered for some specific bed or the part is not treated as spare part. For those cases the solution is to order only the box where the part is sent.

In case there is no stock of the spare part, the option 2) should also be taken.

How to: Spare Parts Tool


The instruction manual that Emma developed is quite user-friendly, let’s take advantage of it and you can see all the numbering of each part. That’s all you need!



When opening the document, you will automatically access the Menu where there are buttons to each bed. The SKUs will be found following the steps below.


Step 1.Identify the Part ID


The part ID can be found in the booklet or in the spare parts tool. To find the part ID in the spare parts tool, click the button for the respective bed. It you automatically take you to the bed design with the parts ID.



Step 2. Search spare part SKU


To find the spare part SKU click on the “Search Spare Part SKU” in Menu, it will take you to a search engine with some filters. It is important to follow all the instructions there. They are:

i) Make sure every filter is disabled

ii) Select the bed model

iii) Click just in the bed size filter you are looking for

iv) Select the part number

v) Check the spare part SKU

vi) Clear all filters


If the spare part cannot be found in the search tool, the box where the part is shipped must be sent to the customer.



Step 3. Check stock availability


The stock availability can be checked when creating the supplement order. If the part does not appear when searching for the spare part SKU, it means that there is no stock available. In this case, the box where the part is shipped should be sent to the customer.


Step 4. Identify the box SKU (Only if out of stock or no spare part SKU)


Click the button for the respective bed. It you automatically take you to the box configuration. Find in which box the part ID is inserted. Then, use the “Box SKU logic” to have the box SKU that will be ordered.



Step 5. Create supplement order


The spare part order or component order should be registered as a supplement order. To place the supplement orders, use the specific articles available in the Knowledge Base: How to: Supplement Order.
