How to identify an order from any Emma store?

Even though all Emma Stores orders are registered in Magento by the Order Management team, purchases in the store happen on Shopify. That's why order numbers provided by customers might have a different format than usual. We need to recognize both and identify if the order was placed at an Emma Store.

â€‹

So, each store purchase has:

  1. Shopify format: XX1234 (country code followed by 4 digits â€“ DE, NL, PT, ES, UK)

  2. Magento format: XXXXXXXXX (9-digit number).

â€‹

Old order number format for Shopify: #1234 (older store customers might provide you this format)

New order number format for Shopify: UK1234 - NL1234 - ES1234 - DE1234 - PT1234


â€‹

â€‹Note: The first order number customers received was the Shopify one, so that's the one that they will probably provide. â€‹
