Cancellation Options

  • You will be presented with two options:

    • Option 1: If you cannot proceed with the cancellation, you will see a message indicating that the items have already been sent to the fulfilment location.

    • Option 2: If cancellation is possible, you will be given the option to proceed with the cancellation.

​

Note: Usually we’re able to process the cancellation up until the Booked status. After that the option is most likely not available, but always make sure to check!

Start Order Cancellation

  • Click the three dots in the top-right corner of the order page.

    • Select Cancel Order Items.

If Cancellation is Possible

  • Cancel the order

  • Check the “Order Progress” tab, it should be marked as “Cancelled”.

​

​

  • Go to the “Storno/Return Lidl” group in Teams;

    • Send our internal order number (W-0001-XXXXXX) and the external order number (490XXXXX) with the information that it’s cancelled. Example:

“W-0001-XXXXXX - 490XXXXXXXX cancelled”

​

This ensures that our Lidl representatives can check if the order is cancelled in the PAP Portal.

  • Inform the customer: Notify the client that the cancellation has already been done in our system.

  • Add a Comment to the Order: Document the situation on the order page for visibility.

  • Close your Sprinklr case.

If Cancellation is Not Possible

  • Inform the Customer: Notify the client that the cancellation cannot be completed as the order has already been shipped. Advise them to refuse the delivery so that the return process can be initiated.

You can find e-mail templates here.

  • Add a Comment to the Order: Document the situation on the order page for visibility.

    • To add a comment:

      • Go to the Comments section on the order page.

      • Click the three dots in the top-right corner, then select Add Comment.

      • Fill out the comment template with the relevant details and click Submit.
