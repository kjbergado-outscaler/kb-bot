Find the Customer’s Order Number

Otto typically provides the customer's ID rather than the order number. Use this customer ID to locate the order in the Diva Portal.

​

​

Access the Diva Portal

  • Login to the Diva Portal.

  • Navigate to the "Aufträge" (Orders) section → Tagesauftrage

​

  • Change the field Tagesauftrage to "All."

  • Enter the customer ID in the Kunder-Nr field.

​

​

  • After entering the customer ID, you will see the corresponding external order number in the “Augfrags-Nr.” field.

Cancel/Return the order in the Diva Portal

  • ​Go to the Diva Portal → Select Retouren (Returns).

​

​

  • Set the Retouren filter as “all”;

  • Fill out the creation date → Search

 

​

  • Look for the customer's name;

  • Once you find the customer, tick the checkbox on the left to select them → Return Recorded.

​

 

  • Inform the customer: Notify the client that the cancellation has already been done in our system.

  • Close your Sprinklr case.

Booking Pick-ups in Diva Portal

Booking and Updating Packaging Materials

Instructions

  • Go to the Diva Portal;

  • Select Übersicht → Retouren (Returns).

  • Set the Retouren filter as “Offen” → Search;

  • You need to act on all orders that have empty information.

  • Locate the order in Fluent using the customer number. Follow the instructions provided on the first section of this article - Find the Customer’s Order Number.

Booking Packaging Material for Pickups (GLS)

  • Book the material shipment via GLS.

  • Save the tracking number and add it on Diva Portal.

  • Add the tracking number in the Comments section on Fluent Commerce.

Refer to this article for detailed instructions on how to book packaging material.

  • Enter the pickup request details in the Portal and confirm the booking:

    • Date of the booking

    • Order number

    • “GLS” + tracking number

  • Click Meldung Senden (Send Message) in the bottom-left corner → OK.

Booking Pick-up (DHL)

  • Proceed to book a pickup with DHL.

  • Update the information in the Portal.

  • Click Meldung Senden (Send Message) in the bottom-left corner → OK.

  • Add the tracking number in the Comments section on Fluent Commerce.

Refer to this article for detailed instructions on how to book pick-ups with DHL.

Tracking and Completing the Pickup

  • Check if the pickup was successful by tracking it every 2 - 3 days in DHL’s system.

  • Add a note indicating the pickup completion.

  • Click Meldung Senden (Send Message) in the bottom-left corner.

  • After the order is done, flag the order and click Send Message again.

  • Add a Comment on Fluent Commerce informing the process is done.
