IN EBAY AGENTS NEED TO ACCEPT RETURN REQUESTS AS A FIRST ACTION STEP

​

If customer change their mind and would like to return mattress (within 14 days or 100/200 nights period) ask if the mattress is packed or unpacked and reason for return. You can use macro Returns -Ask for Reason. If the mattress is Refurbished or Essential please remove wording whether is too soft and firm and that it takes 4-6 weeks to adjust to new mattress (as they have only 14 days to return the mattress and we don’t promote comfort layer offering with these types)

​

Unpack mattress return:

In Magento click on Return Products and create return and select “Full refund”. Don’t click on “Product back at warehouse”

​

Send customer standard macro for Returns- Out of box mattress-return process started

Packed mattresses (return label)

We create return label for mattresses which are still in box unopened (new Essential, Original, Hybrid).

Email customer and ask them when good time is to pick up the mattress (Mon-Fri from 2 days onwards). You can use macro Returns-Returns label-still in box-parcel packed.

Once you have this information please enter it in “Return Label” excel sheet.

If such option is not available, please email Stacey if order originated in B&A warehouse or sale carpenter if order originated in Carpenter. In email heading include tracking number, Magento order number and request details (Send order back to sender). In the email include customer information (name, address, phone number and email address) and product and quantity.

Once the collection date is arranged please ask customer to confirm when collection is completed so we can make sure refund will be send without delay. When the collection is complete please create Return Products in Magento click on “Full refund”, “Packed” and order “Product back at warehouse”. Advise customer refund should be processed within 3-10 working days to original payment method.
