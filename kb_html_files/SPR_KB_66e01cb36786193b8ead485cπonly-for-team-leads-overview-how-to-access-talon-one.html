Overview


In this article, you as a Sleep Experts agent will learn how to use the platform Talon.One to create voucher discounts for customers. This platform is the one responsible for the discount code creation inside COS.


Talon.One is a platform that allows the creation of customizable promotional campaigns and is supported by EmmaCOS.

The campaigns are created by Emma's Marketing team according to each country's sales strategy.


If you are a Sleep Experts agent that need to create a discount code on Talon.One, please check the article Discount code creation - Talon.One.





How to request access to Talon.One


IMPORTANT: ONLY FOR SUPERVISORS AND SLEEP EXPERTS AGENTS CAN REQUEST ACCESS TO THE TOOL.


The access cannot be requested for a group of people, and neither one account can be used for more than one person. Each person needs to request their own access via THIS FORM (click here).


1) You must log in with your Emma email address, and then select the market that you are working for:




2) Then briefly explain why you need access:



2.1) When using the form, you need to raise one request per country that you want to have access to:




3) Just click on "Submit" and wait. In less than 24 hours you will receive an email on your Emma account asking you to sign in.



4) Once you receive the email, you click on "Get started" to finish the account set-up.


IMPORTANT: After 24 hours the email will expire and if you didn't sign in before this, you will have to request access again.




5) In the next step, you will have to create a password for your Talon.One account. Please make sure to create a secure password and never share it with anyone else, not even inside your team. Each agent must have their own account and password.
