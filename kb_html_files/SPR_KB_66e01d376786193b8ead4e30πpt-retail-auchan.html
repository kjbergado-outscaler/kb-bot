Orders

Auchan orders are received for manual creation (with Backoffice) through 2 different e-mails.

​

1. Automatic e-mail with the Purchase Order. It contains store information, external order number and products ordered. It's not necessary to answer to this e-mail.

Example: [#8125593] Encomenda Auchan : Bettzeit GmbH (freshdesk.com)​

2. Store e-mail with the customer's data. It is only necessary to reply to this e-mail with the order creation confirmation.

​

Example: [#8125434] Encomenda 967617252 - 3391853 - COLCHÃO CASAL EMMA:BRANCO 150X200 EMAOR150200AAB - Cliente: CLAUDIO VARELA, 001191073_001 : Bettzeit GmbH (freshdesk.com)

​

We may receive 2 types of orders for Auchan:

  • Stores -> Delivered directly in the customer's house. An estimated delivery date can be provided.

  • ARP -> Delivered in Auchan Warehouse. An estimated delivery date can not be provided because the delivery is scheduled. Please always escalate to Emma before providing an ETA for this type of orders.

For this orders, you don't need to share the SKUs with the Backoffice, as they are already in the PO File.

​

Create Orders

Open the Retail Backoffice Portal and choose Order Creation -> Manual Order Creation -> Auchan

​

Please give the following information to the Backoffice team:

  • Account ID

  • External Number

  • Client’s Name

  • Client's E-mail -> Please always use the store e-mail (i.e, the e-mail of the person that sent us the e-mail with the customer information).

  • Delivery Address

  • PO File

​

After creating the request, paste the Jira number as an internal comment in the Freshdesk Ticket.

​

Wait for the order to be created by BO and pay attention to their reply, you will need to ask to Auchan a date and time for the delivery, inform backoffice and then confirm the order with the retailer. Then you can resolve.
