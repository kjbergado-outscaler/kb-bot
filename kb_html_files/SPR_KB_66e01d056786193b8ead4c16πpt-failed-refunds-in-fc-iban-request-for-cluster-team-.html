Context - Failed Refunds in Fluent Commerce


In the Portuguese market, due to a systemic error in Fluent Commerce, some orders are not able to be refunded automatically. Those orders are currently registered in an Excel Refund error list:


EXCEL REFUND ERROR LIST: South Europe - for bank transfer.xlsx




To be able to process with the refund, the Order to Cash team must process refunds via bank transfer.


Action Required


In order to proceed with the bank transfer, the agents will need to follow steps:


  1. Access the EXCEL REFUND ERROR LIST.
  2. Contact the customer in the Excel File and request their Full name and IBAN.
  3. Fill the EXCEL REFUND ERROR LIST with this information.



- This task should be assigned to an agent. The agent should check the excel file twice a week to guarantee there are no refunds pending. - Please do not save new excel files, stick to the same one aalways.
