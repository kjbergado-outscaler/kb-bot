This SOP outlines the process for handling after-sales support for Home Store and More (HSM) orders, which is based in Ireland (IE). It provides detailed step-by-step guidance on managing credit memos, processing faulty items, and communicating with relevant stakeholders.

This procedure includes:

  • Receiving and processing faulty item reports.

  • Managing credit memos for faulty items.

  • Communicating with Home Store staff.

  • Ensuring customers are directed to the appropriate contact points.

Receiving Faulty Item Reports

Reports are received primarily via email due to previous issues with Sprinklr. They must provide:

  • Customer’s name (if available)

  • Delivery address

  • Order number

  • SKU details

  • Photos and descriptions of the fault

Note: If the client fails to provide complete details, request additional information before proceeding.

​

Processing Faulty Mattress Requests

Check Photo Evidence

  • If valid photos of the faulty mattress are provided, approve the replacement or refund.

    • If insufficient evidence is given, request additional images.

  • You can find the claim pictures requirements and examples here and how to validate the claim here.

Approve Refund/Replacement

  • If approved, inform the store that they may process a replacement or refund.

  • If a refund is chosen, they will send a weekly spreadsheet for the credit memo.

Process the credit memo once the order status is changed to "Complete", never do it in “Booked” status.

Handling Product Claims

If a product is faulty and a claim is approved, no replacement is arranged by us. Simply approve their request and document the reference number.

Collection Policy: HSM does not return faulty mattresses. They are responsible for disposal per agreement with Inna.

Processing Returns

Fluent Commerce

  • Locate the corresponding PO number in Fluent Commerce.

  • Follow these steps:

    • Search for the BU key for HSM.

    • Locate the specific PO number.

    • Identify the mattress type (e.g., "Emmy Premium Hybrid Mattress King Size").

    • Process a return for the identified item.

Create Return Process in Fluent Commerce

  • Click the three dots in the top-right corner → Select "Create Return with Refund".

Fill Out the Return Form

  • In the return form, you may fill out blank fields with a dash (“N/A”). Since it’s a retailer order, the form will auto-fill those fields.

    • The fields Company Name, Street, City, Postcode, and Email are automatically filled and you can leave them as they are.

Returned Item Details

  • Quantity of Items: Select the quantity of items to be returned by clicking the “+” icon.

  • Item Condition: Choose the appropriate condition:

    • Packed - Never used

    • Packed - Used/Re-packed

    • Unpacked

  • Return Reason: Select “Product Claim”.

  • Comments: Add the end customer’s name in the “Comments” field.

  • Click Submit on the top right corner.

  • Flag “No pickup disposal by retailer” → Click Submit on the top right corner. 

Processing Credit Memos

Receiving Credit Memo File

  • Accounts team sends a summary file weekly.

  • File includes SKU details, order numbers, and quantities.

Verifying Details

  • Match order numbers and SKUs in Magento or Fluent Commerce.

  • Ensure SKU corresponds to the correct PO number.

Processing in Magento/Fluent Commerce

  • Locate the order in the tool.

  • Create a credit memo for the faulty item(s).

  • Generate the credit memo document.

Sending Credit Memo to HSM

  • Email the credit memo copy to the accounts team.

  • Confirm completion with Home Store staff.

Ensure all credit memos are processed within the week and make sure that all inquiries have been addressed.

Handling Comfort Layer Requests

Receiving Requests

  • Home Store may request comfort layers for customers.

  • Complete details (customer name, address, SKU, etc.) must be provided.

  • Add the information in this spreadsheet for better control.

  • Approve request based on warranty/coverage policy.

  • Arrange for dispatch if applicable.

Updating the Order Creation File

  • Record details in the internal order creation file.

  • Use the HSSCL (Home Store Comfort Layer) reference format.

  • Indicate whether the faulty item is covered or not.

Handling Customer Inquiries

If a customer contacts us directly, they should be directed to the store. Provide the store’s contact details and kindly as the customer to reach out to HSM.

Do not engage in direct customer communication unless explicitly authorized.
