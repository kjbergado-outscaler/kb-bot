Return Label Creation

  • Navigate to "SDA - Create Labels".

  • Select New Shipment.

  • Enter the following information in the Sender Data field:

    • Company name

    • ZIP code

    • City (ensure the province matches the city, or the label will not be saved)

    • Contact phone number

    • Reference field: Enter the Fulfillment Number

    • Shipping reference: Enter the Fulfillment Number again

    • Number of packages

    • Weight (refer to the provided weight list)

  • In Recipient Data:

    • Click "Recipient Registry"

    • Click Select after the pop-up window appears. This will auto-fill Emma's information.

  • Click Confirm Shipping.

    • The system will generate a PDF file containing a barcode.

    • Send the label to the retailer.

Booking Manual Pickup

  • Navigate to SDA - Request Pickup > New Pickup.

  • Enter retailer details:

    • Company name

    • Contact person

    • Address (make sure the province matches the city)

    • Contact phone number

  • Input availability details:

    • Morning opening and closing time

    • Afternoon opening and closing time

    • Preferred pickup date (allow 2-3 days buffer for reliability)

  • Enter package details:

    • Number of parcels

    • Weight, Height, width, and depth

    • If the pickup is bulky, specify it

  • Confirm and book the pickup.

Note: After the pick-up is confirmed, you'll receive a reference number with this format: "CP10XXXXX". Make sure to take note and save this number for tracking purposes.

Tracking the Pick-up

  • Navigate to SDA - Request Pickup

  • Click Pickup list;

  • Fill-out:

    • Date from: select the first and last day of the pickup month.

    • Reference: paste the Pickup reservation code (CP10XXXXXX);

    • Click Search.

​

You'll then be able to see the shipment details and return status.

  • Select the pickup you want to review.

Tracking Orders

  • ​Log into the Bracchi Portal;

  • Select Shipment Tracking;

  • Fill-out:

    • Date from: select the first and last day of the purchase month.

    • Reference: paste the Fulfillment reference
