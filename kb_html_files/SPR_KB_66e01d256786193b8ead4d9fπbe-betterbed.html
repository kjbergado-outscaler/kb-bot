What is better bed?


BeterBed is a big retailer in Benelux that sells Emma products.


Where do we receive BetterBed requests?


We receive the requests from BeterBed via our info@emma-matras.nl email (password available at Cluster second level or internally Marloes Scibelli). We receive around 2-3 emails per week with a file containing about 35 requests to investigate and handle.


The point of contact for BeterBed is Soraya Kusters (Soraya.Kusters@beterbed.nl / phone: 0031 413 33 88 67).


What type of requests are asked?

  • Delivery information
  • Comfort layers
  • New mattress covers (after receiving photos)


The order can be found under "External #" in Magento, or using the email address in the email field.


We do not create returns for BB orders, they do it themselves.

How to handle credit notes?


If Beter Bed orders have been cancelled, or returned, or if the customer never received the order (lost?), then a credit note can be requested to Ronald (see below)

Condition is, that an invoice has to be issued, unless it is a cancellation via the system, in which case an automatic refund should be issued and does not need to be requested.


Point of contact is Ronald Abram Vergara ronald.vergara@emma-sleep.com
