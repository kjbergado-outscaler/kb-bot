To follow up on that information you always have to access the order related to our Order Management system and find the related information as per the What & how to related article.

Edit Delivery Information
Magento:

Up to Packing Status

​

If the request to change any delivery information arrives with the status still in "Packing", it means the order was not yet sent to the WH. We can make modifications right on the OMS, since it's going to update the information to send to the WH in a near future.

​

You can proceed with modification as per Magento: what and how to article

​

After Packing status

​

If the request to change any delivery information arrives after the "Packing" status, it means that the order was already sent to the WH. In this case, we'll need to contact the WH or Carrier (depending where the order is) so they can update the delivery information on their hand.

​

Please note: Always leave a note with the new address or scheduled time requested by the customer as in case the Carrier company reaches us, we are able to collect information easily.

Change Product/ Size/ SKU

In retail, even if the price can be the same, there's no direct change of products because of invoicing.

Due to that, whenever customers reach us to change the product or size (SKU change), the process to do it will be to cancel the order and the customer place a new - corrected one.

​

Change Delivery Date

The customer may prefer to receive the order at a later date. This option is only available before it is submitted to the WH.

In Magento, there is the option to “Set delivery date.”

In Fluent, it is “Expected Delivery date,” which has an option to edit.

The process is to cancel the barcodes/fulfillment, and after cancellation, create a supplement with a future delivery date and inform the retailer. This is the warehouse order transmission date, so we should set the date 2 or 3 business days before the desired date for all items, and 10 days for the canapés.

If the order is in the warehouse, we cannot ask them to deliver on a future date. They cannot store or hold products. In this case, we need to request the order cancellation and create a supplement with a future delivery date as indicated above.

 

Change Delivery Address

When the order still appears as rejected, the address can be changed in Magento, and there is no need for a JIRA to WH. If it later changes to submitted, it is no longer possible.

Changing the delivery address of an order that is already with the carrier is possible for any postal code. Inform the retailer that it may take a few days.

 

Cases where the shipment is already being delivered:

  • If it is the same postal code, GLS can manage it.

  • If the postal code is different, request to GLS - EMAIL TITLE “New shipment – Novo Envio” with a copy to DBS in the email (subject to delay – Sujeito a atraso).

Cases where delivery has not been made:

  • The address change can be requested, regardless of whether it is the same postal code or not.

Registered cases:

GLS cannot change the address because they do not have the order. CS must wait for the order to reach the center to request the change.
