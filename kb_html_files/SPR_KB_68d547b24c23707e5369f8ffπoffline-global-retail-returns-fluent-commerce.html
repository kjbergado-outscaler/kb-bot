Creating a return with a refund automatically creates a credit memo.

Conditions for Accepting a Return with Refund

  1. Product Claim:
    A return with refund can be processed if the customer requests a refund instead of a new product.

  2. Incorrect Delivery:
    If there has been a wrong delivery, a return with refund may be issued upon the customer’s request.

  3. Undelivered Mattress:
    If the mattress is marked as delivered by the carrier but the customer has not received it, a return and refund can be processed upon customer request.

  4. Failed Delivery (Returned to Sender):
    If the mattress was unable to be delivered and has been returned to us     click on "create return without refund"
