1. How to initiate a return with GLS

The return process is streamlined through Robotic Process Automation (RPA).

1.a) First: Initiate Return on Magento

  • Magento: Create a return request on the OMS. To review the basics, visit Magento return

  • Make sure the address and contact information of the customer are correctly noted in the OMS.

  • Ask the customer for preferred pick-up date. At least 2 working days in advance, and it cannot be during the weekend.

Note: we cannot promise GLS will perform the pick up on that same date, see communication tips below.

1.b) Second: Automated Assignment to the Carrier

  • Once the return is created, it is automatically assigned to 4PL, which is our Fourth Party Logistics partner.

  • 4PL then assigns the return shipment to GLS, the carrier responsible for handling 1MH products.

2. Important information for setting customer expectations

GLS does not communicate with customers about the pick up date. While we ask for a pick up date, GLS is responsible for organizing the day and time of the pick up.

After you have created the return request in the OMS, it will automatically be transmitted to 4PL CS. 4PL CS will schedule a time for collection with the customer within 7 to 10 days.

3. How to verify if the order was created correctly

  • All the customer information is correct

  • The carrier field is set to '4PL CS'

  • The pick up date field is NOT left empty

    There must always be a pick up date filled when you create a return with GLS.

  • The status of the return is 'Created'

4. Status updates in the OMS

Thee status updates in the OMS are automatically updated.

5. How the refund is triggered

Once the return is completed via the drop off, the status in the OMs will get updated automatically to 'Complete'. This status update will then automatically trigger the refund process.

Once the refund is initiated, banks or payment service providers (PSP) will have different deadlines to have the amount available reflected in their account. More detailed timelines per PPS can be found in the article 'Refund request'.

If the customer reaches us after we already completed the refund, they should reach out directly to their bank or payment provider company to get further information and more specific deadlines.

6. When can this return ticket be considered 'Resolved'?

Once you have successfully created the return and have verified that it was created correctly, you can mark the case as 'resolved'.
