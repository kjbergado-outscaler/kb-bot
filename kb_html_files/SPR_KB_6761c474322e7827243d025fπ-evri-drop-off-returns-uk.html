All returns for larger items are managed by Rhenus.

If a customer asks for a return of a larger item (beds or mattresses), follow the steps below.

  1. First, open Fluent Commerce. 

  2. Click on "Filter" and then select "Ref." 

  3. Click on the order number related to the return. 

  4. When you have opened the order, click on the three dots and choose either "Create Return with Refund" or "Create Return without Refund," depending on the nature of the return.

  5. When prompted, click on the (+) button to specify the number of articles you wish to return. 

  6. Fill in the mandatory fields, using a (-) to indicate required information.
    Note that the carrier field must be left empty.

  7. Once all necessary details are provided, submit the form to initiate the return.

What is the status of the return after I create it?

The status must be set to 'Return Created'. 

Never manually move the status to 'Return Organized'.

​

How is the request send to the return carrier?

Once you create the return, the return request will automatically be transmitted to Rhenus, our returns partner for larger items.

​

How is the return completed?

Once the carrier completes the return, the return will automatically be marked as 'Complete' in Fluent Commerce (note, this is not the case for Magento!). For cancelled returns, you (the agent) should manually mark the return as 'Cancelled' in Fluent.

​

How is the refund triggered?

Once the return status gets set to 'Complete', the refund process will be triggered automatically. It will depend on the payment method used how long it takes for the refund to be processed. Usually, this can take up to 14 working days, depending on the payment provider.

If the customer has questions about the status of the refund, they can best contact their payment provider directly.
