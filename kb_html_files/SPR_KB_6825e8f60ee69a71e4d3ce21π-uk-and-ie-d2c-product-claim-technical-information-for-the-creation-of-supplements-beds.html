Each PDF includes:

  • A list of SKUs by variation (size, color, component)

  • A brief description of each SKU

    ​

Note: Not all variations are listed. Focus on matching the description to the customer’s issue.

​

Step by Step

When creating a supplement for any of the beds listed:

  1. Identify the product and the issue (e.g., missing part, damaged component).

  2. Open the corresponding PDF from the PDP or the links above.

  3. Find the correct SKU based on:

    • Product type

    • Size

    • Color

    • Component (e.g., slats, armrests)

  4. Double-check the SKU description to ensure it matches the customer’s need.

  5. Create the supplement using the verified SKU.

​

Note: If you’re unsure about a SKU or can’t find the right one, please escalate to your team lead.

​
