Accessing Orders and Messages

Octopia is a very straight-forward platform. In the main page, you’re able to access order information by simply clicking in the relevant options:

  • Open orders

  • Late orders

  • Messages and claims-related messages

Order Management

Open Orders

  • All Cdiscount orders are sent to Octopia.

  • Orders from Cdiscount are automatically imported from Octopia into Fluent Commerce. No manual import or entry is required.

  • Orders will appear in Fluent with all key details (SKU, customer info, shipping address).

Accessing the Order

  • Select “Orders to be handled”

  • Click in the Order Number (highlighted below)

​

​

​

Note: To access the order in Fluent Commerce, copy the order number and search as External Order Ref.

​

By opening the order, you’ll be able to see the ordered products, price, expected delivery information, and the customer’s information.

Cancellations

Customer Request for Cancellation

  • Step 1: A cancellation request will appear in the Octopia portal.

  • Step 2: Locate the order in Fluent Commerce.

  • Step 3: Attempt to cancel the order in Fluent. If cancellation is successful:

    • Go to Octopia and accept the cancellation there to finalize it.

  • If cancellation is not possible in Fluent (e.g., order is in shipping), reject the request in Octopia.

​

​

​

  • If an order cannot be canceled or corrected due to technical sync issues:

    • Open a JIRA ticket or contact Francisco for manual intervention.

    • Only after confirmation from Francisco can you proceed with cancellation in Octopia.

Returns

  • Returns are handled exclusively in Fluent Commerce. No action is needed in the Octopia portal for returns.

Late Deliveries

  • Often the delay is due to Fluent Commerce failing to sync the tracking number with Octopia. If an order appears in Octopia as a late delivery:

    • First verify if it's truly delayed by checking tracking info in Fluent Commerce.

    • If the tracking information is available, manually input the tracking details into Octopia to correct the status. This resolves false late delivery claims caused by Octopia sync issues.

    • If the tracking information was already correct in the platform, investigate the delay.

Messages

All Cdiscount-related contact are made through Octopia. To answer a message:

  • Click “Messages to be handled” or “Including claim message”, depending on which case you need to handle;

  • Select the message and reply accordingly.
