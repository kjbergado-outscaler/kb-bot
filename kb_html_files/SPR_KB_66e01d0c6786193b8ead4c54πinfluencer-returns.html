Introduction


At Emma, we work with Influencers who promote our products.


Whenever an influencer wants to return their item, internal partners (marketing managers for example), can open a request for influencer return order creation. They can do this through the Influencer creation Jira portal. These tickets will have all the information needed to create a return using OMS and Magento.


However, the process can vary from country to country. For this reason, you will find a list below with the return process and, if needed, additional information.


Influencer Returns: Step-by-step

  1. Get ticket from the Influencer creation Jira portal;

  2. Find the order in Fluent or Magento and open the return, according to the info provided in the ticket and the process described in the KB below;
    2.1 - In case the return is done manually, redirect to the appropriate channel and retrieve the tracking code;

  3. After you created the return and informed the customer, leave a note on the Jira ticket mentioning you created the return;

  4. Move the Jira ticket to resolved.
