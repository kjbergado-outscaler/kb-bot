What is a discount code?


We have different ways to push customers to sales. Marketing teams from countries may create different campaigns to give a discount on the products/ order to incentive customers to close the deal with us.


The coupon code can vary from campaign to campaign and usually go from 5 - 10%, depending on the specific strategy of the country. In some special cases, even more amounts of discounts can be offered.


Types of Discount Codes


As explained above, Emma can have different discounts active according to each country's team sales and marketing strategy. Some examples are:


- Codes for Influencers;
- Specific seasonal campaigns (such as summer sales, or a specific holiday campaign);

- Corporate Discount Codes;
- Codes for Affiliates;

- Marketing campaigns in media (as radio, tv, outdoors, etc).

Or any other reason the marketing team may be able to use to engage customers with the DEAL button!

Common Codes inquiries from Customers


It may happen customers reach out to us to claim the discount code is not working and the discount expected has not been applied to the order.


Today, the requests may be divided into:

  1. Code typo by customer;
  2. Discount information/ policy;
  3. Code - not working.


Freshdesk Ticket Accuracy: Ensure you always provide the related Ticket type & subtype based on the customer request.



Let's see in detail the 3 most Common Code questions from customers:


1) Code typo by customer

Some customers might reach out to us claiming the discount code is not working but this might be because they made a typo when adding the discount code.


You should check if the customer is using the right coupon code, is the customer typing it properly?

If not, provide the related valid one.


2) Discount information/ policy

Some customers might reach out to us regarding the policy of the discount - what is the coupon code giving them as a benefit? How long is the discount code valid?


  • Is the campaign for the coupon code still live?
    If not, explain to the customer the campaign has already finished and the discount code is not valid anymore. Check with your marketing strategy to check if you can proceed with a discount (price match). Follow the market procedure related to it.


  • Specific policies:
    If the discount code is not valid to apply over an existing discount, explain the discount is not cumulative.


  • Active coupon codes:
    Customers may reach out to know if there is any active discount code. As an example, you can always advise customers to subscribe to Emma's newsletter and they will be up to receive directly the promotions we do (some countries also include a code for subscription).


3) Code not working

Some customers might reach out to you claiming the discount code is not working. It can happen eventually that the code is properly typed, but the customer doesn't see the discount reflected on the cart in the website. In those situations, we have to do some verification, but after checking that it's all right, we can provide a discount after the purchase for the customer.


But first, let's check below all the steps to investigate the problem and the steps to handle Discount Code Issues.


How to handle when a Discount Code is not working?
How to check if the Discount Code exists and is valid?
If the customer claims the Code is not working, we need to understand if is a valid code. A "valid code" means that:

1) The code exist in our systems, so it was created by some team inside Emma;

2) It is still active;

3) It's being applied according to the Terms & conditions of the Code.

To check all the 3 situations mentioned, you need to follow the process below:

Option 2: If not found in Magento, check it on Talon.One (How to find discounts in Talon.One)

Ans if it doesn't appear in Talon.One, it means that the code doesn't exist in our systems (it can mean that is a fake code, unfortunately). In the end of this article, you will see what to do in case the discount code that the customer has it's not valid.



What to do with VALID Discount Codes?
Even if the code is valid, it may not work for the customer and this will make them ask for your support:


Step 1: Simulate a purchase on Emma's website (abiding by the code's T&Cs), apply the customer's code, and check if the discount is redeemed in the cart.

Step 2: If works for you, but not for the customer, ask the customer to clear cookies & cache or try to access through an incognito window.


Step 3: If after this, the code is still not working for the customer, ask the customer to send us the picture of the discount not applying, so we can validate and report the issue to the related team.

Step 4: Once we receive this information and have confirmed that the claim is valid, we can advise the customer to go ahead and proceed with the order without the discount code. Make sure to explain to the customer that after the purchase, they should contact us, provide the order number and, once we verify that the order placed meets the voucher T&Cs, we will issue a partial refund in the amount that should have been given as a discount by the voucher.

Step 5: After receiving the order confirmation, trigger the related discount manually the OMS (Magento or FluentCommerce).

Review how to trigger the discount: Magento: what and how to or Fluentcommerce.


Step 6: Send the discount confirmation to the customer after triggering it in the OMS. Manage customer expectations regarding refund time.


IMPORTANT: The refund can only be done if the customer makes a purchase under the same conditions as indicated in Magento/Talon.One for the voucher. For example, if the code's conditions state that it can not be used with a product already in sale, the customer's order must not contain any products on sale, otherwise the refund cannot be issued.



What to do with NOT VALID Corporate Discount Codes?
Now, let's check what to do if it's confirmed that the code is not valid and non-existent in our system:


Situation 1 - The code is invalid because it doesn't show in our systems:
Answer to the customer: "Unfortunately, after checking our system, we were not able to find the code in question. As such, we would recommend you to contact the site/company that provided you with this code, in order to clarify this situation".


Situation 2 - The code is invalid because it has expired (the customer contacted us after the expiration date):

Answer to the customer: "Unfortunately, after checking our system, we can confirm that the discount code in question is no longer valid, as it could only be used until XX/XX."

Situation 3 - The code was valid, but expired between the customer contacting us and our first reply.

-> In this case, as a gesture of goodwill, we can advise the customer to go ahead and make the purchase. Explaining that after this, they should contact us, provide the order number and, once we verify that the order placed meets the voucher T&Cs, we will issue a partial refund in the amount that should have been given as a discount by the voucher.
