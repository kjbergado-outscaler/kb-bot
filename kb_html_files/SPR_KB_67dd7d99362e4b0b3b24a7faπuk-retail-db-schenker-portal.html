Scope

This SOP outlines the process for managing order tracking, delivery status, cancellations, and obtaining proof of delivery (POD) using the DB Schenker Carrier Portal.

Instructions

Accessing the DB Schenker Client Portal

  1. Open the Portal:

    • URL: DB Schenker Client Portal

Searching for Orders

Navigate to the Dashboard:

  • After login, you will be redirected to the main dashboard.

​

​

Select “View Outbound Order”:

  • Go to the Order tab.

  • Click on View Outbound Order.

​

​

Enter Order Reference:

  • Copy the Order Reference Number from Magento.

  • Example Order Reference: 603378645.

​

​

Set the Date Limit:

  • The date limit should always be the first day of the month in which the order was placed.

  • Example: If the order date is February 17, set the Date Limit to February 1.

Activate Search in History:

  • Check the Search in History option.

  • Click Search.

​

​

Validating Order Information

Confirm Order Details:

  • Cross-check the Customer Name in Magento with the result in DB Schenker.

​

​

Review Shipping Information:

  • Navigate to the Shipping tab.

  • Verify the Carrier Information.

  • In this example, the carrier is EMMA RHENUS.

​

​

Check Order Lines:

  • Go to Order Lines or Pick Lines to view the specific items in the order.

​

​

Locate the External Tracking Link:

  • Under the Shipping tab, check if an External Tracking Link is available.

  • The link will redirect to the courier’s platform for live tracking.

​

​

Troubleshooting: When Order is Not Found

  1. Alternative Search Method:

    • If the order does not appear initially, modify the Order Reference by adding an underscore followed by 1 or 2.

    • Example: 603378645_1 or 603378645_2.

  2. Repeat Search Steps:

    • Re-enter the order reference with the modifications and search again.
