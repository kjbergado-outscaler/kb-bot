Rhenus is our return partner for returning larger items.

You must create the return on the order management system (Magento) and then make a request for Rhenus to collect return via Rhenus Portal.

For a general refresher on how to create a return on the order management system, check: How to: Product return (Magento)

​​How to create manual returns for Rhenus

  1. Open Magento

  2. Click the 'Return prodcuts' button on the top right of the order page

  3. Fill all the required fields

    1. Return products: product will be skipped if checkbox not set 

    2. Quantity: quantity of product for return

    3. Quality: Available options: packed,unpacked*, faulty

      *Rhenus will only be applied when the selected quality is 'Unpacked'.

    4. Refund to the customer: Available options: full refund, no

       Depending on selected value, an email template will be choosen

    5. Pickup/Return-service: Available options Based on Manual return assignment rules

      For UK orders for larger items, this should be 'Rhenus'.

    6. Carrier Info:

      1. Track number Track

      2. Link

    7. Warehouse Address: Warehouse address for returned product

    8. Reason: Available options:

      1. The item is received too late

      2. Product is too firm

      3. Product is too soft

      4. Cannot adjust the height for myself

      5. Product is too warm, cause sweating

      6. The cover does not have a nice touch

      7. Ordered wrong product size

      8. Other reasons (the comment is compulsory)

      9. Ordered twice

      10. Product Claim

      11. Logistic Claim RTS / Cancellation Reason of return

    9. Comment: Comment to return

    10. Pickup Date Pickup Date if available

    11. Credit Memo Comment(B2B only!)

    12. Credit External #(B2B only!)

    13. Product back at warehouse: If set yes, return will be moved to complete status

Returns on Magento will have to be tracked on the Rhenus portal and manually updated on the order management system.

Note: for Magento returns, the carrier’s tracking number must be added to the ticket whenever possible.

If return related to UK store, has 'GB' country in shipping address, has 'Expected' status, quality value 'Unpacked' or 'Faulty' then the automatic scheduling job will export it and set it to 'STATUS_RHENUS_SENT' status. No need for you (the agent) to do this.
