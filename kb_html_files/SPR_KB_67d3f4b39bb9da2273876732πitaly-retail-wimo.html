Search for the Order

  • Log into FluentCommerce.

  • Use the external order number to search for the order.

Check the External Order Number

  • Confirm that the external order number matches the one you copied from FluentCommerce.

  • Check the Creation Date.

Access Fulfillment Information

  • Navigate to the “Fulfillments” section of the order details.

Copy the Tracking Label

  • Copy the Fulfillment reference associated with the order.

​

​

​

You might see more than one tracking number if there are multiple items in the order.

Check Delivery Status in Bracchi Portal

  • ​Log into the Bracchi Portal;

  • Select Shipment Tracking;

  • Fill-out:

    • Date from: select the first and last day of the purchase month.

    • Reference: paste the Fulfillment reference

    • Click Search.

Review Tracking Status

You'll be able to see the shipment details and delivery status.

  • Select the shipment you want to review.

​

​

​

Important: Always provide an approximate delivery date, never a specific one.

Handling Discrepancies

If the order is marked as "Delivered" but the customer claims they didn’t receive it:

  • Ask if the customer prefers to receive a supplement or a refund and proceed accordingly.

Create Supplementary Order (if needed)

  • After sending the document, proceed with creating a supplementary order for the customer if required.

  • Add a comment to the order: Document the situation on the order page for visibility.
