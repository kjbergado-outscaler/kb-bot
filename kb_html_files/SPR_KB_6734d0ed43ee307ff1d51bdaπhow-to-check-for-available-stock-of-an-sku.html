You might sometimes need to check if a particular SKU has stock. This might be for instance when you plan to create a supplement. The instructions for how to find out if a particular SKU has stock in a warehouse in your market are outlined in this article.

In what scenarios should you use this article?

  • Product and/or Logistic claims:

    Before creating a supplement for a claim, you need to ensure there is stock for the item. Is no stock available? Inform the customer that we cannot provide a timeline for the delivery of the supplement, and if they would prefer to cancel / return the item.

  • WIMO where Updated Delivery Date = 'Invalid date'

REMINDER: You must never create a supplement to unblock an order that is stuck or delayed.

1. How can I check if the product has stock?

  1. Navigate to the Virtual Catalogue in Fluent Commerce

  2. Select the stock for your market (in this example, we use 'Poland')

  3. Select the Base Virtual Catalogue for your market. That is the bottom line from the options you see for your market. In this example for Poland (PL), it looks as follows.

  4. On the page with stock, filter for 'Ref'

  5. Then, search for the SKU that you need to supplement.

  6. Check if the SKU has:

    1. Location Ref: This will indicate the warehouse that has the stock.

    2. ATS (Available Stock to Sell). The thresholds are:

      1. ATS for Mattresses > 15

      2. ATS for Beds and Accessories > 20

    3. If the answer to step 6.2 is yes, then we have enough stock to create a supplement. If the answer is no, then we cannot promise the customer a supplement.

TIP! Don't know the products and SKUs used in your market? Check the article about the 'Products Overview per Market folder'

2. Important note about bundles

Are you seeing an SKU that starts with 'XEM'? For instance: XEMBDWDB180200_00150:EBDOW180200SIA?

The part before the ':' mark is only to indicate this SKU is part of a bundle. To review the stock for this item, you must only search for the last part of the string. In this example, that is 'EBDOW180200SIA'.

3. FAQ

What does the Updated Date mean?

The updated date can be ignored. It currently does not give accurate timestamp of the last updated date, and must not be used as a reference. The stock value is updated hourly, independent from the 'updated date' that is represented on the page in Fluent Commerce.

How often is the stock updated?

The stock value is updated hourly, independent from the 'updated date' that is represented on the page in Fluent Commerce.

How do I know which warehouse I need to check?

As long as you correctly selected the country, any warehouse in that virtual catalogue can serve that market.
