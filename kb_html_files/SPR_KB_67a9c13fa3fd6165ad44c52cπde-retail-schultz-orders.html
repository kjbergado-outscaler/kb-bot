Overview

Schultz produces and provides us Slatted Frames, and this SOP will teach you the process for managing slatted frame orders from Shultz to KHG. It is a simple process, which starts with the Order Management Team, that sends daily fulfillment reports for this product. This e-mail will look like this:

​

 

Instructions

Open the daily Fulfillment Reports

  • Download the report

  • Select column “A” → Data → Text to Columns

  • Choose “Delimited” → Next

  • “Comma” → Finish

​

 

Filter Slatted Frame SKUs

  • Add filter → Filter the Retailer as “Wholesale DE”

  • Select all the Slatted Frames SKUs - they start with “EBDSF” → OK.

 

​

  • Copy the Order Numbers and paste it in our internal file.

 

​

  • Check if there are duplicates

    • Delete the duplicates (if there are) and forward the new cases for the agents to transmit to Schultz.

Forward Cases to Shultz

  • Send the new orders to sales-team@schultz-schlafkultur.de requesting the point of status. You can use this template:

“Good morning/afternoon,

Enclosed is the new list of orders that we are placing with you manually.
Please add the RFF numbers and the expected delivery dates to the table.

We wish you a pleasant day.

Thank you in advance,

[YOUR NAME]”

Please make sure to add fatih.bektas@emma-sleep.com, alena.lauter@emma-sleep.com and irma-manou.artzt@emma-sleep.com in cc.

​

Wait for Feedback

  • Monitor and await feedback from Shultz regarding the order status.

  • After we receive the feedback, we update the file accordingly. Alena checks the file weekly and updates the orders in FluentCommerce.
