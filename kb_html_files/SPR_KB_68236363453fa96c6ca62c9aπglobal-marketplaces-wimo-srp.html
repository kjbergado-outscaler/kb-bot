Search for the Order

  • Log into Fluent Commerce.

  • Use the external order number to search for the order.

Check the External Order Number

  • Confirm that the external order number matches the one you copied from FluentCommerce.

Access Fulfillment Information

  • Navigate to the “Fulfillments” section of the order details.

  • Click on the “ID” number.

​

Copy the Tracking Label

  • Copy the tracking label associated with the order and check the Carrier's name.

    • We work with three carriers: DB Schenker, DPD, and Trusk, with most orders shipped via DB Schenker.

​

​

​

You might see more than one tracking number if there are multiple items in the order.

Check Delivery Status in Carrier's Portal

  • Access the DB Schenker portal;

  • Select "Tracking";

​

  • Paste the tracking reference and click Search;

  • You'll then be able to see all delivery information

    • Update the customer accordingly.

DPD

  • Access the DPD Portal;

  • Paste the tracking reference and click Validate;

Trusk

  • Access the Trusk Portal;

  • Select "My Orders";

  • Paste the tracking reference and click select the order;

  • You'll then be able to see all delivery information.

    • Update the customer accordingly.

Handling Discrepancies

If the order is marked as "Delivered" but the customer claims they didn’t receive it:

  • Fill out the customer’s name and address and send the “Parcel Not Received” document below. This serves as a binding statement that the parcel has not been received.

    • Send the "RVE DPD" file for DPD deliveries. Otherwise, forward the "RVE_French_Emma" document.
