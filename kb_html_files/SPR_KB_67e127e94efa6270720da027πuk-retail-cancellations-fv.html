Open the order in Fluent Commerce

  • Check if the external order number matches the one you copied.

Handling Cancellation Options

  • You will be presented with two options:

    • Option 1: If cancellation is possible, you will be given the option to proceed with the cancellation.

    • Option 2: If you cannot proceed with the cancellation, you will see a message indicating that the items have already been sent to the fulfilment location.

Note: Usually we’re able to process the cancellation up until the Booked status. After that the option is most likely not available, but always make sure to check!

Start Order Cancellation

  • Click the three dots in the top-right corner of the order page.

    • Select Cancel Order Items.

​

If Cancellation is Possible

  • Cancel the order

    • Check the “Order Progress” tab, it should be marked as “Cancelled”.

​

​

  • Inform the customer: Notify the client that the cancellation has already been done in our system.

  • Add a Comment to the Order: Document the situation on the order page for visibility.

  • Close your Sprinklr case.

If Cancellation is Not Possible

  • For orders in the "Order Processing" or "Received by FL" stages, cancellation cannot be done directly in the platform.

    • If the customer insists on cancellation, proceed by adding all relevant order information into the Jira ticket progress spreadsheet (e.g., PO number, SKU, fulfillment status, tracking number, etc.).

Follow-Up

  • Check the spreadsheet daily to see if the order has been cancelled.

  • Once cancellation is confirmed in the spreadsheet, the order will be automatically marked as "Cancelled" in Fluent Commerce.

  • If the cancellation request is processed, follow up to ensure the customer is refunded.

    • Reach out to the designated person handling the JIRA tickets for cancellation updates, as you may not have direct access to the JIRA system.
