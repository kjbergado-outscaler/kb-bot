Invoice Information

Invoices are an important tool for retailers, so it's very common to receive requests about this document.

Magento

Whenever an invoice is created, you can find it in the "Invoice" tab in the Order View. Click on the file to open the document, download it and send it to the customer.

If you can't find the Invoices on the order page, please go to Magento -> Sales ->Invoices. You can search by Order Number (Order #) or Invoice Number (Invoice #).


Fluent Commerce:

Whenever an invoice is created, you can find it in the "Invoice" on WINSCP.

​

Missing Invoices - Magento

Every time you need to create an invoice that is not available in Magento, open the Retail Backoffice Portal and choose Payments -> Create Missing Invoices


Please give the following information to the Backoffice team:

  • Order Number


Missing Invoices - Fluent Commerce:

If the invoice is not in WINSCP, check if the order is COMPLETE, if not escalate to OPS. If the order is COMPLETE, please open a jira to AR.

Invoice Corrections - MAGENTO

Every time you need to correct an invoice that has an error (such as price, items, or quantities), open the Retail Backoffice Portal and choose Payments -> Correct Invoices

​

Please give the following information to the Backoffice team:

  • Invoice Number

  • Errors

WHEN OPENING THE JIRA REQUESTING THE CORRECTED INVOICE, ALWAYS REQUEST AT THE SAME TIME THE CREDIT MEMO CORRESPONDING THE INVOICE CANCELLED

Invoice Corrections - FLUENT COMMERCE - COS

Every time you need to correct an invoice that has an error (such as VAT, price, items, or quantities), open the https://emma-sleep.atlassian.net/servicedesk/customer/portal/117/group/308/create/1173, that is the AR Portal.


Please give the following information to the AR team:

  • Invoice Number

  • Errors

  • SKU

  • QTY

  • Amount per item


WHEN OPENING THE JIRA REQUESTING THE CORRECTED INVOICE, ALWAYS REQUEST AT THE SAME TIME THE CREDIT MEMO CORRESPONDING THE INVOICE CANCELLED

​

Download Multiple Invoices

Sometimes retailers may ask you to send a list of invoices for a certain period.

​

Magento

1. Open Magento -> Sales -> Invoices

​

2. Include the information about the invoices we need. This will depend on what the client asks for (e.g. 1 invoice, a list of invoices, all invoices for the last month, invoices for the whole quarter, etc) and click on the search button:


3. Download the files in the format requested by the customer: if he wants Excel format it will be CSV export and if he wants PDF format it will be PDF invoices:

Fluent Commerce

Search for the invoice number on Fluent Commerce

  • Open the order;

  • Click "Attributes"

  • Roll the page down until you find "invoicePDFLink"

  • Copy the invoice number.

WinSCP

  • Open WinSCP;

  • Select the relevant Invoice folder.

  • Select "Find Files"

  • Paste the invoice number in the following format: *123456789*

  • Click "edit"

  • Copy the remaning invoice numbers in the "Include Files" field;

  • Click "OK".

The invoices will then be downloaded.
