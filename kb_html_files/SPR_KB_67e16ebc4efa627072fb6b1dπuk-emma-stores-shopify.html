We use Shopify exclusively to validate order details and check their statuses. No action should be done in this platform. Please refer to the relevant articles if you need to proceed with Cancellations, Returns, Product Claims, Logistic Claims etc.

Login & Credentials Management

Shopify allows only two active users per credential. As we are using a single credential at the moment, it requires shared login management.

  • Access Shopify;

  • Add your Emma email and select Continue with email;

  • Select Log in with Single sign-on.

​

​

  • If authentication is required, use the store’s authentication app.

Accessing the Order

To retrieve customer order information and verify details before taking action.

  • Log into Shopify.

  • Navigate to the Orders section.

  • Use the order number or customer email to locate the correct order.

  • Verify that the order details (customer name, address, products ordered) match the request.

​

​

​

Order Verification

Ensures that customer-provided details match Shopify records before any actions are taken.

  • Cross-check the order details with the customer request.

  • Verify:

    • Order status (Processing, Shipped, Delivered, etc.)

    • Payment status (Paid, Pending, Refunded, etc.)

    • Shipping details and tracking number

  • If discrepancies exist, escalate to the relevant team.
