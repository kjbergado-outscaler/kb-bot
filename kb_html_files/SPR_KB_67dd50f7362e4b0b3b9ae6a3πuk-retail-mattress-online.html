This SOP provides a detailed guide on handling post-sales concerns for Mattress Online. This process applies to handling inquiries related to returns, refunds, and faulty issues for Mattress Online. It does not involve order processing, WIMOs or direct customer interaction.

Process Overview

  • Mattress Online is a truck shipment store, similar to Mattress Man and Mattress Next Day.

  • Orders from the store are sent to Inna, who uploads them.

  • Inquiry handling is done via Sprinklr.

  • Most common issues involve faulty mattresses.

  • No collection is required for Mattress Online.

  • Customers dispose of faulty mattresses, and the store sends a debit note to process a credit memo.

  • Credit memos are processed only after the order status is marked as "Complete."

Instructions

Identifying and Handling Inquiries

Check Inquiry Type

  • All inquiries should come through Sprinklr.

    • Common issues: Faulty mattress, refund request, or replacement approval.

    • The order number is listed in the e-mail subject.

    • Mattress Online orders are identified using the PO Number, not the order number.

Note: If a customer reaches out directly, advise them to contact Mattress Online.

Search for the Order

  • Use Fluent Commerce to search for the PO number.

If the PO is missing in Fluent Commerce, check Magento.

Verify Order Status

  • If the order is still in "Booked" status, it is not eligible for a refund yet. The logistics team (Inna) updates the order status to "Complete."

    • Add a comment on the order advising the replacement is approved. Make sure to add the order number in the comment.

    • Keep track of the Order Status to proceed after it’s changed to “Complete”.

Processing Faulty Mattress Requests

Check Photo Evidence

  • If valid photos of the faulty mattress are provided, approve the replacement or refund.

    • If insufficient evidence is given, request additional images.

  • You can find the claim pictures requirements and examples here and how to validate the claim here.

Approve Refund/Replacement

  • If approved, inform the store that they may process a replacement or refund.

  • If a refund is chosen, they will send a debit note for the credit memo.

Process the credit memo once the order status is changed to "Complete", never do it in “Booked” status.

Handling Product Claims

If a product is faulty and a claim is approved, no replacement is arranged by us. Simply approve their request and document the reference number.

  • Advise Mattress Online to replace the faulty mattress from their stock.

  • Request a debit note from them for processing credit memos.

Collection Policy: Mattress Online does not return faulty mattresses. They are responsible for disposal per agreement with Inna.

Processing Returns & Credit Memo

  1. Locate the corresponding PO number in Fluent Commerce or Magento.

  2. Follow these steps in Fluent Commerce:

    • Search for the BU key for Mattress Online.

    • Locate the specific PO number.

    • Identify the mattress type (e.g., "Emmy Premium Hybrid Mattress King Size").

    • Process a return for the identified item.

Create Return Process in Fluent Commerce

  • Click the three dots in the top-right corner → Select "Create Return with Refund".

Fill Out the Return Form

  • In the return form, you may fill out blank fields with a dash (“N/A”). Since it’s a retailer order, the form will auto-fill those fields.

    • The fields Company Name, Street, City, Postcode, and Email are automatically filled and you can leave them as they are.

Returned Item Details

  • Quantity of Items: Select the quantity of items to be returned by clicking the “+” icon.

  • Item Condition: Choose the appropriate condition:

    • Packed - Never used

    • Packed - Used/Re-packed

    • Unpacked

  • Return Reason: Select “Product Claim”.

  • Comments: Add the end customer’s name in the “Comments” field.

  • Click Submit on the top right corner.

  • Flag “No pickup disposal by retailer” → Click Submit on the top right corner. 

Finalize and notify Mattress Online that the credit memo will be sent to their accounts team automatically upon completion.
