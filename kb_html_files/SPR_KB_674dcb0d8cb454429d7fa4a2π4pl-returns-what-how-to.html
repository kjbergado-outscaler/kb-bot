4PL Central Station (referred to as 4PL or 4PLCS) is an intermediary party between Emma and return carriers. They organize returns for some product categories in some markets for us.

1. What does 4PLCS do?

This company takes over a few tasks:

  • Deciding which return carrier to use for which products

  • Arranging the return with the return carrier

  • Communicating with the carrier when something did not go as planned

  • Tracking the status of the returns

  • Notifying Emma when the product was returned so we can start the refund process

2. How to: Request a return from 4PLCS?

In Fluent Commerce, the assigned to 4PL is automatically.

In Magento, you will then need to select “4PLCS” as the return method in Magento (including the pick-up date for category 2 items) and then the return will automatically be arranged.

3. How to: Handle customer requests, when 4PLCS manages the return?

In the 4PLCS team, there is a team of operations managers who will handle your requests regarding returns. You can reach them via the 4PL Jira Portal [link to 4PL Jira portal].

  • Return cancellation

  • Return adjustment (e.g. change pick-up address or customer contact info)

  • Pick-up re-scheduling

  • Others

You will also need to provide the order number and the return ID from the OMS (Magento or Fluent).

5. FAQ

When cancelling a return that is handled by 4PL, you must cancel the return in the OMS AND cancel the return with 4PL directly. Only when both have been completed, you have correctly cancelled the return.

  1. Go to the 4PL Jira portal and raise a 'Request to 4PL'.

  2. As the 'Request to 4PL' select 'Return cancellation'

  3. Ensure all other fields have been filled

    1. Emma Order Number: the order number

    2. Return ID: the return order number

    3. Market: the market

    4. Raise a request for the 4PL Central station to action or clarify (Add Emma Order Number in this field): 'Return cancellation of [put your order number]'

  4. Once you receive confirmation on the Jira ticket if the return was cancelled, the return will automatically be cancelled in the OMS too.

You must not cancel the return in Fluent Commerce, as that will be done by 4PL.
