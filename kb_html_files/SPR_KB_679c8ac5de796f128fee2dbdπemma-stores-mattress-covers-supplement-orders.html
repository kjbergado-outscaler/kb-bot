Scope

This SOP focuses on shipment of covers for the mattresses from the warehouse KNR1_DE.

This applies to all stores.

WH Location

The all covers will be located in the KNR1_DE warehouse.

WH assignment - KNR1_DE

Carrier assignment - GLS

Orders from: DE, NL, IT, FR, PT and ES (UK & IE not included)

​

Note: In case any cover is WAITING FOR STOCK, inform the WH Manager of KNR_DE.

How to check the Available Stock

Click here: Latest stock (IML)

This option is useful if you want to get a rough idea of the stocks in other countries, when you don’t have a stock file at hand (ex: BENEDACH). However, we advise to use option as a last resort, as Tableau may have data issues.​

How to create a Cover Order

To ship a cover, an order must exist in Magento.

  • Locate the order in Magento

  • Look for the mattress in the list below and find the corresponding cover.

  • Create a supplement with the specific SKU. You can check the proccess clicking here.

​

Note: The customer can keep the old cover, and it is not necessary to request pick-up and return.
