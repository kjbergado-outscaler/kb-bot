Return Handling Process

  • Navigate to Order Summary > Return Communication.

​

​

  • Identify new return requests.

  • Update the QVC Return Tracker:

  • Highlight collected and refunded orders in green.

  • Highlight pending collections in white.

​

​

  • Retrieve the return details:

  • Customer Name, Address, Contact Number, Email, Product SKU, and Reason for Return.

​

​

  • Open Fluent Commerce and locate the order.

​

​

  • Process the return:

  • Select Process Return with Refund.

  • Choose Return Reason (e.g., Change of Mind, Damaged, Misspick).

  • Submit the return request.

​

​

  • Record the processed return in the QVC Return Tracker.

​

​

Refund Processing

  • Check previously processed returns in QVC Return Tracker.

​

​

  • Identify items marked collected but not refunded.

  • Log in to QVC Commerce Hub.

  • Search for the External Order Number.

​

​

  • Cross-check with Fluent Commerce to verify order completion.

  • Click Submit Refund in Commerce Hub.

​

​

  1. Confirm that the refund has been issued.

  2. Update the QVC Return Tracker with the refund status.
