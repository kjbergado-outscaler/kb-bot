Modifying Customer Information

Customer information such as e-mail, TIN and name can be changed, but must be forwarded to the OPS team for further investigation. If you receive a request to modify account information, please forward it to the OPS team.

Changing Delivery Information

  • If the request to modify delivery information arrives with the status still on “BOOKED”, this means that the order has not yet been sent to the WH. We can make changes directly on OMS, as it will update the information to be sent to the WH in the near future.

  • If the request to modify the delivery information arrives after the “BOOKED” status, this means that the order has already been sent to the carrier. In this case, we will need to contact the deliverer or carrier (depending on where the order is located) so that they can update the delivery information they have.

Note: always leave a note with the new address or estimated time requested by the customer, so if the transport company contacts us, we can easily collect the information.

Changing product/size/SKU

For this reason, whenever customers contact us to change the product or size (SKU change), the process to be followed will be to cancel the order and the customer to place a new, corrected order.
