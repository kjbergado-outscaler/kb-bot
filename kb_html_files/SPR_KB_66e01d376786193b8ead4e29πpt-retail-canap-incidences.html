Canapés Components

Size until 160x200

Canapés until 160x200 have 4 components:

​

1x A -> Cajón (All the outside box structure of the canapé)

1x B -> Herrajes (Metalic structure of the canapé)

1x C -> Tapa (Upper part of the canapé)

1x D -> Fondo (Bottom part of the canapé)


Size 180x200

​

Canapés 180x200 have 8 components:

​

1x A -> Cajón (All the outside box structure of the canapé)

2x B -> Herrajes (Metalic structure of the canapé)

2x C -> Tapa (Upper part of the canapé)

2x D -> Fondo (Bottom part of the canapé)

1x E -> Cajón Larguero (Middle structure of the cajón, only for 180x200 canapés)


Supplements of Canapés

Magento

When creating supplements for product or logistic claims of the canapé, please remove the bundle SKU and add the component SKU available in PT B2B - SKUs.xlsx. Please note that there are 2 colors of the canapé, and we should send the right one.


In case of doubt in the component we need to create supplement for, please escalate.

Fluent Commerce

When creating supplements please always check first if the order is complete, if the order is not complete, please escalate, after OPS change to complete, you can procced with supplement. DO NOT REQUEST A NEW ORDER TO BO.
