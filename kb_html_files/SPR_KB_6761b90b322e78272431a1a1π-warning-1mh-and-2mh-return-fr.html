1. How to initiate a return with Warning

  1. In case a return was not requested via the Return Portal, you can create it directly in Fluent.

  2. Make sure that the correct phone number & e-mail address of the customer are available in the order and the pick-up address is correctly added in the return. This is used by the carrier to contact the customer to schedule an appointment.

  3. Select all the items, inform the quantities, product conditions (packed/unpacked) and return reason. 

  4. If the customer has to return a big item (mattress or bed) and an accessory, create only 1 return and add all the items. This way all items get picked up by the same carrier and the customer does not need to return any product separately – besides that Emma also saves additional costs.

  5. Fill the carrier with WARNING_FR.

  6. The pick-up date field must be left empty.

  7. Don't fill the fields parcelLabReturnId and targetStatus.

  8. Once you created the return, the status will automatically marked as 'CREATED'. Your part is done. It will be transmitted to the carrier within the next hours.

  9. Don't move it to ORGANISED! Otherwise it will not be transmitted to the carrier.

2. Important information for setting customer expectations

  • Warning will contact the customer by email or SMS.

  • The customer will have to schedule the return with Warning directly.

3. How are status updates handled for the return?

All status update for returns with Warning are automated.

4. How the refund is triggered

Once the return is completed by the carrier, the status in the OMS will get updated automatically to 'Complete'. This status update will then automatically trigger the refund process.
