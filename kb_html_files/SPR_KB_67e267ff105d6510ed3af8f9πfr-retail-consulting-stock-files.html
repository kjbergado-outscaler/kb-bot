Overview

This SOP provides step-by-step instructions on how to consult stock files for monitoring inventory levels. This document applies to agents, sales teams, and logistics personnel who need to check stock availability, forecast future stock levels, and plan production or sales campaigns accordingly.

The Stock Monitoring File serves as a reference to:

  • Track available stock levels in warehouses.

  • Plan sales and production schedules based on stock availability.

  • Forecast stock shortages and replenishment needs.

When to use it?

  • Before making sales commitments.

  • To verify inventory levels before placing production orders.

  • To monitor stock replenishment schedules.

Data Update Schedule

  • Stock on Hand: Every Monday

  • Forecast updates: Every Tuesday

  • Waiting for Stock (Backlog): Every Monday

​

DO NOT edit any information on this file.

​

Accessing the Stock Monitoring File

  1. Open the company’s shared drive.

  2. Access this folder.

  3. Open the latest version of the REN1_FR_Stock_Monitor.

  4. Access the Stock Movement sheet. It is the primary reference for checking stock availability.

Breakdown by Sheet

Stock Movement

This sheet contains the stock levels of all SKUs in real-time. It is the primary reference for checking stock availability.

Key Columns:

  • SKU – Unique identifier for the product.

  • Description – Name and details of the SKU.

  • Stock on Hand (SOH) – Number of available units. This is the most relevant information when consulting the file.

  • Beginning – Stock available at the start of the week. It is usually the same quantity as “SOH”, unless the SKU is “Waiting for Stock”.

  • Waiting for Stock – Units expected but not yet received.

  • Coverage – Estimated number of weeks stock will last based on demand.

  • Replen Week – Expected week for restocking.

  • Forecast – Projected sales for the upcoming weeks.

  • Overstock – Indicates excess stock.

How to Use:

  1. Open the Stock Monitoring File.

  2. Navigate to the Stock Movement sheet.

  3. Use Ctrl + F → Find all to search for the SKU.

  4. Review the current stock level (column V, “SOH”).

  5. If stock is low, check the replenishment week.

IB_DATA

Provides information on expected replenishment dates and quantity.

Key Columns:

  • Item – SKU

  • Year; Month; Day – Full expected replenishment date.

  • Onordqty – Quantity of items expected to arrive.

How to Use:

  1. Open the ID_DATA sheet.

  2. Use Ctrl + F → Find all to search for the SKU.

  3. Check expected replenishment date and quantity.
