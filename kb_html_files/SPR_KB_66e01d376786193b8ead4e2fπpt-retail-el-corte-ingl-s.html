El Corte Inglés

El Corte Inglés is a retailer with 3 different stores:

  • ECI Lisbon

  • ECI Gaia

  • ECI Web

​

When handling El Corte Inglés orders, special attention must be paid, since we have contracted requirements that increase the complexity of the logistics chain:

  • Deliveries of all the products in the order at the same time

  • Scheduling with the customer (pickups and deliveries)

  • Old mattress collection (OMR)

​

Because of this complexity:

Delivery Time: 10 Business Days

Calls

​

Deliveries

Collections

Product Claims Collections: Collect at the same time of delivery with XPO by adding SKU EOMRM999999ABA to the supplement.

100 Nights Trial Collections: Charity Pick-Up

Exchanges and Returns

Exchanges

  1. Note in Magento, informing whether there is collection or not

  2. Create Product Claim, if necessary

  3. Create Return

  4. Create Supplement with Pickup SKU (EOMRM999999ABA)

Returns

  1. Note in Magento

  2. Product Claim, if necessary

  3. Create Retun in Magento

  4. Ask Charity Return

Specific Processes

Confirm Order in Magento


For e-mails like “PEDIDO 91083301 CLT ANA MOREIRA” you'll need to confirm:

​

Is the address correct and complete?

  • Before packing status: Update directly in Magento

  • After packing status: Inform WH / Carrier

​

Product quantities are correct?

  • If not, escalate to OPS Team

​

There's any special observation / request?

  • Escalate to OPS Team

​

After confirming that everything is correct, close the ticket.
