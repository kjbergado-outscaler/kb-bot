Checking the Order in Magento

  • Open Magento and search for the order using the external order number;

​

​

  • Roll the page to the Purchase Orders section;

  • Review the order status:

    • Fulfilled & Delivered → Proceed to Delivered Order.

    • Unfulfilled & Not Uploaded → Proceed to Order Not Uploaded in Magento.

    • Tracking Issue → Proceed to Order Tracking Issues.

​

​

Delivered Order

Inform the Customer

  1. Confirm the delivery date in Magento and in the Courier portal.

  2. Reply via Sprinklr or Teams, stating that the order has been successfully delivered on [date].

    • Include POD/tracking details.

Make sure to add the Store e-mail in CC: emmastore-westfieldlondon@emma-sleep.com / emmastore-manchester@emma-sleep.com

Order Not Uploaded in Magento

  1. Verify Shopify to check if the order exists.

  2. If the order is missing in Magento but exists in Shopify:

    • Notify the Store Manager in Teams to escalate to Sam Smith’s team for manual upload, as per Shimul’s instruction.

    • Wait for confirmation from Sam’s team.

  3. Once uploaded, proceed with order verification and inform the customer.

​

​

Items Waiting for Stock

  • If the items are waiting for stock, please let the customer know that the stock has been delayed, and check the expected stock arrival date for the order in the backlog file and update the customer of the expected date. If they are unable to wait, please check with Offline CE as in some cases we may be able to offer an alternative.

  • If the customer would like to split the order and receive the items that are available first, please follow the process of requesting for the WFS items to be rolled back with Order Management.

Order Tracking Issues

If tracking is missing in Magento:

  • Check DB Schenker Portal using the order number.

    • Copy the Order Number;

​

​

  • Log in to DB Schenker Portal;

    • Click Order → View Outbound Orders;

    • Paste the Order Number and select the Data Limit as the first day of the purchase month.

​

​

  • Click on the order → Shipping;

​

​

  • Here, you’ll have the Freight Note (Tracking number) and you can also click on External Tracking Link to track the order.

​

​

  • If DB Schenker fails, check DPD Portal as an alternative.

DPD Tracking

  • Click in the Search option

​

​

  • Paste the Order Number or the Freight Note → Enter

​

​

​

  • You’ll then be able to see the customer’s order process

​

​

UPS Tracking

  1. Go to the UPS Tracking Website.

  2. Paste the tracking number and click Track.

  3. You’ll be able to see the tracking status and download the POD.

​

​

  • Click on "View all shipping details" to check the delivery history.

​

​

Rhenus Tracking

  • Click on “Job Tracking” under the “Tracking” field.

​

​

Review Tracking Status

  • Paste the tracking reference into the “Web Booking Ref.” field.

    • If not available, you can enter the postcode or internal order number in the relevant search bar.

    • Should any information not show upon entering the internal order number, you must place an underscore symbol ( _ ) followed by 1,2,3 after the order number (e.g. 600000000_1).

  • Click “Refresh Data” to update the information.

​

​

  • Scroll down in the window and select “Track Job” to proceed.

  • The POD can be downloaded automatically by clicking on the relevant option in the portal:

​

​

  • If tracking is found:

    • Inform the customer via email with delivery details.

  • If tracking is not found:

    • Contact the Courier’s support via email or phone.

    • Await a response and update the customer accordingly.

Replying to the Customer

  • Draft an email using the WIMO template. Include:

    • Tracking details (if available).

    • Expected delivery date.

  • Send the email and log the response in Sprinklr.

Make sure to add the Store e-mail in CC: emmastore-westfieldlondon@emma-sleep.com / emmastore-manchester@emma-sleep.com
