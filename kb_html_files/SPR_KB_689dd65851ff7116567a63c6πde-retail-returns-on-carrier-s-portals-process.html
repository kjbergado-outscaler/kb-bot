Customer Service Process

Customer Service (CS) agents manage the returns of the products by acquiring all relevant and necessary information of the final customer from the respective B2B client. Following the previously mentioned logic, the CS agent then goes to the required carrier for the process, logs-in to their platform and manually creates a collection of returned products.

Booking Pick-ups

This process is done when the customer requests a pick-up.

Start by clicking on the “Abhol- / Rückholauftrag” option.

​

​

​

Enter Customer Information

  • In the “Abholadresse” section, enter the customer’s details.

    • Required fields:

      • Name

      • Address

      • Country

      • Postal Code

      • City

      • Contact Person's Name

      • Telephone Number

      • Email Address

    • For the “Adresse GLS-Kunde” field you need to select the second option that is called “Produktion-Service GmbH”.

    • Ensure that the “Services” field is set to “Pick&Return Service”.

    • Click Continue to proceed.

Confirm and Complete the Information
On the next page:

  • Verify that the customer’s details are correct in the “Abholadresse” section.

    • Enter the package weight and both the internal and external order numbers.

Return Label

This process is followed when a customer wants to return a product that is still in its original packaging and opts to deliver it to a parcel shop. This applies to all retailers.

Start by clicking on the “Retourenpakete” option.

​

​

​

Warehouse Information
The first set of information “Versender” needs to be selected from the dropdown list by selecting the second option called “Produktion-Service GmbH”.

​

​

​

  • Enter Customer Information

    • In the “Absenderinformationen der Retourenpakete” section, fill out the following mandatory fields:

      • Name

      • Address

      • Country

      • Postal Code

      • City

      • Contact Person’s Name

      • Telephone Number

      • Email Address

  • Enter Order Numbers

    • In the “Referenz-Nr” field, enter both the Internal and External Order Numbers.

    • If you need to add an additional order, click “Weitere/s Paket/e versenden” to include it.

​

  • Create the Return Label
    Once all the required information has been entered:

    • Click “Paketschein erstellen” to generate the return label.

  • Save the Return Label

    • Important: Save the return label before closing the PDF, as it cannot be retrieved again afterward.

  • Record the Package Number

    • In your FluentCommerce system, always include the “Paketnummer” (package number) reference in the comments section for tracking purposes.
