Overview

While Moebel Martin does not have an internal portal and there are no specific guidelines for email formats, this document details the steps required for when transmission errors occur. This happens when an order is initially placed on the retailer’s side but then cancelled. FluentCommerce prevents the creation of duplicate orders using the same external order number. In such cases, we must request manual order creation.

Review the e-mail

  • Open the attached document.

  • Identify the external order number: you can find it where its marked in red below.

​

Search for the Order in FluentCommerce

  • Open FluentCommerce and search for the external order number provided in the email.

  • If the order is marked as cancelled in FluentCommerce, proceed with the following step.

Open a Jira Ticket for Manual Order Creation

  • Create a Jira ticket requesting the manual creation of the order, providing all relevant details from the email and attached document.
