If customer change their mind and would like to return mattress (within 14 days or 100/200 nights period)

  • ask if the mattress is packed or unpacked and reason for return. You can use macro PostD-Returns-Manual Return-Ask for Reason. Remove wording whether is too soft and firm and that it takes 4-6 weeks to adjust to new mattress (as they have only 14 days to return the mattress and we don’t promote comfort layer offering with these types)

Unpacked mattress return:

In Magento click on Return Products and create return and select “No refund”. Don’t click on “Product back at warehouse” as it needs to be collected.

Send customer a standard macro for PostD-Returns-Manual Return: Out of box mattress – Wowcher (in ths macro we ask customers to let us know once their mattress has been collected so we can speed up the refund process. In fact this is the only way we will now it’s been collected as we need to request the refund manually)

Packed mattresses (return label)

We create return label for mattresses which are still in box unopened (new Essential).

  • Email customer and ask them when is good time to pick up the mattress (Mon-Fri from 2 days onwards). You can use macro PostD-Returns-Manual Return-Still in Box.
  • Once you have this information please email Stacey if order originated in B&A warehouse to help you create a return label for a consignment (and please include the tracking number in the email)

  • Once the collection date is arranged please ask customer to confirm when collection is completed so we can make sure refund will be send without delay. When the collection is complete please create Return Products in Magento click on “No refund”, “Packed” and order “Product back at warehouse”. Advise customer refund should be processed within 3-10 working days to original payment method.
