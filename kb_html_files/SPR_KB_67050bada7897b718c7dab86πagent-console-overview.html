Icons on the left-hand side menu

As per the image above, you have 5 options on the left-hand side menu:

  • Home: It takes you to the Agent Persona homepage, where you can check metrics related ti the cases, announcements you might have missed, etc.

  • Console: It is the Agent Console, the main tool that you will use to handle voice, email and live chats cases.

  • Reports: It shows a dashboard where you can see the number of cases assigned to you, average case responses, handling time, etc

  • Create Email Case: To be used for Outbound Emails ONLY if the customer don't have already a case with us. For example: It is an escalation case that was assigned to you via JIRA and now you have to contact the customer via e-mail.

  • Knowledge Base: It shows all the folders with KB articles about processes, products, tools, etc.
