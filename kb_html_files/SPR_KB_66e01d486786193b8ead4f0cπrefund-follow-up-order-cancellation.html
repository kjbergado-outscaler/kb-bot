Overview


For order cancellation with refund request in Magento, orders will be cancelled in 3-5 business days and customers will receive the refund within 14 business days (2-3 weeks for credit card orders) depending on the payment method used. (Refer to the table below for the refund timeline.)



This article will help the Customer Service Agents to handle refund follow-up for Order Cancellation.


Checking the Order Status in Magento


If a follow-up ticket was received from the customer regarding refund for order cancellation, first thing you should check is the current order status. For order cancellation, there are different possible order status:

  • Canceled
  • Cancel requested


Handling of the refund follow-up depends on the order status.


  • Order Status - Canceled




If the order status is already "Canceled", it means the refund was already done and the refund should have received by the customer within 14 business days (2-3 weeks for credit card orders) depending on the payment method used after the refund is processed in Magento.


Confirmation of refund can be seen on the comments history:



If the refund is still not received by the customer within 14 business days (2-3 weeks for credit card orders) after the refund was processed in Magento, escalate the ticket to Refunds Emma and use the Refund Request Template. 



If a failed refund note is seen on the comments history, escalate the ticket to Refunds Emma and use the Refunds Escalation Template. 



For partial cancellations, order status will revert back to Payment Received order status after the partial cancellation is processed. If this is not the case, escalate the ticket to Refunds Emma to revert the order status back to Payment Received.



  • Order Status - Cancel Requested w/ Invoice or Shipped


Once the order cancellation is initiated in Magento, order status changes to "Cancel Requested". If the order status is stuck in Cancel Requested for more than 5 business days, there are two possible reasons:


  • Order has an invoice - if the order for cancellation and the invoice was already created, the refund cannot be processed via cancellation. A product return (PR) refund request has to be created. Once the PR refund request is created and all of the PR refund requirements are met, the refund will be processed within 3-5 business days.



  • Order is tagged as shipped - if the order for cancellation and the order is already tagged as shipped, the refund cannot be processed via cancellation. An invoice (Manual Invoice Creation - Magento) and product return (PR) refund request has to be created. Once the PR refund request is created and all of the PR refund requirements are met, the refund will be processed within 3-5 business days.


  • Order Status - Cancel Requested w/o invoice and not shipped


Once the order cancellation is initiated in Magento, order status changes to "Cancel Requested".



The refund for the order cancellation should be processed within 3-5 business days; however, there are instances where the cancellation is not automatically processed by Magento.


If the order is on "Cancel Requested" status for more than 5 business days or customer still hasn't received the refund for more than 14 business days, escalate the ticket to Refunds Emma and use the Refund Request Template.
