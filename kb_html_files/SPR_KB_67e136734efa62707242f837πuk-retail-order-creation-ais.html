Creating the Order:

  • Review the received POs via email.

  • Copy the PO into the Drop shipment Folder. You can find them here: To Do

​

​

  • The BO team will retrieve orders from the folder and update them in Fluent.

Make sure you copied the correct PO number.

​

  • Orders are tracked using this internal tracking sheet where the following fields are filled:

    • PO Number: The purchase order number provided by the retailer.

    • SKU: The specific item’s stock-keeping unit (e.g., mattress with dimensions).

    • Order date: The date the order was placed

    • Lead Time Promise Date: Usually set to 14 days based on replenishment timelines.

    • Discount Code: If applicable, for free mattress covers.

Once an order is successfully created, ensure the Fluent Commerce order number is added into the tracking file.

  • Use the external order reference to check if the order is already created in Fluent Commerce.

  • Update the status to "Complete" and "Delivered" once the order is confirmed as delivered.

Closing the Sprinklr case

  • Click on the three dots → Apply Macro → Assign to Me

​

​

  • After the case is assigned to yourself, add a comment indicating: “ORDER CREATED - NO RESPONSE REQUIRED”. Click on the arrow on the right side of the text to Send.

  • The comment will be visible after you send it.

​

​

 

  • Click on the three dots option again → Apply Macro → No Response Required

Tracking the Order:

  • Daily Review: Each day, review the order tracking file for updates and status changes.

    • Look for changes in status, such as “Delivered” or “Declined.”

    • If there are any changes in delivery (e.g., a delivery is declined by the customer), reach out to Furniture Village for clarification and to assist in resolving the issue.

    • The status of the order will change (e.g., from red to green) once the order is successfully created, indicating the process is complete.
