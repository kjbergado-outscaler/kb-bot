Open Products

When a customer requests to return an open mattress, packaging material must be sent to them before the carrier can collect the item. After the request for packaging material has been completed, the collection of the returns must be booked with DHL.

Solvere is our partner for the packaging materials we use to return opened mattresses. When a request is made to them, the CS Agent needs to create and share a GLS label with the customer's address that will be used by Solvere to ship the packaging material.

Recipient: emma@solvereggmbh.de

Frequency: Send one consolidated email at the end of each day for all returns generated that day.

Attachments: All GLS labels created for the packaging shipments (PDF format).

Email Language: German

Email Body Must Include:

  • Customer Name

  • Internal Order Reference (e.g., W-0001-XXXXX)

  • External Order ID

  • Mattress Size (for packaging size)

  • Paketnummer (from GLS label)

Expected Delivery Time: 2–4 working days.

Escalation: If Solvere does not respond or there is a delay, resend the email or escalate to your corresponding Emma Customer Excellence Manager.

Reverse Process Flow

There currently exists two different processes for the different items we sell, one for Germany (DE) and one Netherlands (NL). The main difference between the two countries, apart from the carriers we use, is that for opened products in Germany, the client receives a shipment of packaging material before the collection of the products to be returned.
