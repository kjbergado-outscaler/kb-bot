Scope

This SOP outlines the process for handling customer inquiries, managing returns, and processing refunds for B&Q via the Channel Engine, Fluent Commerce, and Reenus.

  • This task is performed daily.

  • Estimated time:

    • Customer inquiries and issue resolution: 1-2 hours.

Instructions

Customer Inquiry Management

  • Log in to B&Q Customer Support Portal:

  • Portal URL: B&Q Support

  • Login: retail-support@emma-sleep.com

  • Password: Kindly request to your supervisor.

​

​

  • Navigate to the Messages Section to check for new customer inquiries.

​

​

  • Prioritize messages tagged as Critical, as these are escalated by B&Q.

  • Open each customer request and review the details:

  • Product issues (damaged, defective, incorrect size, etc.)

  • Delivery issues (delayed shipment, incorrect address, missing tracking info, etc.)

  • Return and refund requests

​

​

  • Respond to the customer accordingly:

  • If further details are needed (e.g., photos of the damaged item), request them.

  • If an item qualifies for a return, proceed with the Return Handling Process (see Section 2).

  • If the issue involves tracking, retrieve tracking details via Fluent Commerce.

  • If the issue requires escalation, forward it to B&Q Customer Support.

Returns Process

Identify return requests from customer messages.

​

​

Retrieve Order Details:

  • Copy the External Order Number from the message.

  • Search for the order in Fluent Commerce.

​

​

​

Verify the return eligibility:

  • Ensure that the request follows the B&Q return policy.

  • If the product is damaged or defective, confirm evidence (photos, videos).

Log return details in the B&Q Return Tracker:

  • Customer Name

  • Address

  • Product SKU and Quantity

  • Return Reason (e.g., Damaged, Change of Mind, Incorrect Item Received)

​

​

Initiate the return request in Reenus:

  • Open the Reenus Return Management Portal.

  • Enter the customer details.

  • Assign the return to the appropriate warehouse.

  • Schedule a pick-up date.

  • Once a return request is scheduled, update the B&Q Return Tracker.

  • Confirm the pick-up with the customer via the B&Q portal.

Cancellation Process

​

Cancel in Fluent Commerce (FC):

  • Orders must be cancelled before they are dispatched to Fulfillment Center (FC).

  • Go to the pending order → Click "Cancel order item" → Input quantity and reason → Submit.

  • Add the cancellation information in this tracker (Cancel Sheet): MKP CS TO OM request for channel engine support.xlsx

If cancellation in FC is not possible, CS must fill out the Cancellation Request File and notify Shimul.

Cancellation Request File

  • Access the file here;

  • Use this file when cancellation cannot be completed via FC and notify Shimul;

  • He will then proceed with the cancellation.

  • Fields to fill:

    • Order number

    • External order number

    • SKU to cancel

    • Fulfillment reference

    • Return to sender status

    • Tracking number

    • Carrier code

    • Reason for cancellation

  • Color Codes:

    • White: Not processed

    • Orange: Ticket raised

    • Green: Completed

Note: If an order has multiple items that need to be cancelled, fill out one line by Fulfillment/SKU.

​

Cancel on the B&Q Portal:

  • Log into the B&Q portal.

  • Locate the corresponding order.

​

  • Select the “Cancel” button and choose between “full cancellation” or “partial cancellation”.

Refund Processing

  1. Check collected returns in Rhenus.

  2. Identify items collected but not refunded.

  3. Log in to B&Q Commerce Hub.

  4. Search for the External Order Number to confirm the refund eligibility.

  5. Cross-check with Fluent Commerce to verify order completion.

  6. Click Submit Refund in Commerce Hub.

​

​

​

Verify the refund processing status:

  • If funds are available, the refund is processed immediately.

  • If funds are not available, notify the customer that the refund may take longer.

  • Update the B&Q Refund Tracker with the status.

Key Notes & Special Handling

  • Warranty - All Refurbished products have a 6 month warranty. New products have a 10 year warranty.

  • B&Q Support Workflow: Customers first contact B&Q support, which may redirect them to Emma Sleep’s retail support team.

  • B&Q Messages Visibility:

    • Messages appear in B&Q Portal and Sprinklr.

    • If a message appears in Sprinklr but is already answered in the B&Q portal, close the Sprinklr ticket.

  • Tracking Information Requests:

    • Retrieve tracking details from Fluent Commerce.

    • If the item has not been shipped yet, escalate to B&Q Logistics.

  • Refund Delays:

    • Standard refunds are processed in 7-10 business days.

    • If no funds are available, inform the customer of potential delays.

  • Critical Cases:

    • Any case marked as Critical in B&Q must be addressed urgently.

    • If unresolved within 24 hours, escalate to B&Q Support & Murat.

  • Partial Refunds:

    •  To refund an order partially:

      1. In the back office sidebar menu, select Orders > All orders > Orders to Process (status).

      2. Click an order.

      3. On the order details page, click the drop-down button next to the Refund button.

      4. Click Refund part of the order.
        The "Refund part of the order" page appears.

      5. Select the quantity to refund and the refund reason. For partial refunds, select the quantity 0.

      6. Select the amounts included in the refund.

      7. You can specify the amount to be refunded via the Specify amount to refund drop-down link.

      8. When done, click Refund.
        The "Confirm refund" window appears.

      9. Click Confirm.

       

      On the delivery options the only settings currently allow you to have either a next day service (or standard service) (Can’t be both) & Click & Collect if this is applicable. We don’t have tiered charged delivery options at present.

       

      Where possible, it would be best to try & offer customers the fastest delivery time possible.
