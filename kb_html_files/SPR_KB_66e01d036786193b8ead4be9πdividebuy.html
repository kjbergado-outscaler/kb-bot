New payment plan, which is live on our website;

To familiarize ourselves with this payment plan

  1. Customers can now have a plan with a minimum of 50 pounds unlike Klarna which was 279

  1. Customers can select the varying installment plan based on items purchased, However with 50 pounds customers can apply for the 3 month plan.

Status Update

  • Status differ from Klarna, Dividebuy does not have the Accept, sending links to complete application and all other processes
  • We do not see decline customers in Magento, Dividebuy communicates this to customers
  • All Dividebuy orders either have status as processing or payment received and these orders have been processed and can be shipped
  • If you cannot pull an order Magento, please reach out to me.

Return Process

  • Customers can now return any item from their order regardless the open amount left to complete the installment plan. This is possible as Dividebuy has the access to disable a plan for an item within an order
  • We do not need to cancel an entire order or request for whole order return to process a refund.
  • Create normal returns same procedure in Magento

Refund and Discounts Process

  • Refunds are created in Magento as there is no additional portal for Dividebuy. When a return is complete, customer will receive a refund
  • Discounts are created in Magento, same procedure for regular payment methods
