Note: The partnership between Emma and Conforama ended on June 13th, 2024. From this day on, if for any reason there are orders placed by them, we should not process them. If this happens, we can contact the company to inform that order will not be processed.

Do not hesitate to escalate the situation so that Emma can contact the stores and explain the situation.

Conforama Orders

Conforama orders enter via EDI, so no orders should be created manually. The orders can take up to 24h to pass from Conforama's System to Emma's system. If after this time the order is not available in Magento, the issue should be escalated to Emma.

​

Search orders by the PO Number here: Status_Orders_Conforama.xlsm

​

If you check that the order is in "Packing" for a long time, then the SKU is probably wrong, and we'll not be able to deliver the item. You can check the correct SKUs in PT B2B - SKUs.xlsx.

​

Orders in "Packing" with wrong SKUs can also have other particularities like a random delivery date and "Out of Stock Dummy Warehouse" assigned:
