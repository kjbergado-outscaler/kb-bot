Scope

This SOP outlines the step-by-step process for handling customer inquiries and other tasks for Wowcher via Sprinklr, Fluent Commerce, the Wowcher Merchant Portal.

This task is performed daily. Additional time may be required if customer inquiries require manual order searches or corrections (around 45 minutes).

​

Instructions

Customer Inquiry Handling via Sprinklr

  1. Access Sprinklr and check the latest customer inquiries.

  2. Identify if the inquiry is from Wowcher or LivingSocial (all use the same email system).

  3. Retrieve the voucher code from the customer inquiry.

  4. Log in to Fluent Commerce and filter by External Order Reference to locate the order details.

    • URL: Fluent Commerce Portal

  5. Check the order status and any previous agent comments to avoid duplicate responses.

  6. If the case was previously handled, tag it as “No Response Required” and close it.

  7. If the issue is unresolved:

    • Check for stock availability in the internal Teams group chat.

    • If the SKU is out of stock, offer the pre-approved upgrade SKU.

    • Document the resolution and respond to the customer.

  8. Apply the correct Sprinklr case tag:

    • Logistics: Incorrect SKU received, delayed order, etc.

    • Verification: Ensure voucher code is provided.

    • Status: Waiting for response, resolved, or closed.

Handling Cancellations, Returns, Refunds, and Product Claims

Cancellations

Cancel in Fluent Commerce (FC):

  • Orders must be cancelled before they are dispatched to Fulfillment Center (FC).

  • Go to the pending order → Click "Cancel order item" → Input quantity and reason → Submit.

  • Add the cancellation information in this tracker (Cancel Sheet): MKP CS TO OM request for channel engine support.xlsx

If cancellation in FC is not possible, CS must fill out the Cancellation Request File and notify Shimul.

Cancellation Request File

  • Access the file here;

  • Use this file when cancellation cannot be completed via FC and notify Shimul;

  • He will then proceed with the cancellation.

  • Fields to fill:

    • Order number

    • External order number

    • SKU to cancel

    • Fulfillment reference

    • Return to sender status

    • Tracking number

    • Carrier code

    • Reason for cancellation

  • Color Codes:

    • White: Not processed

    • Orange: Ticket raised

    • Green: Completed

Note: If an order has multiple items that need to be cancelled, fill out one line by Fulfillment/SKU.

Returns

  1. Customers must receive the product before requesting returns/refunds.

  2. No direct replacements except for faulty mattress covers (if stock is available).

  3. Return & Refund Policy:

    • 14 days for refurbished mattresses.

    • 200-night trial period for new mattresses.

    • 10-year warranty for first-life mattresses.

    • 5 year warranty for refurbished mattresses.

Note: If a customer insists on a longer warranty period for a refurbished mattress, request a screenshot showing the product advertised with a 10-year warranty on Wowcher. If the customer provides valid proof, we must honor the extended warranty.
