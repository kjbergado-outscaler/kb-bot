Implementation Details

  • A new panel has been added to the "Set Email" guided path.

  • Agents will now see three options when setting an email:

    1. Case Level Only – Updates the email only for the selected case without affecting the profile.

    2. Profile Level Only – Updates the email for the profile but keeps the case email unchanged.

    3. Both Case and Profile – Updates the email for both the case and the profile.

How to Use the Updated "Set Email" Function

  1. Navigate to the case requiring an email update.

  2. Click the Set Email button located at the top right of the page.

  3. Select one of the three options based on the situation:

    • Case Level Only if the email should change only for this specific case.

    • Profile Level Only if the customer's email should be updated for all cases under their profile.

    • Both Case and Profile if the email correction applies to both levels.

  4. Confirm the selection and save the changes.
