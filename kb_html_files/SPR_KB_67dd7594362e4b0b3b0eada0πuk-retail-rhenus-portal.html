Scope

This SOP outlines the process for managing order tracking, delivery status, cancellations, and obtaining proof of delivery (POD) using the Rhenus Carrier Portal.

​

Process Flow

Accessing the Rhenus Portal

  1. Log in to the Rhenus Portal:

    • Portal URL: Rhenus Tracking

    • Primary Credentials:

      • Username: Emma

      • Password: Kindly ask your supervisor.

    • Alternative Credentials (if orders not found):

      • Username: Emma

      • Password: Kindly ask your supervisor.

  2. Navigate to the Tracking Section:

    • Click on Job Tracking under the Tracking menu.

​

​

Order Tracking and Status Check

​

Search for an Order:

  • Enter the Traking Reference in the Web Booking Ref field or Booking Reference in the Your Reference field.

​

​

​

​

Review the Order Status:

  • Possible statuses include:

    • In Transit

    • Delivered

    • Awaiting Collection

    • Failed Delivery

View Delivery Details:

  • Click on Track Job to view the delivery timeline.

​

​

Managing Delivery Actions

Cancel or Amend Delivery:

  • Select Cancel Order from the Action Menu.

​

​

Request Redelivery:

  • For failed deliveries, use the Change Service or Hold Order options.

Obtain Proof of Delivery (POD):

  • If available, click View POD under the Search Results.

​

​

​

Locating Consignment Numbers in Magento

Find the Rhenus Tracking Number:

  • Magento displays the Rhenus tracking number, which is used for searching orders in the Rhenus portal.

Search by Tracking Number:

  • Input the tracking number in the Rhenus search field to retrieve order details.

Additional Functions

Map View of Delivery Route:

  • To view the delivery route, click on the Map Icon under Tracking Info.

Submitting a Cancellation Request:

  • Fill in the Reason for Cancellation and Submit.

For Issues or Inquiries:

Send an email to emma@uk.rhenus.com with the Order Reference, Customer Information, and a Brief Description of the Issue.

Important Information

For updating the delivery address, you will be able to change the address on the customer's order until it is booked.

You can put the order on hold at any time and also release the order. However, if for any reason, the portal won't give you the option to make amendments, you can email over or send an internal message and Rhenus can action it for you.

For collection and delivery job orders:

The COLLECTION JOB ORDER will be automatically canceled if they haven't received the customer's confirmation after 14 days.

As for the DELIVERY JOB ORDER, they will cancel it if the customer does not confirm the schedule after 28 days.

Return Collection Requests (End Customer)

This is how agents can create a request for Rhenus to collect returns from end customers.

If the product has been opened, the return must be requested through Rhenus.

Instructions

  • Login into the Rhenus Portal for Emma Offline.

Website Link: Login | Rhenus Home Delivery

Username: Emma

Password: Please reach out to your Supervisor to request this information

Category

Details

 

Order Format

W-0006-xxxxx or M-0006-xxxxx or 60xxxxx

W is for Wholesale Orders

M is for Marketplace orders

60 is for Emma O&O orders

  • Navigate to: UPLOAD JOBS → UPLOAD SPREAD SHEET

Prepare the Upload File

  • Fill in the spreadsheet using the following mandatory guidelines:

Field

Instruction

Order Date

Must be in this format: dd.mm.yyyy → e.g., 04.01.2025

Your Reference

Enter the Order Number or Customer Reference

Retailer

Always set to Emma

Contact Name

Enter the customer's full name

Address

Enter the complete delivery/collection address

Postcode

Enter the valid postcode

Telephone Number

Must start with the number 7 (e.g., 71234567890)

SMS Number

Same as Telephone Number (must be a mobile number starting with 7)

Email Address

Enter the customer’s email address

Service Level

Use Bronze for returns to Emma

Type of Delivery

Always set to Collection

Product Description

Enter the name of the item to be collected

No. of Actual Boxes

Always set to 1 (If more than one quantity of the same item, use another column)

Total Volume

Default is 0.9 (unless exact volume is known)

Total Item Weight

Default is 25 (unless exact weight is known)

Save & Upload the File

  • Save your completed spreadsheet file.

  • Go to the Watched Video section (within the Rhenus platform).

  • Click on Upload File, then select your saved spreadsheet.

  • Click Validate to process the file.

Error Handling

❌ Common Error:

"THIS REFERENCE HAS ALREADY BEEN USED FOR A PREVIOUS UPLOAD"

  • This means the same order number was already uploaded for a previous job.

  • Solution: Add a suffix like 00, 01, 02 at the end of the reference number.

    • Example: 602342544 → 60234254400, 60234254401, etc.

❌ Invalid Phone Number:

  • Ensure both Telephone Number and SMS Number:

    • Are identical

    • Start with the number 7

    • Are valid mobile numbers

❌ Missing Contact Information:

  • At least one contact method is mandatory:

    • A valid mobile number or

    • A valid email address

WARNING: If neither is provided, the system will reject the file.

✅ Checklist Before Upload

  •  Date is in dd.mm.yyyy format

  •  Phone number starts with 7 and matches SMS number

  •  Email or mobile is present

  •  Order Reference is unique or has a suffix if reused

  •  Service Level is correctly selected (Gold or Bronze)

  •  Total Volume is set (default 0.9)

  •  Weight is set (default 25)
