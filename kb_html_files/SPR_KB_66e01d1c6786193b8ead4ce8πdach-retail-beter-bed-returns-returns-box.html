1. Every Thursday or Friday morning we receive an e-mail from Soraya with the returns file. We save the file once and open it.

2. we place the saved file in the respective month of the folder Beter Bed -> Retourenabholung (Returns collection) - the link can be found in the weekly guide under Returns.

3. Then we create a delivery bill. To do this, we can open a previous delivery bill in Word and adapt it.

Attention: Always press "Activate editing", otherwise it won't be possible to edit the document.

4. Here we adjust the following:

Pick-ups are always scheduled for Tuesday (Uden) and Wednesday (Hoogeveen and Vennep). The drop-off is therefore always one day later.

In the subject line we update the week in which the return will be collected. Here in CW27, the file date is the day of collection of loading point 1, in this case 13.07.

Loading points 2 and 3 are collected one day later, here on 14.07.

And the drop-off at PS one day later on 14 / 15.07.

The entire returns list must then be copied once from the Excel list and entered into the Word file as a graphic.

Then we drag it so that everything is visible and save this delivery bill once as a Word file and once as a PDF. We then move it back into our folder.

5 We will then enter the return in the checklist:

6. We inform Dusan by e-mail that he can organize the return collection and put Beter Bed in CC.

Please do not forget to attach the PDF of the delivery bill.

7. We can then also inform Soraya and send the delivery bill.

And always send from Beter Bed and put Beter Bed in Cc :)

8. The Returns Box is best processed once a day:

For this we find the returns by mail via the Inbox Returns bedtime.

For this we need the folder Return -> Beterbed -> the respective month.

9. Drag the files from the mail into this folder in the respective month.



10. in the mailbox, we then move the mails to the folder with the respective month after assignment.

11. these mails Betalingsadvies come in the corresponding folder
