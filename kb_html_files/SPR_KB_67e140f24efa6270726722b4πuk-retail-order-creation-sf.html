This document outlines the step-by-step process for handling Sterling Furniture order creation.

This procedure applies to all employees handling Sterling Furniture orders, including downloading order forms, verifying details, updating trackers, and responding to inquiries via Sprinklr.

​

Instructions

Checking for New Order Forms

Open Outlook and Check Emails

  • Search for new order forms on Sprinklr with the Sterling Furniture email address: customerorders@sterlingfurniture.co.uk

  • Verify if an updated order form has been sent.

​

​

​

​

Downloading and Uploading Order Forms

Download the order form from the email.

  • Save it in the designated folder under the appropriate category.

  • Locate the PO (Purchase Order) number.

  • Cross-check the order date to ensure it's current.

​

​

 

Upload the Order Form to the Tracker

  • Navigate to the Sterling Furniture tracker. Upload the order form file will all relevant information.

    • This is our internal tracker and it must always be updated.

Identifying the Correct Store and Warehouse

Check the Order Form for Branch Information

Each store has a unique code. If the order is directly from the store, use the corresponding store code.

  • If delivered to a warehouse, identify the correct email and phone number associated with that warehouse.

Communicating with Sterling Furniture

Weekly Updates

  • The Sterling team will request the Order Book weekly.

  • Copy all orders from the Sterling spreadsheet and paste it in a new one;

  • Include any new PO numbers received.

  • Send an email to Sterling Furniture summarizing all orders.
