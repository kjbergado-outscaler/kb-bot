Drop Shipment Order Intake Process (UK/IE)

Scope

This SOP outlines the process for handling drop shipment Purchase Orders
(POs) received from retailers in the UK and Ireland, from initial receipt to
order upload and file management. This procedure applies to all Customer
Service (CS) and Backoffice (BO) agents involved in the drop shipment order
intake process.

  • Frequency: Daily

Point
of Contact: Shimul Talukder

Responsibilities

·       Customer
Service (CS) Agents: Receive POs from retailers via email. Save POs as
files (PDF, document) in the "To Do - Drop Shipment" folder.

·       Backoffice
Agents: Process POs in the "To Do - Drop Shipment" folder.

Instructions

1. Open the "To Do"
Folder

1.    Navigate to the shared folder "Order Intake BO_UK". Inside, locate the "To Do" folder. Open the "To Do - Drop Shipment" subfolder.

2.    Identify the files available. If there are
multiple files, process them one by one.

3.    If any PO lacks an external number, escalate to Shimul Talukder for
resolution.

2. Order Entry into Excel

1.    Open the Excel template provided for order intake.

2.    Copy the template and create a new file with the
current date in the following format:
UK_DS_YEARMONTHDAY
Example: UK_DS_20250331

3.    Enter the order details from each PO into the
newly created Excel file.

Ensure each order is recorded in a new row in the Excel
file.

4.    Ensure accuracy when inputting retailer
information, product details, quantities, and order numbers.

5.    Once all necessary information is entered,
proceed with the bulk order intake.

6.    Confirm successful upload before moving
to the next step.

3. Move Processed Files to
"Done" Folder

1.    Inside the "Done - Drop Shipment" folder, create a new subfolder with the
current date in the format:
UK_DS_YEARMONTHDAY
Example: UK_DS_20250331

2.    Move the processed POs into this newly created
date folder.

3.    Save the used Excel file in the
same date folder.

If applicable, confirm the order was placed in the correct subfolder (Drop
shipment vs. Truck shipment).
