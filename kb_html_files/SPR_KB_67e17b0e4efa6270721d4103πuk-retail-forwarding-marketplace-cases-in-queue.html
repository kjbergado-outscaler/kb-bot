This is the process agents should follow when receiving cases from marketplaces in their queue that are not part of Offline (e.g., Amazon).

Forward Case

  • The agent must forward the case to the following email address: cs-marketplaces@emma-sleep.com.

    • Make sure that all necessary details and information are included when forwarding the case.

Close the Case in Sprinklr

  • Once the case is forwarded, the agent must proceed to close the case in Sprinklr.
