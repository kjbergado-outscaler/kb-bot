How to create manual returns for Relais Colis

  1. Open Magento

  2. Click the 'Return prodcuts' button on the top right of the order page

  3. Fill all the required fields

    1. Return products: product will be skipped if checkbox not set 

    2. Quantity: quantity of product for return

    3. Quality: Available options: packed,unpacked, faulty

    4. Refund to the customer: Available options: full refund, no

       Depending on selected value, an email template will be choosen

    5. Pickup/Return-service: Available options Based on Manualreturn assignment rules

      For FR orders, this will usually be 'Relais Colis Return Carrier'

    6. Carrier Info:

      1. Track number Track

      2. Link

    7. Warehouse Address: Warehouse address for returned product

    8. Reason: Available options:

      1. The item is received too late

      2. Product is too firm

      3. Product is too soft

      4. Cannot adjust the height for myself

      5. Product is too warm, cause sweating

      6. The cover does not have a nice touch

      7. Ordered wrong product size

      8. Other reasons (the comment is compulsory)

      9. Ordered twice

      10. Product Claim

      11. Logistic Claim RTS / Cancellation Reason of return

    9. Comment: Comment to return

    10. Pickup Date Pickup Date if available

    11.  Credit Memo Comment(B2B only!)

    12. Credit External #(B2B only!)

    13. Product back at warehouse: If set yes, return will be moved to complete status

Afterwards, returns on Fluent Commerce will be automatically tracked and updated on the system. Returns on Magento will have to be tracked on the Relais Colis portal and manually updated on the order management system.

Note: Note: for Magento returns, the carrier’s tracking number must be added to the ticket whenever possible.

If the return is for an FR store and has 'Expected' status, then Relais Colis automatically exports the order and sets the status as 'Status_Relais_Colis_Sent'.

What to do when there are return issues (Relais Colis)

If a customer is facing return issues, first check the status of the return in Relais Colis’ portal.

If the return cannot be found in the system, needs to be adjusted, is stuck or have a false status in the system, first contact the customer service of Relais Colis:

1. Load a case in your software webenseigne

2. In a second time you can send a message to src@relaiscolis.com with in object for each issue the tracking number of the shipment (LS+ 10 digits in your case)

3. Call the +331 72 08 00 60. This number is dedicated to our customers and should not be communicate to your final customers.

If the issue cannot be resolved through contact with the carrier’s customer service, please escalate to Emma’s internal teams so that it can be thoroughly investigated and resolved adequately.

Remember to always keep the customer updated and ensure that we are doing our best to sort out the issue.
