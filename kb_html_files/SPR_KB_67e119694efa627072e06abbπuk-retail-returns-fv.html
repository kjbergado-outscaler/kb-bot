Conditions for Accepting a Return with Refund

  1. Product Claim:
    A return with refund can be processed if the customer requests a refund instead of a new product.

  2. Incorrect Delivery:
    If there has been a wrong delivery, a return with refund may be issued upon the customer’s request.

  3. Undelivered Mattress (as per GLS tracking):
    If the mattress is marked as delivered by GLS but the customer has not received it, and they have signed the RVE document, a return and refund can be processed upon customer request.

  4. Failed Delivery (Returned to Sender):
    If the mattress was unable to be delivered and has been returned to us, a return and refund may be processed if the customer requests it.

Truck Shipment Returns

This is the process for handling truck shipment returns from Furniture Village, which includes tracking customer requests, processing returns, and ensuring credit memos are generated for completed returns. Returns initiated from truck shipments are considered more complex due to the additional logistics involved. It’s crucial to manage these returns promptly to avoid delays or miscommunication.

​

Receiving Return Requests

  • FV reaches out to Emma with the SRF form containing PO number and customer name > Emma reviews the claim and upon approval, provides the debit authorization reference which is the same as the return reference on Fluent when generating a Credit Memo.

  • Review the PO number to determine whether it pertains to a truck shipment return.

    • Example of Truck Shipment PO number: 9899XXXXXXXX.

    • There is NO NEED to ask for the WH collection details as only the return reference is needed at this point.

  • Look up the associated order using the PO number in the internal tracking system.

Tracking and Updating the Spreadsheet

  • Upon receiving a return request, update the internal tracking spreadsheet (“Returns” sheet) with the following details:

    • Date

    • Agent’s Name

    • Emma Order Number

    • PO Number

    • SKU

    • Customer’s Name

    • Return Order Reference (Found in the “Related” tab in FluentCommerce)

      ​

    • Status

  • Use the internal spreadsheet to manage the order and return references for future tracking.

All returns must be accurately tracked in the spreadsheet, with complete order information, to make sure to do proper follow-up.

Return Processing

 

OLD PROCESS: FV collects the order from the customer and arrives back at FV's smaller warehouses (RDC) > FV reaches out to Emma to request collection of items 

 

NEW PROCESS: FV collects the order from the customer and arrives back at FV's main warehouse (NDC) > FV reaches out to Emma to request collection of items on a weekly basis.

 

Note: FV will now send Inna a list of items to be collected on a weekly basis from their NDC instead of sending one by one queries to CS for multiple collections from different warehouses
