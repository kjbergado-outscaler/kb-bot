This SOP details how to manage the weekly GLS returns list.

Instructions

Managing the Returns List

  • Every Friday, a list of returned items is received from Kühne und Nagel.

  • The list includes tracking numbers, but not always order numbers and fulfillment numbers.

Validating and Booking Returns

  • Check if the order number or fulfillment number are available in the spreadsheet.

  • If the any of these numbers is available:

    • Open Fluent Commerce and search for the order.

    • Double-check if the Order Number, Fulfillment Number, SKU and Tracking number match.

    • Check if a return is already booked.

    • Always check the Comments and the case history.

    • Ensure the correct process is followed before booking the return.

Example: It might be possible that a mattress was returned/the wrong product was sent, and a replacement was sent. If the return is also booked on that case, the customer would receive both a refund and the supplement, so this shouldn't be done.

  • Track the order in the GLS platform to confirm if the product was successfully returned.

    • Please refer to this article for instructions on how to track orders.

  • If applicable, create the return in Fluent and mark it as organized and complete.

  • Mark the row in green after the process is done.

When Not to Book a Return in Fluent Commerce

  • If the customer still possesses the main product and only a supplement (e.g., cover) was returned.

  • If the return has already been processed in the system.

  • If the issue is related to an undelivered supplement that does not impact the main product.

  • If the order is still in transit and has not been confirmed as returned.

If the return was related to Otto, you also need to register it in the Diva Portal. Refer to this article for detailed instructions.

  • If the order/fulfillment number are not available:

    • Track the order in the GLS portal;

    • Copy the fulfillment number;

    • Search for the order in Fluent Commerce.

      • Select the order and cross-check all relevant information.

      • Proceed with the relevant analysis and procedure.

If a supplement order was issued (e.g., a mattress replacement), do NOT process a return for the original item, as the customer would receive the new product and the refund.
