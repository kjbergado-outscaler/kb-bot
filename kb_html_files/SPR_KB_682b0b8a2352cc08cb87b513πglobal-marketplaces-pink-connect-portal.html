The goal of this SOP is to provide a structured guide for managing orders, messages, cancellations, and returns in the Pink Connect platform for Veepee orders.

​

Login Information

Portal link: Pink Connect

  • Use your personal login credentials.

​

​

Navigation Overview

Once logged in, you'll have access to:

  • Inbox (customer messages)

  • Orders (new, processing, and historical)

  • Returns

  • Tracking

  • Delivery Address Editing

Use the menu tabs or dashboard cards to navigate between sections.

​

​

Order Management

Viewing Orders

  • Go to Orders > All Orders.

  • Orders will be listed by status (e.g., Processing, Shipped).

  • Click an order to view full details: product, customer, and shipment info.

​

​

Editing Delivery Address

  • Within the order detail view, click the Edit (pencil icon) next to the delivery address.

  • Save changes.

​

​

Note: Use the “Marketplace Order Number” to find the order in Fluent Commerce.

​

Cancellations

  • Order cancellations must be handled first in Fluent Commerce.

  • After cancelling the order in Fluent Commerce, you must cancel orders manually in Pink Connect.

​

​

Returns

  • Return requests are visible under the Returns tab.

  • You can identify returns via icons or flags in the order list.

Note: Like cancellations, returns are processed in Fluent Commerce only. You do not need to take action in Pink Connect.

​

​

Customer Messages

  • All contact/messages should be handled inside Pink Connect.

  • Go to the Inbox to check new or pending customer messages.

  • Select the message you wish to respond.

  • Review and respond accordingly.

​

​

​

Note: Be sure not to open a message unless you're the agent assigned to handle it — once opened, the message moves from the "unread" state.

​

  • You can also message the customer directly in the order, by clicking New Message.

​

​

Order Tracking & Delivery

  • The Tracking section displays shipping status and tracking numbers.

  • If necessary, edit or update tracking info directly in Fluent Commerce.

  • Pink Connect is for viewing only in this area.
