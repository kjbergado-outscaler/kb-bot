Generate Invoices & Shipping Notes

Sometimes, barcodes don't update, and the Invoices and Shipping Notes are not created in Magento. In this cases, we can create them manually.

​

​CAUTION: Make sure ANY barcodes is in Packing / Waiting for Stock status. If we invoice or ship an order with items that are not transmitted to the WH yet, these ones will not be shipped and create a problem in CS & Operations.

​

Create an Invoice

1. Make sure ANY barcodes is in Packing / Waiting for Stock status. If we invoice the order with items that are not transmited to the WH already, these ones will not be shipped and create a problem in CS & Operations.

​

2. Go to the order and click Invoice at the top of the page.

3. Scroll at the bottom of the page and click Submit Invoice.

​

Create a Shipping Note

1. Make sure ANY barcodes is in Packing / Waiting for Stock status. If we ship the order with items that are not transmited to the WH already, these ones will not be shipped and create a problem in CS & Operations.

​

2. Go to the order and click Ship.

​

3. Under Items to Ship, enter the Qty to Ship and Submit Shipment.

Note: Double check the quantity and SKUs shipped. For partial shipment, kindly put 0 for the unshipped SKU.
