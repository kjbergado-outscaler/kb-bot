Invoice Information - MAGENTO

Invoices are an important tool for retailers, so it's very common to receive requests about this document.

​

Whenever an invoice is created, you can find it in the "Invoice" tab in the Order View. Click on the file to open the document, download it and send it to the customer.

Missing Invoices - MAGENTO

Every time you need to create an invoice that is not available in Magento, open the Retail Backoffice Portal and choose Invoice/Claims Concerns -> Send out Missing Invoices.

​

Please give the following information to the Backoffice team:

  • Order Number

Invoice Corrections - MAGENTO

Retailers might ask you to correct Invoices.

If this request is related to the Billing Address, you can change it directly in Magento:

1. Change the Billing Address to the desired one

2. Print the invoice again

Every time you need to correct an invoice that has an error (such as price, items, or quantities), open the Retail Backoffice Portal and choose Invoice/Claims Concerns -> Other.

​

Please give the following information to the Backoffice team:

  • Invoice Number

  • Errors
