This article indicates how you can search, select, and add a Canned Response to reply to a customer.

IMPORTANT: At this moment, we only have Canned Responses in English inserted into the system.

How to add a Canned Response to reply to a customer?

With the case open, go to the bottom-left side where we have the Reply Box, where the message can be written and sent to the customer. There is a line with a few icons, the one you need looks like a notepad.

  1. Click on the Canned Responses button;

  2. Search for the keywords you need;

3. Look at the number next to the Topics-Titles.
If shows more than 2, please make sure to click on “Show All” to be able to check everyone Canned Response under that topic.

4. Click directly on the template you want.
It will automatically appear in the “Reply Box” space and then you can personalize the message and add details if needed before clicking on “Send”.
