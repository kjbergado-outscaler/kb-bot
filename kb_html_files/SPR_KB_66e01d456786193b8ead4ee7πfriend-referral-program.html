The Friend Referral Program

Friend referral is a loyalty campaign we have available to all of our customers.

"Hey, do you know Emma? I have a mattress and I have been sleeping much better after I bought it.
Have you consider buying a new one? I have a referral code that I can share with you for a discount!"

​

The program tends to motivate customers to refer Emma products to their friends and by doing so, both benefit from it. On one side, the customer who is referring wins a Bonus, on the other side, the "friend" buying a new product has a discount.

What is needed from your side?

From your side, we will only need you to know some basic information about this process in case the customer reaches out and raises some questions about the program.


+If you sense that the customer is satisfied with the support they received and could be considered a promoter, feel free to suggest the Friend Referral Program as a way for them to earn additional rewards from Emma. You’ll find the referral link for your market at the end of this article.

Friend Referral Process (How does it work)

Upon finishing the purchase process, the customer will be informed about the referral program and the benefits for both parties.

Ilustrative example

The customer in this case will be considered the "Advocate" and will create a personalized code for their referral code.

Ilustrative example

​

How can the Friend use the code?

  • Option 1 (Not available for FR): By Link sharing, the Advocate can send their friends the referral link.


This will generate a discount code the Friend will be able to use on their purchase.



  • Option 2: By using the Advocate personalized code


The customer proceeds to do their normal order and before checkout, they select the option "Been Refered by a friend?"


Friend proceeds to fill the space with the Advocate code receiving the coupon code to use.

FAQs:

Q: Are there any amount restrictions?
A: Not at the moment, and we don’t expect this to change.

Q: The customer is experiencing issues — where should we escalate?
A: Please collect as much relevant information as possible and ask your supervisor to raise the issue with the Emma Internal Team.

Q: The customer's discount code isn’t working — can I generate a new one?
A: No. These codes are generated by our partner’s tool and must be tracked by the Internal Teams. Please do not create a manual code.

Q: Does the discount code apply to bundles?
A: This depends on the market. If this question comes up, raise it with your supervisor for further guidance.
