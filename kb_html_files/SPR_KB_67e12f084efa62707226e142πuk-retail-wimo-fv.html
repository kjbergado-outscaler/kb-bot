Instructions

Search for the Order

  • Log into FluentCommerce.

  • Use the external order number to search for the order.

Verify the External Order Number

  • Confirm that the external order number matches the one you copied from FluentCommerce.

  • Check the order status: If it's Booked or Waiting for Stock, it indicates the order is still in progress and we are waiting for the replenishment.

Check the Stock Report

The stock update file is sent daily by the BackOffice team. If not available, use the previous day’s file for order tracking and customer queries.

  • Navigate to the stock report to confirm if the item is available.

  • If the stock is available, proceed with tracking the order in Rhenus’s platform.

  • If stock is unavailable, check the replenishment date (for example, Week 5: January 27 to February 2).

    • Inform the Furniture Village that replenishment is scheduled for that specific week, and provide an estimated delivery date based on the replenishment schedule.

We generally expect replenishment to take place by Friday of the specified week. However, to provide more accurate information and avoid confusion, we give an estimated delivery date for the following Friday.

For example, if replenishment is scheduled for Week 8, we anticipate it will occur by Friday, February 21st. When communicating with the customer, though, you should inform them that the estimated delivery date is Friday, February 28th, which corresponds to Week 9.

Handling Special Delivery (UPS) Requests

Special Circumstances for UPS

  • UPS is used only in special cases, such as delayed orders that require expedited delivery.

  • This usually occurs when an item has been delayed for an extended period, and the customer is offered a replacement.

Escalation Process

  • If UPS needs to be used for special deliveries, escalate the situation to your supervisor.
