What is a discount code?


It is a code the customers will be able to add in their order once they are on the check out page. It will allow to buy their orders for a cheaper price. Image below on where to add it:


Types of codes


Influencers


These are the most common, mainly at times of big discounts. They are closed collaborations with influencers, who advertise the products on our social networks and give our customers a discount code to use. Usually they are 10%.



The discount codes provided to Influencers, Affiliates, Newsletters, or used in any other Marketing action such as Telepromotions, cannot be combined with each other, that is, only one will be applied at the time of placing the order, not accumulating.


This one can be used at the same time as the friend code.


*It does not apply for packs.



Newsletter



This is the code that is shared with customers when they subscribe to our Newsletter. This is equivalent to 10%.


The discount codes provided to Influencers, Affiliates, Newsletters, or used in any other Marketing action such as Telepromotions, cannot be combined with each other, that is, only one will be applied at the time of placing the order, not accumulating.


This can be used at the same time as the friend code.


*It does not apply for packs.


Whenever a customer tells us that they are not allowed to use their code, we must verify several things:


- That the customer has subscribed to the newsletter.


- That this person has not used the code for another purchase. If they have made a purchase and it has been cancelled: in this case, we will proceed to verify that they comply with the rest of the conditions, and if so, we will create a discount for the value. This is because this code can only be used once, so if you have already made another purchase and it has not been cancelled, it cannot be used again.



Friend referral


*Not available for new orders in COS. Nevertheless, all codes can still be used.


After purchasing an Emma mattress, the customer has the possibility to share a personalised discount code with their friends. Each of the customer's friends who receive this code will be able to use it on their purchase and receive a discount of 20€. The code is only valid on purchases made on www.emma-colchon.es, and the minimum order spend of your friend(s) must be €290. A maximum of 10 friends can be referred.


This code can be used in conjunction with all the rest of the codes with a maximum of 2 codes.


Whenever a customer tells us that he/she is not allowed to use your code, we must verify several things:


- That the order is more than 290 euros: if it is less than this price, this code does not apply.


- That it has not been used by more than 10 people: if it has already been used 10 times, it cannot be used.


- That this person has used the code for another purchase that has been cancelled: in this case, we will proceed to verify that it meets the rest of the conditions, and if so, we will make the discount manually once you have received the order at home.



Affiliates


Affiliates are a network through which advertisers and affiliates come together. They offer advertising space in exchange for a commission established by the advertising company. Always on the advertiser's website, passing previously through a link or banner that the affiliate has on its website.

*It does not apply for packs.


Telepromos or other collaborations


These are the codes that can be shared in TV commercials, social media or specific platforms that we colaborate with like Vodafone.


*It does not apply for packs.



Can the codes be used at the same time?

The friend referral code is the only one that can be used with the rest of the codes, adding up to a maximum of 2 codes. The rest of the codes can never be used with each other as they are mutually exclusive. All other combinations of more than 1 code are incompatible as the codes are mutually exclusive.



The only code that works for packs is the Sleep Experts one. The rest cannot be used.
