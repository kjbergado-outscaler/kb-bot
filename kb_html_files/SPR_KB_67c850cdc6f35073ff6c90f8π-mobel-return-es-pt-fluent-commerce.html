When to use Mobel? Mobel is a carrier we only use:

  • On Fridays,

  • In Spain and Portugal,

  • For packed, large products

How to create a return with Mobel?

In Fluent Commerce:

  1. Create the Return in Fluent Commerce

  2. In the Carrier field during the Return creation, add: 'MOBEL_ESPT'

  3. The Pick-up date must be left empty

  4. Inform the customer that Mobel will contact them to schedule the date for the pick up.

  5. All returns that contain "MOBEL_ESPT" in the Fluent carrier field and whose return status is "return created" will be automatically transmitted to Mobel.

  6. As soon as we receive the tracking number from Mobel, the return is updated to "return organized" automatically and the tracking number is added to the return order.

  7. Mobel does not give status updates.

How are status updates handled with Mobel?

There are no automated status updates for Mobel. You will have to manage the status changes yourself.

How is the refund triggered?

Once the return is performed by Mobel, our Emma team will ensure the return gets marked as 'complete', which will automatically release the refund. There is no work for you (the agent) or the customer to do.
