In the Sprinklr Tool, Online Reputation agents can reply directly to comments or post made on Social Media platforms. However, if you only reply to the post as a whole, it will only appear as replying to one of the comments made.

If there are multiple comments in a single post, it is important you reply to each individual comment. All you have to do is put the mouse over each message and then a small arrow indicating a reply option will appear.
