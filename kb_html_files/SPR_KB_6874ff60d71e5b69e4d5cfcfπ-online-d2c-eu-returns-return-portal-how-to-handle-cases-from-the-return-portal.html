What is the Return Portal?

The Return Portal is available to customers in specific EU country , providing a fast, fully self-service way to initiate a return 24/7—no emails, phone calls, or chat sessions required. By enabling customers to manage routine returns on their own, we remove a high volume of low-complexity inquiries and free up our agents for higher-value tasks. This streamlined process also speeds up resolution times, reduces back-and-forth communication, and ultimately delivers a smoother, more satisfying experience—especially for customers who see returns as a normal part of shopping.

Reminder

Heads-up on reasons:

  • Outside the trial window: The portal only shows claim-based reasons like “Defective”

  • Inside the trial window: The full menu appears, so the customer can select any reason they like

  • If they select a claim based reason it will promp them to add pictures

At this stage, the portal checks the item and the customer’s region.

  • If automation-eligible: it finalizes the return and emails the shipping label to the customer.

  • If manual review is required: it sends an email to the CS team for follow-up.

Accessory returns run end-to-end through the portal automatically.
Returns for items return via 1-man or 2-man still need an agent’s manual action

For accessory-only returns (meaning no other product will be returned), customers can handle the entire process in the Return Portal.

Once they click Submit, the portal has an integration with Fluent Commerce and ParcelLab, that automatically creates the return and emails the customer with the label required for the drop-off—no Customer Support involvement is required.

If a customer still contacts about an accessory return, agents should:

  1. Guide them to the portal and walk them through the steps.

    or

  2. Create the return for them by opening the portal and submitting the request on their behalf.

Note:

Products considered as accessories for return via drop-off: pillows, bed linens, duvets, mattresses protectors, disassembled/self-assembly slatted frames, baby/kids mattresses, packed toppers or toppers up to 120cm.

When a customer initiates a return for a mattress, bed, or large topper, it will escalate the case to Customer Service / Sleep Experts, via sprinklr ticket, please follow the steps below.

Item within its trial period

  1. Review the reason for return.

  2. Attempt to deflect the return by offering tailored solutions:

    • troubleshooting and setup guidance

    • comfort-adjustment tips

    • supplements or other remedial accessories

  3. Approve the return only if no satisfactory solution is possible.

  4. If the return reason is a claim, handle it according to the Claims Knowledge Base.

Item outside its trial period

  • In the customer portal, the only selectable return reasons will be claim-related.

  • Process these returns strictly under the Claims Knowledge Base guidelines.

5. Return Registered

When the portal displays the “We have registered your return” confirmation (see example below), your job in the system is complete—there’s no need to double-check the order in OMS.

Next step: send a single follow-up email to the customer that includes:

  1. Return confirmation – a brief note that their return has been approved and registered.

  2. Relevant return instructions – clear, step-by-step guidance tailored to this order. For example:
    “Our carrier partner will contact you within x business days to schedule a pickup.”

  3. Refund timeline – highlight when they can expect the credit-memo (e.g., “within 14 business days after we receive your parcel”) to set clear expectations.

This quick email wraps up the process, keeps the customer fully informed, and eliminates unnecessary system checks.
