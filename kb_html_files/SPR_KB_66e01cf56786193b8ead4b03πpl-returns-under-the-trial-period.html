Step 4: Accessories - Create a return label

  1. Provide the customer this link here depending on the market (copy the link as is, do not click in before):

    • PL: https://www.dhl.de/retoure/gw/rpcustomerweb/OrderEntry.action?hash=3edff50f0871f4980c149691d7d5b6dd82e3ba0db6ded07294cc3582ba847035

  2. Ask the customer to enter his/her data and to also add the order nr. in the reference field (important to track the return afterwards)
  3. If the customer has more than one parcel, ask him/her to create as many return labels as there are parcels using the link. (or do it for the customer)
  4. (If you want to take the time and do the customer a favour, you may do steps 2./3. and 4. for the customer and send the customer the finished return label)
  5. Ask the customer to print this label and to paste it on the box
    (multiple parcels: repeat for every parcel with a different return label. DHL will only accept one-time usage of a label.)
  6. Ask the customer to return the parcel the following ways:
    • PL: Poczta Polska parcel shop
  7. Product(s) must be returned inside a square-shaped cardboard box or plastic bag with following maximum dimensions, otherwise it might be rejected at the drop-off point.:
    • PL: 120x60x60cm / 31.5kg maximum
  8. Ask the customer to keep the return receipt until the refund is issued.
  9. To trigger the refund: The customer must send the proof of pick-up to Emmaâ€™s customer service once the item has been picked-up. With the proof of pick-up, we can initiate the refund process. Remember to inform the customer that it can take ~14 days for the refund to be processed.
