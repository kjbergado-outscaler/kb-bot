1. How to Initiate a Return with DHL 2MH in Germany

For Fluent Commerce:

Step 1 - Verify Customer Details: Ensure the correct phone number, email, and pick-up address are in the order.

Step 2 - Select 2MH Service: Choose the 2MH pick-up/return service for all items, including accessories and mattresses. This ensures that all products get picked up at the same time.

Step 3: Create Return:

  • Quality: Select "Packed" (A), "Unpacked" (B), or "Faulty" (C).

  • Leave carrier info, pick-up date, and ParcelLab ID fields empty.

    (Note: After submitting, the carrier info field might be filled with 'DHL_2MH_DE' automatically. This is no issue)

Step 4 - Inform Customer: Provide important information for setting customer expectations.

2. Important Information for Customers

  • Unpacked Mattresses: Customers will receive packaging material to wrap the mattress.

  • Beds: Customers must disassemble the bed themselves.

  • Scheduling: DHL will contact the customer to schedule a pick-up appointment. Do not promise a specific date.

3. Verifying the Return

Fluent Commerce

Ensure customer details are correct, return status is "Created", and return date is empty.

4. Status Updates

  • Status updates are automated and reflected in the Order Management System (OMS).

  • It may take 7-10 days for updates to appear in Fluent Commerce.

  • For detailed updates, check 4PL's platform, Optilo.

  • If you see no update after 10 working days, please reach out to 4PL's Jira Portal to request for an update.

5. Refund Process

Refunds are initiated automatically once the return is marked as complete. Banks or payment service providers have different timelines for processing refunds.

6. Resolving the Return Ticket

Mark the case as "resolved" once the return is created and verified correctly.
