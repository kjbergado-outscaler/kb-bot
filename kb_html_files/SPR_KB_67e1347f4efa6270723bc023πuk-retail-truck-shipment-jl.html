Unlike standard shipments, truck shipments consist of bulk orders delivered to John Lewis without individual acknowledgment numbers, making tracking more complex. Orders are primarily identified using purchase dates and product details, as John Lewis does not provide PO numbers for these cases.

Most inquiries revolve around defective mattresses, requiring careful verification of the purchase date and order details. This SOP outlines the necessary steps to locate and process such orders effectively.

​

Order Tracking and Identification

Locating Orders in Magento

  • If John Lewis contacts regarding an issue, obtain the delivery date of the mattress.

  • Start the search by looking for orders shipped in the month prior to delivery.

  • Use the John Lewis email (johnlewis@example.com) to filter orders.

  • Open each order and look for the specific product model and size in the results.

  • Match the product description and purchase date with the customer's details.

Processing Returns and Replacements

Creating a Return for a Faulty Mattress

  • Locate the order in Magento.

  • Click in Return Products.

​

 

  • Fill-out the information accordingly:

    • Return Products: select the returned product(s)

    • Quantity: add the quantity of returned SKUs

    • Refund to the customer: mark if the return should be followed by a refund. If it’s just a collection case, select “No”.

    • Reason: select if it’s a Delay, Product Claim, Logistic Claim, etc.

​

​

  • Click Save in the top-right corner.

The return will then be created and will appear in the Order page, as Product Returns.

​

​

  • Request the collection in the Collection Spreadsheet.

  • Provide the reference to John Lewis for tracking.

Handling Discontinued Products

  • If the mattress model is discontinued, replace it with the updated version. Follow these steps in Magento:

    • Go to Manage Customers.

    • Locate John Lewis (ID: 215957).

    • Click Create Order.

    • Select UK as the region.

    • Click Add Products and enter the new SKU.

    • Mark the price as Zero (custom price field).

    • Enter the customer’s shipping details.

    • Select Free or Standard Delivery.

    • Add the Case Number provided by John Lewis in the External ID field.

    • Submit the order.

​

Tracking Replacement Orders

  • If John Lewis follows up, search using the case number in Magento.

  • Provide them with the new order reference for tracking.
