Overview

While KHG does not have an internal portal and there are no specific guidelines for email formats, this document details the steps required for when KHG contacts us regarding an order confirmation that has not been received. These emails typically look like the example below:

​

​

Review the e-mail

  • Open the attached document.

  • Identify the external order number: you can find it where its marked in red below.

​

Search for the Order in FluentCommerce

  • Open FluentCommerce and search for the external order number provided in the email.

If the Order is Found

  • Confirm if the order confirmation has been sent.

    • If the confirmation was sent, close the case.

    • You do not need to follow up with KHG since the confirmation was already sent, and no further action is required from them.

If the Order is Not Found/Confirmation was Not Sent

  • If the order is not found in FluentCommerce, or if no confirmation was sent, create a Jira ticket to request the order creation.
