How to add comments in Fluent Commerce

​

Always use comments to explain why orders remain open. It ensures visibility and supports follow-up.

​

  • Go to the order in Fluent.

  • Click the “Comments” tab (top menu).

  • Open the three-dot menu (top-right) and select “Add Comment.”

  • Write a clear reason why the order is still open (e.g., waiting for stock, customer confirmation, etc.).

  • Click Save.

Supplements for Truck Shipments - UK Marketplaces

Raising a Supplement Order under a truck/pallet order for comfort layers:

  1. Go to the supplement order

  2. Click the CREATE PALLET FULFILLMENT option on the upper right side of the screen

  3. Populate the following fields:
    Location - DBS1_GB
    Shipment reference - N/A
    Dispatch date - Next day or when the order is required
    Product - Qty needed

  4. Tick the "Ship as Parcel" box

  5. Click Submit

  • Customer Service (CS) can raise supplement orders for truck shipment orders.

  • In such cases, CS must:

    • Align with the Operations Account Managers (OAM).

    • Inform the Order Managers (OM) of the supplement order.

    • The Order Manager will then process and ship the supplement order as a parcel.

    • The first supplement was received incorrectly or incomplete or damaged.

    • Supplement is taking too long to arrive.

    • Additional items are needed to fully resolve the issue.

    Exceptions (Approval Required)

    Limit of ONE Supplement per order, with some exceptions. Use this reason when a supplement has already been sent, but a second is required due to a special or complex case.

    In such situations, escalate to a Senior Agent for approval.Senior Agents must add a clear Fluent note documenting the approval or rejection, along with the reason.

    If the supplement was initially created by a Senior Agent, Supervisor approval is required.

    Only in exceptional cases,already escalated to Supervisors,should this be escalated internally.

    Examples include:

    • The first supplement was received incorrectly or incomplete or damaged.

    • Supplement is taking too long to arrive.

    • Additional items are needed to fully resolve the issue.
