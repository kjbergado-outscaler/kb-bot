Scope

This SOP outlines the standard procedure for processing shipments of mattress covers from the KNR1_DE warehouse via Magento.

​

Warehouse and Carrier Details

The all covers will be located in the KNR1_DE warehouse.

  • Warehouse Location: KNR1_DE

  • Carrier Assigned: GLS

  • Markets Covered: Germany (DE), Netherlands (NL), Italy (IT), France (FR), Portugal (PT), and Spain (ES)
    ❌ Not applicable for UK and Ireland (UK & IE)

In case any cover is in the WAITING FOR STOCK (WFS) status, please inform the Warehouse Manager of KNR_DE; @Dauren Omarov.

Order Creation

How create a order for the cover? To ship a mattress cover, you must first create a supplement order in Magento.

Steps:

  1. Locate the original mattress order in Magento.

  2. Identify the mattress SKU to determine the correct cover.

  3. Create a supplement with the appropriate cover SKU: Manual Work - Fluent | :1_one_circle_blue: Supplements

Example:

If the mattress SKU is EMAED150200BAC, the corresponding cover SKU to use is ECVDM150200BAA.

​

Do not request return of the original cover — the customer may keep it.
