Refunds

Notes:

  • Refunds process is different based on the payment method used to place the order.

  • Refunds can only be returned to the method of payment used to purchase the item(s).


When opening a Jira ticket for a Refund request at the AR D2C Portal, one key field that needs to be properly inserted is the refund amount.

For how to do a refund in Magento, please check global article here.


Pagolight Refunds

There is no need to create a JIRA


Whenever a customer initiates a return for a refund, we just have to follow the process for Fluent.

​

In fact, the refund team, whenever the item is triggered as returned in Fluent, will process the refund accordingly (if the option "with refund" was chosen when the return was created".
