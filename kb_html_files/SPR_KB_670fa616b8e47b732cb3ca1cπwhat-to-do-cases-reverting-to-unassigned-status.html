What is the reason for this?

After an agent handles the case, they have to complete the necessary ACW. If they do not, the case will get unassigned until the case handling has been finalized (no matter which status was inserted, even awaiting response).

This happens because any agent can handle the case later, it doesn't stay with one person only. The only exception is if we are pending on the customer, and they reply in less than 8 hours. If this is the case, it system will automatically re-assign the case to the same agent as before.
